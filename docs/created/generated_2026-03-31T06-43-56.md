# 생성된 문제

## 완성된 문제
## 검증

### 모든 조건이 필요한가?

**(가) "서로 다른 세 실근" 제거 시:** $f(x) = -(x-3)^3$ 등도 가능. 분기 분석이 달라져 답이 달라짐(또는 성립 불가). → 필요. ✓

**(나) "$f(0) = 15$" 제거 시:** 등간격의 중심 $p$가 결정되지 않아 $f$가 유일하지 않음. → 필요. ✓

**(다) "실근 7개" 제거 시:** $d$ (간격)가 결정되지 않음. → 필요. ✓

**최고차항 계수 $-1$ 제거 시:** $a$가 자유 → $d$도 자유 → $f$ 미결정. → 필요. ✓

### 불필요한 조건이 없는가?

4가지 조건 모두 필수. 불필요한 것 없음. ✓

### 수능 어투 / 자연스러움

수능 기출의 합성함수 방정식 문제와 유사한 구조. 발문 "모든 실근의 합"은 수능에서 자주 출제되는 형태. ✓

### 풀이 가능성 (손계산)

1. $f(x) = -(x-r_1)(x-r_2)(x-r_3)$, $a = -1$.
2. $f(x-f(x))=0$ → $g(x) = x - f(x) = x + (x-r_1)(x-r_2)(x-r_3)$.
3. 7개 실근 → $g$의 극대·극소값이 각각 $r_3, r_1$과 같아야 함 → 등간격 유도.
4. $d = 2$ ($(-1) \cdot d^2 = -4$), $p(p^2-4)=15$ → $p=3$.
5. 각 분기의 근 나열 → 합 계산.

**시험 시간 내 손계산으로 충분히 해결 가능.** ✓

## 완성된 문제

> **최고차항의 계수가 $-1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.**
>
> **(가)** 방정식 $f(x) = 0$은 서로 다른 세 실근을 갖는다.
>
> **(나)** $f(0) = 15$
>
> **(다)** 방정식 $f(x - f(x)) = 0$의 서로 다른 실근의 개수는 $7$이다.
>
> **$f(x - f(x)) = 0$의 모든 서로 다른 실근의 합을 구하시오.** [4점]

### 정답: $\boxed{21}$

### 풀이 요약

$f(x) = -(x-\alpha)(x-\beta)(x-\gamma)$ ($\alpha < \beta < \gamma$)로 놓으면

$g(x) = x - f(x)$는 최고차항의 계수가 $1$인 삼차함수이고, $g(r) = r$ ($r = \alpha, \beta, \gamma$).

$f(x-f(x))=0 \;\Leftrightarrow\; g(x) \in \{\alpha, \beta, \gamma\}$.

$g(x) = \alpha,\;\beta,\;\gamma$ 각각은 삼차방정식이므로 실근은 최대 $3$개씩, 합계 최대 $9$개.

$7$개가 되려면 가능한 분배는 $(2, 3, 2)$뿐:
- $g(x)=\alpha$: 2개 (접선) → $g$가 극솟값 $\alpha$를 가짐 → $g'$의 한 근에서 $g = \alpha$
- $g(x)=\gamma$: 2개 (접선) → $g$가 극댓값 $\gamma$를 가짐
- $g(x)=\beta$: 3개

극점에서 $g' = 0 \Leftrightarrow f' = 1$이고 $g(\text{극점}) = \text{극점} - f(\text{극점})$이 근과 일치하는 조건에서, 계수 비교로 **세 근이 등간격** $\alpha = p-d,\; \beta = p,\; \gamma = p+d$임을 얻고 $(-1) \cdot d^2 = -4$에서 $d = 2$.

$f(0) = p(p^2-4) = 15 \;\Rightarrow\; p = 3$. 따라서 $f(x) = -(x-1)(x-3)(x-5)$.

$g(x) = x^3 - 9x^2 + 24x - 15$, $g' = 3(x-2)(x-4)$.

- $g(x)=1$: $(x-4)^2(x-1)=0$ → $x = 1, 4$
- $g(x)=3$: $(x-3)(x^2-6x+6)=0$ → $x = 3, 3 \pm \sqrt{3}$
- $g(x)=5$: $(x-2)^2(x-5)=0$ → $x = 2, 5$

모든 실근: $1, 2, 3-\sqrt{3}, 3, 3+\sqrt{3}, 4, 5$ — 7개.

합 $= 1+2+(3-\sqrt{3})+3+(3+\sqrt{3})+4+5 = \mathbf{21}$.

## 문제 구조
### 문제 초안
> 최고차항의 계수가 $a$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** 방정식 $f(x) = 0$은 서로 다른 세 실근을 갖는다.
>
> **(나)** $f(0) = c$
>
> **(다)** 방정식 $f(x - f(x)) = 0$의 서로 다른 실근의 개수는 $7$이다.
>
> $f(k)$의 값을 구하시오.
>
> *(여기서 $a$, $c$, $k$는 수치 조정 단계에서 결정될 구체적 상수)*

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 숨은 사실과의 관계 |
|---|---|---|
| **(가)** "$f(x)=0$은 서로 다른 세 실근" | 삼차함수의 세 실근 $r_1<r_2<r_3$ 존재 — 분기 $f(x)=x-r_i$의 개수 확정 | 분해 구조의 전제를 제공하되, 분해 자체는 암시하지 않음 |
| **(나)** "$f(0)=c$" | 함수의 자유도(변곡점 $x$좌표 $p$) 제거 — $f(0)=-a\cdot p(p^2-d^2)=c$ | 등간격 근이 밝혀진 뒤에야 $p$ 결정에 사용됨; 단독으로는 근 구조를 드러내지 않음 |
| **(다)** "$f(x-f(x))=0$의 실근 $7$개" | [사실 A] 평행선 분해 + 분기별 해 개수 분포 $2+3+2$ 강제 → [숨은 사실] 두 분기에서 자기접선 → $f'(r_1)=f'(r_3)=1$ → [사실 B] 변곡점 대칭 $r_1+r_3=2r_2$ | 숨은 사실의 **전체 추론 사슬**이 이 한 조건 안에 압축되어 있음 |
