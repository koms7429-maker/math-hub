## 아이디어 셀렉터

### 수학적 상황
**수학적 객체:** f(x) = x³ + ax² + bx + c (자유 파라미터 a, b, c)

**제약 A (접선–재교차 구조):**
직선 y = x가 곡선 y = f(x)의 점 P(t,t)에서 접선이 되고, P 외에 정확히 한 점 Q에서 곡선과 다시 만난다.

f(t)=t, f'(t)=1이므로 f(x)−x = (x−t)²(x−s). 계수 비교로 b = 1−3t²−2at, c = 2t³+at². **미결: a, t (자유도 2).** b, c는 종속.

**제약 B (합성방정식 근의 개수):**
f(f(x)) = f(0)의 서로 다른 실근이 정확히 N개.

f(y)=c를 풀면 y(y²+ay+b)=0. 수준값 y=0과 이차식의 근 r₁,r₂가 생기고, 각 f(x)=yᵢ의 실근 수는 yᵢ와 f의 극값의 대소에 따라 1/2/3으로 분기. **단독으로 (a,b,c) 공간에서 열린 영역 → 자유도 ≥ 2.**

**공유 파라미터:** a와 t. 제약 A 적용 후 제약 B의 모든 핵심량이 (a,t)의 비선형 함수:
- 이차식 판별식: D = (a+4t)²−4(t²+1)
- 극값 존재 조건: (a+3t)²>3
- 극댓값: a와 t의 삼차식
- 극솟값: a와 t의 삼차식
- 수준값 r₁, r₂: a와 t의 무리식

**강제 결과:** 총 근의 개수 = N을 만족시키려면, 특정 수준값이 극값과 정확히 일치하는 경계 조건이 필요. 이 경계 조건은 a, t에 대한 대수방정식을 생성. 판별식 D의 부호에 따른 케이스 분류에서 대부분의 케이스가 N과 모순되어 탈락하고, 생존하는 유일한 케이스에서 "수준값 = 극값"이라는 두 개의 대수방정식이 (a,t)를 이산적으로 결정. 특히 수준값 r₁이 극솟값과 일치하는 조건과 r₂가 극값 밖에 있는 조건이 동시에 작동하여, a와 t 사이에 예상치 못한 정수비 또는 대칭 관계가 출현.

**영감 source_pid:** P002 (합성방정식 근 분해와 임계값 정렬), P029 (영점 차수 전이 — 이중근 구조가 합성의 수준값 구조를 결정)

### 수험생의 발견 순간
1. **처음 시도:** 제약 A를 전개. f(t)=t, f'(t)=1에서 f(x)−x=(x−t)²(x−s)로 인수분해하여 b, c를 a, t로 표현. 여기까지는 순조롭지만 a, t 두 개가 남아 함수가 확정되지 않음.

2. **막히는 순간:** 제약 B로 넘어가 f(f(x))=c를 전개. y(y²+ay+b)=0의 수준값 3개를 구하고, 각각에 대해 f(x)=yᵢ의 근의 수를 세려 하지만, 극댓값·극솟값이 a, t의 복잡한 식이라 대소 비교가 바로 안 됨. 케이스가 여러 개로 갈라져 어디서 시작해야 할지 막막.

3. **연립 과정:** 총 근의 개수 N에서 역추적. 최대 가능 근의 수(3+3+3=9)에서 출발하여, 어떤 수준값이 극값 밖에 있으면 근이 줄어드는 구조를 파악. N을 만족시키려면 정확히 하나의 수준값이 극값 경계에 놓여야 함을 인식. 이 "경계 조건"(수준값 = 극솟값 등)을 a, t로 쓰면 방정식이 하나 추가되고, 판별식 D의 부호 케이스 중 다수가 N과 모순되어 탈락.

4. **발견:** 생존 케이스에서 "수준값 = 극솟값" 방정식과 "D > 0" 조건을 연립하면, a와 t 사이에 깔끔한 관계(예: t = −a/3 + 상수)가 출현하여 (a,t)가 이산적으로 확정. "접선 y=x의 접점 위치 t가, 합성방정식의 수준값이 극솟값과 일치하는 조건에 의해 유일하게 결정되다니!" — 접선의 기하와 합성방정식의 대수가 하나의 등식으로 수렴하는 순간.
