## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: 모닉 삼차함수 $f(x) = x^3 + ax^2 + bx + c$와 그 곡선 위의 점들.

**사실 A가 부과하는 제약 (P018 — 평행접선 대칭)**:
곡선 위 서로 다른 두 점 $P(\alpha, f(\alpha))$, $Q(\beta, f(\beta))$에서의 접선이 평행하다는 조건 $f'(\alpha) = f'(\beta)$. 이 조건은 이차식인 $f'$의 대칭축을 통해 $\alpha + \beta$의 값을 변곡점 좌표의 2배로 고정한다.

**사실 B가 부과하는 제약 (P012 — 할선 잔차 인수분해)**:
$P$와 $Q$를 잇는 할선(secant) $L(x)$를 $f$에서 차감하면, 잔차 $R(x) = f(x) - L(x)$는 $x = \alpha, \beta$를 이미 근으로 가지며, 모닉 삼차식이므로 $(x - \alpha)(x - \beta)(x - \gamma)$로 완전히 인수분해된다. $\gamma$는 비에타 관계 $\alpha + \beta + \gamma = -a$에 의해 유일하게 결정된다.

**숨은 사실 (A + B 동시 적용)**:
$\alpha + \beta = -\tfrac{2a}{3}$를 $\gamma = -a - \alpha - \beta$에 대입하면 $\gamma = -\tfrac{a}{3}$, 즉 **변곡점의 $x$좌표**. 따라서:

> **삼차곡선 위에서 접선 기울기가 같은 두 점을 잇는 직선은, 기울기 값에 관계없이, 반드시 변곡점에서 곡선과 다시 교차한다.**

---

### 수험생의 발견 순간
1. **처음 시도하는 것**: 조건에서 주어진 두 점의 접선 기울기가 같다는 식 $f'(\alpha) = f'(\beta)$를 세우고, $f'$이 이차식이므로 두 점의 $x$좌표 합 $\alpha + \beta$를 구한다.

2. **막히는 순간**: 할선 $L(x)$를 $f(x)$에서 빼고 제3교점 $\gamma$를 구해야 하는데, $\alpha$와 $\beta$ 각각의 값은 모르고 합만 알고 있어서, $\gamma$를 구체적 수치로 결정할 수 없을 것 같다고 느낀다.

3. **발견**: 비에타 관계 $\alpha + \beta + \gamma = -a$에 이미 알고 있는 $\alpha + \beta = -\tfrac{2a}{3}$를 대입하면, **$\gamma = -\tfrac{a}{3}$으로 즉시 결정된다**. 이 값은 $\alpha, \beta$ 각각의 값이나 공통 기울기 값에 전혀 의존하지 않으며, 삼차함수의 변곡점 그 자체이다.

4. **STEP 2-2 대응**: 이 발견은 "평행접선 대칭(A)이 고정하는 $\alpha + \beta$"와 "할선 잔차의 비에타 합(B)이 결정하는 $\gamma$"가 정확히 맞물려, **제3교점이 매개변수와 무관한 상수점(변곡점)으로 고정되는** 숨은 사실과 정확히 대응한다. 수험생은 "등기울기 할선이 어떤 기울기든 항상 변곡점을 지나는구나"라는 예상치 못한 기하적 불변량을 발견하게 된다.

## 2. 프라블럼 메이커 (Problem Maker)

### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** $\; f'(-2) = f'(4)$
>
> **(나)** 곡선 $y = f(x)$ 위의 두 점 $(-2,\; f(-2))$, $(4,\; f(4))$를 지나는 직선과 곡선 $y = f(x)$의 교점 중 이 두 점이 아닌 점을 $R$이라 하자. 점 $R$에서 곡선 $y = f(x)$에 접하는 직선의 기울기는 $-5$이다.
>
> **(다)** $\; f(0) = 7$
>
> 점 $R$의 $y$좌표를 구하시오. $\qquad$ [$\quad$점]

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 학생에게 보이는 모습 |
|---|---|---|
| **(가)** $f'(-2)=f'(4)$ | 사실 A — 평행접선 대칭 ($\alpha+\beta$ 고정 → $a$ 결정) | 단순한 도함수 등식 조건 |
| **(나)** 전반부: 점 $R$의 정의 | 사실 B — 할선 잔차 인수분해의 무대 (제3근 $\gamma$ 존재) | 할선과 곡선의 교점을 묻는 표준적 상황 설정 |
| **(나)** 후반부: $R$에서 접선 기울기 $= -5$ | $\gamma$ 결정 이후 $b$를 확정하는 조건 | $R$의 좌표를 알아야만 사용 가능한 "잠긴 조건" |
| **(다)** $f(0)=7$ | 상수항 $c$ 결정 | 무해한 함수값 조건 |
| **(숨은 사실)** $\gamma = -\tfrac{a}{3}$ (변곡점) | A+B 결합: $\alpha+\beta=-\tfrac{2a}{3}$을 $\gamma=-a-\alpha-\beta$에 대입 | 어느 조건에도 명시되지 않음. (가)와 (나)를 연립하는 과정에서 발견 |
