# STEP 1. 구조 분석 (기호만 사용)

## 1-1. 조건이 만드는 방정식 체계

**함수 형태:** 최고차항 계수 1인 삼차함수, 조건 (나)에 의해 이중근 $\alpha$ 보유 ⟹

$$f(x) = (x-\alpha)^2(x-\beta), \quad \alpha \neq \beta$$

**(나) 분석:** 점 $(\alpha,0)$에서의 접선 $y = f'(\alpha)(x-\alpha)$. 곡선과의 교점에서 $f(x) - f'(\alpha)(x-\alpha) = (x-\alpha)^2(x-\gamma)$. 제3교점 $\gamma$에서 $y$좌표 $= f'(\alpha)(\gamma-\alpha) = 0$. $\gamma \neq \alpha$이므로 $f'(\alpha) = 0$ 강제. 즉 **$\alpha$는 이중근**.

**파라미터:** $\alpha$, $\beta$ (2개)

| 조건 | 기호 방정식 |
|------|-----------|
| (가) $f(0) = k$ | $\alpha^2(-\beta) = k$, 즉 $-\alpha^2\beta = k$ ··· ① |
| (다) 극댓값 $= m$ | $\alpha > \beta$ 일 때 극대점 $x = \frac{2\beta+\alpha}{3}$에서 $\displaystyle -\frac{4(\beta-\alpha)^3}{27} = m$ ··· ② |

②에서 $d = \alpha - \beta > 0$으로 놓으면 $m = \dfrac{4d^3}{27}$.

**$m$이 양의 정수가 되려면** $d^3 \equiv 0 \pmod{27}$, 즉 $d = 3n$. $n = 1$이면 $d = 3$, $m = 4$.

이로부터: $\beta = \alpha - 3$, ①에 대입하면

$$-\alpha^2(\alpha - 3) = k \quad \Longleftrightarrow \quad \alpha^3 - 3\alpha^2 = -k \quad \cdots (*)$$

| 항목 | 수 |

| 파라미터 | $\alpha$, $\beta$ → 2개 |
| 등식 조건 | ① $-\alpha^2\beta = k$, ② $\alpha - \beta = 3$ → 2개 |

## 1-4. 탐색 전략 수립

$\beta = \alpha - 3$을 ①에 대입: $\alpha^2(\alpha - 3) = -k$, 즉

$$\alpha^3 - 3\alpha^2 + k = 0 \quad \cdots (*)$$

$(*)$의 판별: $h(\alpha) = \alpha^3 - 3\alpha^2$. $h'(\alpha) = 3\alpha(\alpha-2)$.
- 극대: $h(0) = 0$
- 극소: $h(2) = -4$

$(*)$이 **실수해 1개만** 가지려면 $-k > 0$ (즉 $k < 0$) 또는 $-k < -4$ (즉 $k > 4$).

**정수해 후보:** $\alpha^3 - 3\alpha^2 = -k$.

| $\alpha$ | $\alpha^3 - 3\alpha^2$ | $k$ | $\beta$ | 실수해 유일? |
|----------|----------------------|-----|---------|-------------|
| 4 | $64 - 48 = 16$ | $-16$ | 1 | $(*)$: $\alpha^3 - 3\alpha^2 - 16 = (\alpha-4)(\alpha^2+\alpha+4) = 0$, 판별식 $1-16 < 0$ → **유일** ✓ |

**채택 후보:** $\alpha = 4$, $\beta = 1$, $k = -16$, $m = 4$.

# STEP 2. 수치 탐색

## 후보: $\alpha = 4, \, \beta = 1, \, k = -16, \, m = 4$

$$f(x) = (x-4)^2(x-1) = x^3 - 9x^2 + 24x - 16$$

**검증:**
- (가) $f(0) = -16$ ✓
- (나) $\alpha = 4$에서 $f(4) = 0$, $f'(4) = 0$ → 접선 $y = 0$, 이 접선이 곡선과 만나는 다른 점: $f(x) = 0$의 근 $x = 1$. 점 $(1,0)$의 $y$좌표 $= 0$ ✓
- (다) $f'(x) = 3(x-2)(x-4)$, 극대점 $x = 2$, $f(2) = 4$ ✓

---

### $f(f(x)) = 0$ 풀이

$f(t) = 0$의 근: $t = 4$ (이중근), $t = 1$ (단근).

따라서 $f(f(x)) = 0 \iff f(x) = 4$ 또는 $f(x) = 1$.

---

**$f(x) = 4$ (극댓값과 일치):**

$$x^3 - 9x^2 + 24x - 20 = 0$$

$f(2) = 4$이고 $f'(2) = 0$이므로 $x = 2$는 이중근:

$$(x-2)^2(x-5) = 0$$

검증: $(x^2-4x+4)(x-5) = x^3 - 9x^2 + 24x - 20$ ✓

→ 서로 다른 실근: $x = 2, \; 5$ → **2개**

---

**$f(x) = 1$ (극솟값 $0$과 극댓값 $4$ 사이):**

$$x^3 - 9x^2 + 24x - 17 = 0$$

$f$의 극댓값 $4 > 1$, 극솟값 $0 < 1$이므로 수평선 $y = 1$은 곡선과 **3점**에서 교차.

→ 서로 다른 실근: **3개**

---

**중복 확인:** $f(2) = 4 \neq 1$, $f(5) = 4 \neq 1$ → 겹치는 근 없음 ✓

| 시도 | 후보 | $k$ | $m$ | 함수 | 답 | 유일성 | 손계산 | 판정 |
|------|------|-----|-----|------|----|--------|--------|------|
| 1 | $\alpha=4,\,\beta=1$ | $-16$ | $4$ | $(x-4)^2(x-1)$ | **5** | ✓ 유일해 | ✓ | **채택** |

### 조건-상황 대응표
| 조건 | 수학적 상황의 요소 |
|---|---|
| **(가)** $f(0) = k$ | 함수값 제약: $q = k$. 이중근 구조와 결합 시 $-\alpha^2\beta = k$로 전환. 단독으로는 $b, p$를 건드리지 않아 구조적 정보 없음. |
| **(나)** 근 $\alpha$에서의 접선 재교차점의 $y$좌표 $= 0$ | 사실 B의 기하적 위장. 접선의 제3교점 $y$좌표가 $f'(\alpha)(\gamma-\alpha)$이므로, $\gamma \neq \alpha$와 결합하면 $f'(\alpha)=0$이 강제됨. 즉 $\alpha$는 이중근이라는 숨은 사실을 유발. |
| **(다)** 극댓값 $= m$ | 이중근 $\alpha$에서의 극값 $f(\alpha) = 0$은 자동 확보. 나머지 극점 $x = \frac{2\beta+\alpha}{3}$에서의 함수값이 $m$이 되어야 하므로 $\alpha, \beta$에 대한 2번째 방정식 제공. (가)와 연립하여 함수 확정. |
| **발문** $f(f(x))=0$의 실근 개수 | 사실 A: $f(x) = \alpha$ (이중근, $f(\alpha)=0$) 또는 $f(x)=\beta$ (단근)으로 분해. 각 하위방정식의 실근 수는 $\alpha, \beta$가 $f$의 두 임계값($0$과 $m$) 사이·위·밖 중 어디에 위치하는지에 따라 결정됨. |
