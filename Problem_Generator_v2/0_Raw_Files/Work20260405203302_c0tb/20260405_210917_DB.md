# [STEP 1] 구조 분석 (기호만 사용)

## 1-1. 조건이 만드는 방정식 체계

**함수:** $f(x) = x^3 + ax + b$, $b > 0$.

### 조건 (가): 극댓값과 극솟값을 갖는다

$$f'(x) = 3x^2 + a = 0 \text{이 서로 다른 두 실근} \;\Longrightarrow\; a < 0$$

극점: $x = \pm\sqrt{-a/3}$

### 조건 (나): 직선 $y = f(2)$가 곡선 $y = f(x)$와 $x \neq 2$인 점에서 접한다

$f(x) - f(2)$을 인수분해한다.

$$f(x) - f(2) = x^3 + ax + b - (8 + 2a + b) = x^3 + ax - 8 - 2a$$

$x = 2$는 항상 근이다: $8 + 2a - 8 - 2a = 0$ ✓

$$f(x) - f(2) = (x - 2)(x^2 + 2x + 4 + a)$$

접선 조건: $x = t\;(\neq 2)$에서 접한다는 것은 다음 두 조건을 동시에 만족:
- $f(t) = f(2)$ → $t$는 $x^2 + 2x + 4 + a = 0$의 근
- $f'(t) = 0$ → $3t^2 + a = 0$ → $t^2 = -a/3$

$x^2 + 2x + 4 + a = 0$에 $t^2 = -a/3$을 대입:

$$-\frac{a}{3} + 2t + 4 + a = 0 \;\Longrightarrow\; t = -\frac{a}{3} - 2$$

$t^2 = -a/3$에 대입, $u = -a/3 > 0$으로 치환:

$$(u - 2)^2 = u \;\Longrightarrow\; u^2 - 5u + 4 = 0 \;\Longrightarrow\; (u-1)(u-4) = 0$$

| $u$ | $a$ | $t = u - 2$ | $t \neq 2$ 여부 |
|-----|------|-------------|----------------|
| $1$ | $-3$ | $-1$ | ✅ |
| $4$ | $-12$ | $2$ | ❌ (기각) |

**결론:** $a = -3$, 접점 $t = -1$.

### 조건 (다): $|f(x)| = f(0)$의 서로 다른 실근의 개수가 5

$f(0) = b > 0$이므로 $|f(x)| = b$, 즉:

$$f(x) = b \quad\text{또는}\quad f(x) = -b$$

**[방정식 ①]** $f(x) = b$: $x^3 + ax = 0$ → $x(x^2 + a) = 0$

$a = -3$ 대입: $x(x^2 - 3) = 0$ → 근: $x = 0,\;\pm\sqrt{3}$ → **항상 3개**

**[방정식 ②]** $f(x) = -b$: $x^3 + ax + 2b = 0$

$a = -3$ 대입: $x^3 - 3x + 2b = 0$

**공통근 불가:** ①과 ②를 동시에 만족하면 $2b = 0$이지만 $b > 0$이므로 공통근 없음.

∴ 전체 실근 개수 = $3 + (\text{②의 서로 다른 실근 수}) = 5$

→ **②가 정확히 2개의 서로 다른 실근**을 가져야 함 → **이중근 존재**

## 1-2. 분기 분석

$g(x) = x^3 - 3x + 2b$, $g'(x) = 3(x^2 - 1)$.

| 극점 | 극값 | 이중근 조건 (극값 = 0) | $b$ |
|------|------|----------------------|-----|
| $x = -1$ (극대) | $g(-1) = 2 + 2b$ | $2 + 2b = 0$ | $b = -1$ (❌, $b > 0$) |
| $x = 1$ (극소) | $g(1) = -2 + 2b$ | $-2 + 2b = 0$ | $b = 1$ (✅) |

### 판정 1-2

$b = 1$이 유일한 후보. 분기가 1개로 수렴하며, SENDBACK 사유 없음. **CONTINUE.**

## 1-3. 자유도 계산

| 파라미터 | 조건 | 결정값 |
|---------|------|-------|
| $a$ | (나)에서 유일 결정 | $a = -3$ |
| $b$ | (다)에서 유일 결정 | $b = 1$ |

**자유도:** 파라미터 2개, 독립 조건 2개 → 자유도 = 0

### 판정 1-3

자유도 0, 해 유일. **CONTINUE.**

## 1-4. 탐색 전략 수립

$(a, b) = (-3, 1)$이 유일 후보. 수치 대입으로 전 조건 검증 후, 발문 $f(2)$의 적절성과 포장을 확정한다. **CONTINUE.**

# [STEP 2] 수치 탐색 + 포장 설계

## 후보: $(a, b) = (-3, 1)$

$$f(x) = x^3 - 3x + 1$$

### 전 조건 검증

**(가)** $f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$.
- 극대: $f(-1) = -1 + 3 + 1 = 3$
- 극소: $f(1) = 1 - 3 + 1 = -1$

극댓값 $3$, 극솟값 $-1$ 모두 존재. ✅

**(나)** $f(2) = 8 - 6 + 1 = 3$.
$$f(x) - 3 = x^3 - 3x - 2 = (x - 2)(x + 1)^2$$
$x = -1$에서 이중근 → 접선. $-1 \neq 2$. ✅

> **숨은 구조 확인:** $f(2) = f(-1) = 3$ = 극댓값. 즉 직선 $y = f(2)$는 극대점을 지나는 수평선이며, 이것이 접선 조건의 정체.

**(다)** $|f(x)| = f(0) = 1$:
- $f(x) = 1$: $x^3 - 3x = 0$ → $x = 0,\;\pm\sqrt{3}$ → **3개**
- $f(x) = -1$: $x^3 - 3x + 2 = 0$ → $(x-1)^2(x+2) = 0$ → $x = 1,\;-2$ → **2개**
- 공통근 없음 (확인: $f(0) = 1 \neq -1$, $f(\pm\sqrt{3}) = \mp 2\sqrt{3}+1 \neq -1$)
- 전체: $3 + 2 = 5$개 ✅

> **숨은 구조 확인:** $f(1) = -1 = -f(0)$, 즉 극솟값 $= -b$. 이것이 $f(x) = -b$에서 이중근이 생기는 이유.

### 발문 검토

$f(2) = 3$: 양의 정수, 0~999 범위, 깔끔한 값. ✅

### 포장 설계

| 항목 | 결정 | 근거 |
|------|------|------|
| 발문 | $f(2)$의 값 | PM 원안 유지. 조건 (나)에 $f(2)$가 등장하므로 자연스러운 연결 |
| 조건 표면 형태 | 변경 없음 | (나)의 기하적 접선 포장, (다)의 절대값 방정식 — 모두 핵심 구조를 효과적으로 은닉 |
| 변수 은닉 | $a, b$를 직접 묻지 않음 | $a = -3, b = 1$을 경유해야 $f(2)$를 구할 수 있어 풀이 경로가 자연스럽게 강제됨 |
| 유일성 | 확보됨 | (나)→$a$ 유일, (다)→$b$ 유일, 발문 변경 불필요 |

### 실패 후보: 없음 (첫 번째 후보에서 성공)

### 채택

> **$(a, b) = (-3, 1)$, $f(2) = 3$**
