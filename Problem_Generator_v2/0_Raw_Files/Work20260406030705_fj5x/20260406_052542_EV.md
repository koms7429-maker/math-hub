### 1-3. 조건 연립 + 풀이

**연립 순서:** (나)를 먼저 전개하여 $b = -\frac{9+4a}{2}$ 획득 → (가)에 대입

**【진행 1】(나) 전개**
$b = -\frac{9+4a}{2}$. $a$가 자유 파라미터로 남음.

**【전환 지점 1】** (나) 단독으로는 $a$가 미결 → (가)의 부등식을 끌어와야 함을 인식.

**【진행 2】(가)에 (L) 대입**

*Case I ($a \ge 0$):* $b = -\frac{9+4a}{2} < 0$ (분자 $> 0$) → $b \ge 0$ 조건에 **모순**. 탈락.

*Case II ($a < 0$):* $b \ge \frac{2a^2}{9}$ 조건에 대입:

$$-\frac{9+4a}{2} \ge \frac{2a^2}{9}$$

양변 $\times 18$:

$$-9(9+4a) \ge 4a^2 \;\Longrightarrow\; 4a^2 + 36a + 81 \le 0$$

**【전환 지점 2】** 좌변을 인수분해하면:

$$\boxed{(2a+9)^2 \le 0}$$

완전제곱 $\ge 0$이므로 **등호만 가능**: $a = -\dfrac{9}{2}$.

**【결과】**
$$a = -\frac{9}{2}, \quad b = -\frac{9 + 4(-9/2)}{2} = -\frac{9-18}{2} = \frac{9}{2}$$

$$f(x) = x^3 - \frac{9}{2}x^2 + \frac{9}{2}x = x\!\left(x - \frac{3}{2}\right)\!(x - 3)$$

**검증:**
$$g(x) = \frac{x^2(x-3)^2}{4} \ge 0\;(\forall\,x>0)\;\checkmark, \quad g(3) = \frac{9 \cdot 0}{4} = 0\;\checkmark$$

**최종 답:**
$$f(5) = 5 \cdot \frac{7}{2} \cdot 2 = \boxed{35}$$

### 1-4. 유일성 확인

$(2a+9)^2 = 0$에서 $a = -9/2$가 **유일**. $b$는 (L)에 의해 $9/2$로 종속.
복수해 없음. → **O (유일해)**

### 1-5. 조건 과잉/부족 체크

- **(가) 제거:** (나)만 남으면 $b = -(9+4a)/2$, $a$는 자유. 예: $a = -3 \Rightarrow b = 3/2$, $f(x) = x^3 - 3x^2 + \frac{3}{2}x$, $f(5) = 125 - 75 + 7.5 = 57.5 \ne 35$. → **답이 달라짐. (가)는 필수.**

- **(나) 제거:** (가)만 남으면 $(a,b)$가 포물선 위쪽 영역의 임의의 점. 예: $a = 0, b = 0$이면 $f(5) = 125 \ne 35$. → **답이 달라짐. (나)는 필수.**
