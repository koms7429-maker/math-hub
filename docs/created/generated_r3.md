## [0] 이번 시도의 차별성 선언

- **아이디어 조합**: 이전(P004+P006: 등도함수값 대칭 + 접선-할선 연쇄) → 이번(P019+P005: 접선 차감 중근 승격 + 차분방정식 환원) — P004와 P006은 모두 "삼차곡선의 근에서 출발하는 접선 기하"라는 동질적 메커니즘을 공유했기 때문에, P004의 대칭 조건이 P006의 출발점을 일방향으로 결정하는 순차 구조가 불가피했다. 이번에는 **접선 차감(국소적·미분적 구조)**과 **차분방정식(전역적·차분적 구조)**이라는 이질적 메커니즘을 선택하여, 하나가 다른 하나의 입력이 되는 순차 의존을 원천 차단한다.

- **연동 방식**: 이전(하나의 조건이 파라미터를 확정 → 나머지 조건에 대입) → 이번(**대칭함수 직교 분해**: 한 조건이 근 간격의 합 $p+q$를 구속하고, 다른 조건이 근 간격의 차 $(p-q)^2$를 구속하여, 어느 한쪽만으로는 개별 근이 결정되지 않고 두 조건을 연립하는 순간 $p,\,q$가 동시에 확정되는 구조) — 이전의 "변곡점 = 근" 같은 즉시적 기하 귀결이 아니라, 두 조건이 **서로 다른 대칭함수 축**에 각각 하나씩 방정식을 배치하는 쌍방향 구속이다.

## [2] 각 아이디어가 만드는 계수 관계식 (기호 수준)

공통 객체: 모닉 삼차함수 $f(x) = (x-\alpha)(x-\beta)(x-\gamma)$.

근 $\alpha$를 기준점으로 삼고 **간격 변수** $p = \beta - \alpha$, $q = \gamma - \alpha$을 도입한다. 그러면

$$f(\alpha + h) = h(h - p)(h - q)$$

이며, 함수의 모든 정보가 $(\alpha,\, p,\, q)$로 인코딩된다.

---

### 관계식 R₁ (P019에서 유도)

**조건**: 근 $\alpha$에서의 접선 $y = f'(\alpha)(x - \alpha)$를 $f$에서 차감한다.

$$g(x) \;=\; f(x) - f'(\alpha)(x - \alpha) \;=\; (x - \alpha)^2\bigl(x - \delta\bigr)$$

여기서 $f'(\alpha) = (-p)(-q) = pq$이고, 비에타 합으로부터

$$2\alpha + \delta = \alpha + \beta + \gamma \quad\Longrightarrow\quad \delta = \beta + \gamma - \alpha = \alpha + (p + q)$$

**$g(x) = k$ (특정 수준값)의 실근 개수 조건**: $g$는 $x = \alpha$에서 극솟값(또는 극대) 0을, $x = \alpha + \tfrac{2(p+q)}{3}$에서 유일한 비자명 극값

$$g\!\Bigl(\alpha + \tfrac{2(p+q)}{3}\Bigr) = -\,\frac{4(p+q)^3}{27}$$

을 가진다. "$g(x) = k$가 정확히 $N$개의 실근을 갖는다"는 조건은 $k$가 이 극값과 어떤 대소 관계에 있느냐를 강제하며, 이는 $k$와 $(p+q)$의 관계식

$$\boxed{R_1:\quad (p + q)^3 = \Psi(k) \quad\text{(}k\text{와 근 개수 }N\text{이 결정하는 대수식)}}$$

을 부여한다. 이 관계식은 **합 $p + q$**를 $k$에 종속시키지만, $p$와 $q$를 개별적으로 결정하지는 못한다.

---

### 관계식 R₂ (P005에서 유도)

**조건**: 곡선 $y = f(x)$ 위의 두 점 $(\alpha,\, 0)$과 $(\alpha + h,\, f(\alpha + h))$가 축정렬 정사각형(또는 이에 준하는 기하 도형)의 꼭짓점이 되어, $f(\alpha + h) = \pm h$ (또는 $ch$)를 만족한다.

$f(\alpha + h) = h(h-p)(h-q)$이므로

$$h(h - p)(h - q) = \pm h$$

$h \neq 0$으로 나누면:

$$(h - p)(h - q) = \pm 1$$

$$h^2 - (p+q)\,h + (pq \mp 1) = 0$$

이 이차방정식의 판별식은

$$\Delta = (p+q)^2 - 4(pq \mp 1) = (p - q)^2 \pm 4$$

"$h$의 실근이 정확히 $M$개"라는 조건은 $\Delta$의 부호를 강제하며, 이는

$$\boxed{R_2:\quad (p - q)^2 = \Phi(\text{부호 선택},\; M) \quad\text{(이산적 등식 또는 부등식)}}$$

을 부여한다. 특히 $M$이 임계적인 경우(예: 정확히 하나의 실근, 이중근 등) $\Delta = 0$이 되어 $(p-q)^2$가 상수로 확정된다. 이 관계식은 **차 $p - q$ (또는 그 제곱)**를 구속하지만, $p + q$에 대해서는 아무 정보를 주지 않는다.

## [3] 파라미터 공유 확인

| | 직접 구속하는 대칭함수 | 관여하지만 구속하지 않는 것 |
|---|---|---|
| $R_1$ | $p + q$ (합) | $p - q$ (미결정) |
| $R_2$ | $(p - q)^2$ (차의 제곱) | $p + q$ (미결정) |

**공유 확인**: 두 관계식 모두 동일한 파라미터 쌍 $(p,\, q)$에 작용하나, **서로 다른 대칭함수 축**을 구속한다.

- $R_1$만으로: $p + q = s$ (기지)이나 $p - q$는 자유. $\Rightarrow$ $(p,q)$가 직선 $p + q = s$ 위의 1-파라미터 족.
- $R_2$만으로: $(p - q)^2 = d$ (기지)이나 $p + q$는 자유. $\Rightarrow$ $(p,q)$가 쌍곡선 $(p-q)^2 = d$ 위의 1-파라미터 족.
- **$R_1 \cap R_2$**: 직선 $p + q = s$와 곡선 $(p-q)^2 = d$의 교점. $p = \frac{s \pm \sqrt{d}}{2}$, $q = \frac{s \mp \sqrt{d}}{2}$로 **유한 개(최대 2개) 해가 즉시 확정**된다.

→ 어느 한 조건만으로는 해가 닫히지 않고, **두 조건을 연립하는 순간 비로소** 개별 근 간격 $p$, $q$가 결정된다. 이것은 진정한 **쌍방향 구속**이다.

## [4] 수학적 상황 서술

> **상황**: 모닉 삼차함수 $f$에 대해, 한 근에서의 접선 차감으로 생긴 중근 삼차식의 수준집합 근 개수 조건(P019)과, 같은 근을 기준점으로 한 축정렬 기하 도형의 꼭짓점 조건(P005)이 **동시에** 부과된다.

### 구조 전개

1. **공통 좌표계 설정** — 근 $\alpha$를 기준점으로 고정하고 $h = x - \alpha$로 이동하면, $f(\alpha+h) = h(h-p)(h-q)$로 정규화된다. 문제의 모든 조건이 $(p, q)$ 2차원 공간 위에 놓인다.

2. **P019 적용 (합 방향 구속)** — 접선 차감 $g(x) = f(x) - pq(x-\alpha)$는 $(x-\alpha)^2(x-\alpha-(p+q))$로 인수분해된다. $g(x) = k$의 실근 개수 조건이 $g$의 유일한 비자명 극값 $-4(p+q)^3/27$과 $k$의 대소를 강제한다. 이로써 $(p+q)^3$이 $k$의 함수로 결정된다 → **합 $p+q$ 확정**. 그러나 $(p-q)$에 대한 정보는 전혀 없다.

3. **P005 적용 (차 방향 구속)** — 기하 조건 $f(\alpha+h) = \pm h$를 $h \neq 0$으로 나눈 잔여 이차식 $h^2 - (p+q)h + (pq \mp 1) = 0$의 근 개수 조건이 판별식 $(p-q)^2 \pm 4$의 부호를 강제한다. 임계적 경우 $(p-q)^2$가 상수로 확정된다 → **차 $(p-q)^2$ 확정**. 그러나 $(p+q)$에 대한 정보는 전혀 없다.

4. **연립의 닫힘** — $R_1$에서 $p+q = s$, $R_2$에서 $(p-q)^2 = d$가 확정되면:
$$p = \frac{s + \sqrt{d}}{2},\quad q = \frac{s - \sqrt{d}}{2} \quad\text{(또는 교환)}$$
이로써 $\beta = \alpha + p$, $\gamma = \alpha + q$가 결정되고, $\alpha$ 자체는 별도의 단순 조건(예: 특정 점에서의 함수값)으로 고정된다.

### 핵심 충돌(비자명성)이 드러나는 지점

**충돌 1: 대칭함수 축의 직교성**. $R_1$은 $p+q$ 축, $R_2$는 $p-q$ 축을 각각 독점적으로 구속한다. 이것은 우연이 아니다: 접선 차감의 제2교점 $\delta$는 $\alpha + (p+q)$에만 의존하므로 자연스럽게 합 정보를 인코딩하고, 차분방정식의 $h$로 나누기 후 판별식은 $(p+q)^2 - 4pq = (p-q)^2$로 변환되므로 자연스럽게 차 정보를 인코딩한다. 두 메커니즘이 $(p,q)$ 공간의 **직교하는 좌표축**에 각각 정보를 배치하는 것이다.

**충돌 2: 곱 $pq$의 이중 역할**. $pq = f'(\alpha)$는 접선 기울기(P019의 출발점)이면서 동시에 P005 잔여 이차식의 상수항($pq \mp 1$)에도 출현한다. $p+q$와 $(p-q)^2$가 각각 확정된 후, $pq = \frac{(p+q)^2 - (p-q)^2}{4}$로 결정되어 접선 기울기가 "사후적으로" 확정된다. 즉, **접선의 기울기 자체가 두 조건의 연립 결과**로서만 알 수 있는 양이 된다. 이것은 P019가 접선 기울기를 "사용"하지만 그 값을 결정하는 것은 P005와의 연립이라는 순환적 구조를 만든다.

**충돌 3: 이산적 후보의 출현**. $(p-q)^2 = d$에서 $d > 0$이면 $p - q = \pm\sqrt{d}$의 두 부호 선택이 가능하고, 이는 $\beta$와 $\gamma$의 교환에 대응한다. 만약 문제가 $\beta < \gamma$ 같은 순서 조건을 추가로 부과하면 하나가 제거되지만, 그렇지 않으면 두 해가 모두 유효하여 **$f$의 개수를 묻는 문제**가 자연스럽게 성립한다. 또한 $d = 0$인 경계 경우 $p = q$, 즉 $\beta = \gamma$가 되어 이중근이 발생하며, 이것이 허용되는지 여부가 추가 분기점이 된다.

## [5] 수험생의 발견 지점

> **발견 1 (좌표 이동의 위력)** — 수험생은 근 $\alpha$를 기준으로 $h = x - \alpha$ 치환을 하면 $f(\alpha + h) = h(h-p)(h-q)$로 정규화되어, 두 조건이 모두 $(p,q)$만의 문제로 환원됨을 깨닫는다. 이 치환 자체는 단순하지만, 두 조건이 모두 같은 좌표계에서 자연스럽게 표현된다는 것은 비자명한 인식이다.

> **발견 2 (접선 차감이 "합"을 인코딩함)** — P019의 접선 차감을 수행하면 $(x-\alpha)^2(x-\alpha-(p+q))$가 나오고, 비자명 극값이 $(p+q)^3$에만 의존한다는 것을 발견한다. 수험생은 처음에 $p$와 $q$를 개별적으로 구하려 하지만, 이 조건이 **합만 결정**한다는 한계에 부딪힌다.

> **발견 3 (차분 나눗셈이 "차"를 인코딩함)** — P005의 기하 조건에서 $h \neq 0$으로 나누면 이차식이 되고, 그 판별식이 $(p+q)^2 - 4pq = (p-q)^2 \pm 4$임을 계산한다. 여기서 $(p-q)^2$가 등장하는 것을 보고, 발견 2에서 막혔던 "합만으로는 불충분"했던 문제가 **"차"를 구하면 해결**된다는 것을 깨닫는 순간이 핵심 전환점이다.

> **발견 4 (두 조건의 직교적 조합)** — $R_1$에서 $p+q = s$, $R_2$에서 $(p-q)^2 = d$를 얻으면, $p$와 $q$가 $\frac{s \pm \sqrt{d}}{2}$로 깔끔하게 확정된다. 복잡해 보이던 삼차함수 결정 문제가, 두 조건이 마치 **직교좌표의 $x$축과 $y$축처럼** 독립적 정보를 제공하여 교차점 하나로 수렴하는 것을 체감한다. 이 구조적 단순화가 풀이의 최종 통찰이다.

## [0] 이번 시도의 차별성 선언

- **조건 형태**: 이전(접선 기울기 동치(등식) + 접선-재교점 연쇄(존재성·수치) + 도함수 수치 고정) → 이번(**접선 평행이동의 교점 개수(개수 조건) + 곡선-직선 교점 개수(개수 조건)**). 이전은 등식/수치 조건의 직렬 연쇄였으나, 이번은 두 조건 모두 "개수"라는 동질적 외관을 띠면서도 내부적으로는 서로 다른 대칭함수를 구속한다. 조건의 형태가 유사해 보이므로 수험생이 두 조건의 역할 차이를 인식하기 더 어렵다.

- **충돌 표현**: 이전(간접적 위장 — 순차 구조로 (가)→(나)→(다)의 일방향 파이프라인) → 이번(**직교적 분해** — (가)가 합 $\beta+\gamma$를, (나)가 차 $(\gamma-\beta)^2$를 각각 독립적으로 구속하여, 어느 한 조건만으로는 근이 결정되지 않고 두 조건의 연립 시점에서 동시 확정). 순차가 아닌 쌍방향 구속이므로 이전과 구조적으로 다르다.

## [무대 설정]

## [조건 배치 계획]

| 조건 | 번역 대상 | 표면적 인상 | 숨겨진 효과 |
|:---:|---|---|---|
| **(가)** | P019 — 접선 차감에 의한 중근 승격 + 수준집합 근 개수 | "원점 접선을 $k$만큼 평행이동한 직선이 곡선과 두 점에서 만난다"는 **교점 개수** 조건. 접선 기하의 정량화로 보이며, 개별 근 $\beta,\gamma$와의 관계는 불투명 | 접선 차감 $g(x)=f(x)-\beta\gamma\cdot x = x^2\bigl(x-(\beta+\gamma)\bigr)$의 유일한 비자명 극값이 $-\tfrac{4(\beta+\gamma)^3}{27}$이므로, 평행이동량 $k$가 이 극값과 일치할 때만 교점 2개. ∴ $(\beta+\gamma)^3$이 $k$의 함수로 확정 → **합 $\beta+\gamma$ 결정**, 차 $\gamma-\beta$는 미결정 |
| **(나)** | P005 — 차분방정식 환원에 의한 해 개수 제어 | "곡선과 직선 $y=-x$가 두 점에서 만난다"는 **교점 개수** 조건. (가)와 형태가 유사하여 독립적 기하 상황으로 읽힘 | $f(x)+x = x\bigl[(x-\beta)(x-\gamma)+1\bigr]$에서 이차 잔여식의 판별식이 $(\gamma-\beta)^2-4$이므로, 교점이 정확히 2개 ↔ $(\gamma-\beta)^2=4$ → **차 $\gamma-\beta$ 결정**, 합 $\beta+\gamma$는 미결정 |
| **없음 (추가 불필요)** | — | — | $(\beta,\gamma)$는 2-파라미터이고, (가)+(나)가 각각 독립적인 축을 구속하여 **2조건으로 계가 닫힘**. 별도 조건 (다) 불요 |

**충돌 지점 은폐 확인**:

- **(가)만 보면**: 접선의 평행이동이라는 기하 작업 속에서 $x^2(x-s)$ 형태가 등장하고, 극값 조건이 $s = \beta+\gamma$만을 결정한다. 그러나 이것이 "합만 결정하고 차는 미결정"이라는 사실은, 실제로 풀어보기 전까지 인식하기 어렵다.
- **(나)만 보면**: 직선 $y=-x$와의 교점 개수는 (가)와 무관한 별도의 기하 상황이다. 인수분해 후 판별식에 $(\gamma-\beta)^2$가 등장한다는 것은 계산 결과로만 확인된다.
- **핵심 은폐**: 두 조건이 겉보기에는 동일 유형("교점 개수")인데, 하나는 합을, 다른 하나는 차를 구속한다는 **직교성**이 문제 표면에 드러나지 않는다. 수험생이 (가)를 처리한 후 "$\beta+\gamma$만 나왔다"는 한계에 부딪히고, (나)를 처리한 후 "$(\gamma-\beta)^2$만 나왔다"는 별개의 한계에 부딪힌 다음, 두 결과를 합치는 순간 $\beta,\gamma$가 동시 확정되는 **돌파의 순간**을 경험한다.
- **추가 은폐 층위**: $f'(0) = \beta\gamma = \frac{(\beta+\gamma)^2 - (\gamma-\beta)^2}{4}$이므로, 발문이 $f'(0)$을 물으면 "접선 기울기를 (가)에서 사용했는데 그 값은 (나)까지 풀어야 확정된다"는 순환적 구조가 최종 단계에서 드러난다. 이것이 아이디어 셀렉터가 명시한 "충돌 2: 곱 $pq$의 이중 역할"에 해당한다.

# [0] 이번 시도의 차별성 선언

- **수치 설정**: 이전(Round 1: $\beta=2,\gamma=4$, 답 $8$) → 이번($\beta=2,\gamma=4$ 동일 근이지만, **반대 부호 경쟁해 $\beta=-4,\gamma=-2$도 조건 (가)+(나) 충족**하도록 재설계). 이전에는 $s^3=-27k/4$에서 $s$가 유일했으나, 이번은 $s^2=36$으로 $s=\pm 6$ 분기 발생.
- **조건 추가/변경**: 이전((가)에서 $k=-32$ 고정 → $s$ 유일 결정) → 이번((가)에서 "두 교점의 $x$좌표의 곱 $=-8$" → $s^2=36$만 결정, 부호 미결). **양수 전제가 $s=6$을 선별하는 실질적 역할** 수행.
- **발문 형태**: 이전($f'(0)=8$, 두 경우 모두 동일값) → 이번($f(1)$, $s=6$일 때 $3$, $s=-6$일 때 $15$로 **양수 전제 없이는 답 미확정**).
- **유일성 처리**: "0이 아닌 두 근은 모두 양수" 전제가 $s>0$을 선별하여 유일해 확보. 이 전제 제거 시 답이 $3$ 또는 $15$로 변동 → **잉여 조건 없음**.

# STEP 1. 파라미터 결정 가능성 검토

**세팅.** $f(x)=x(x-\beta)(x-\gamma)=x^3-sx^2+px$, $\;s=\beta+\gamma$, $\;p=\beta\gamma=\dfrac{s^2-d^2}{4}$, $\;d=\gamma-\beta>0$. 자유 파라미터 2개.

### (가) 분석: 접선 평행이동의 교점 $x$좌표 곱 → $s^2$ 결정

원점 접선 $\ell:\;y=px$. 평행이동 직선 $y=px+k$.

$$f(x)-px=x^2(x-s)\overset{\text{def}}{=}g(x)$$

$g(x)=k$가 서로 다른 두 실근을 가지려면 $k$가 $g$의 극값과 일치해야 한다.

$$g'(x)=x(3x-2s),\qquad g(0)=0,\quad g\!\left(\tfrac{2s}{3}\right)=-\tfrac{4s^3}{27}$$

$k\neq 0$이면 $k=-\dfrac{4s^3}{27}$이 유일한 선택.

이때 $g(x)-k=(x+\tfrac{s}{3})(x-\tfrac{2s}{3})^2=0$이므로 두 교점의 $x$좌표는

$$x_1=-\frac{s}{3}\;(\text{단근}),\qquad x_2=\frac{2s}{3}\;(\text{이중근})$$

$$x_1\cdot x_2=-\frac{s}{3}\cdot\frac{2s}{3}=-\frac{2s^2}{9}$$

주어진 곱 $=-8$:

$$-\frac{2s^2}{9}=-8\;\;\Longrightarrow\;\;s^2=36\;\;\Longrightarrow\;\;s=\pm 6$$

**효과.** $s^2$만 결정되어 $s=6$ 또는 $s=-6$. **자유도 $2\to 1$** (부호 미결+$d$ 미결).

### (나) 분석: $y=-x$와의 교점 개수 → $d$ 결정

$$f(x)+x=x\bigl[(x-\beta)(x-\gamma)+1\bigr]=x\bigl[x^2-sx+(p+1)\bigr]=0$$

$x=0$은 항상 해. 이차식 $h(x)=x^2-sx+(p+1)$의 판별식:

$$D_h=s^2-4(p+1)=(\beta-\gamma)^2-4=d^2-4$$

서로 다른 두 교점 $\;\Longleftrightarrow\;$ $h$가 이중근($\neq 0$)을 가짐 $\;\Longleftrightarrow\;$ $D_h=0$:

$$d^2=4,\quad d=2$$

이중근 $x=s/2$. ($s\neq 0$이므로 $s/2\neq 0$ ✓)

**효과.** $d=2$ 확정. **자유도 $1\to$** $s$의 부호만 미결.

### 전제 분석: "$0$이 아닌 두 근은 모두 양수"

| 경우 | $s$ | $d$ | $\beta=\frac{s-d}{2}$ | $\gamma=\frac{s+d}{2}$ | 양수 여부 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| A | $6$ | $2$ | $2$ | $4$ | **양수** ✓ |
| B | $-6$ | $2$ | $-4$ | $-2$ | **음수** ✗ |

양수 전제가 경우 B를 제거하여 **유일해 확정**. **자유도 $0$.**

### 조건별 기여도 검증

| 제거 조건 | 결과 | 답 변동? |
|:---:|---|:---:|
| 양수 전제 | $s=\pm 6$ 모두 가능 → $f(1)=3$ 또는 $15$ | **예** |
| (가) | $s$ 미결 → $f(1)$ 미결 | **예** |
| (나) | $d$ 미결 → $\beta,\gamma$ 미결 | **예** |

**모든 전제/조건이 해 결정에 실질적으로 기여. 잉여 조건 없음.** ✅

# STEP 2. 수치 결정

$$s=6,\;d=2\;\;\Longrightarrow\;\;\beta=2,\;\gamma=4$$

$$f(x)=x(x-2)(x-4)=x^3-6x^2+8x$$

### 전수 검증

**(가)** $f'(x)=3x^2-12x+8$, $\;f'(0)=8$. 접선 $\ell:\;y=8x$.

$g(x)=x^2(x-6)$. 극솟값 $g(4)=16\cdot(-2)=-32$이므로 $k=-32$.

$$x^2(x-6)+32=(x+2)(x-4)^2=0$$

두 교점: $x_1=-2$, $x_2=4$.

$$x_1\cdot x_2=(-2)(4)=-8\;\checkmark$$

접선 $y=8x-32$ 위의 두 점: $(-2,\,-48)$과 $(4,\,0)$ — **서로 다른 두 점** ✓

**(나)** $f(x)+x=x(x^2-6x+9)=x(x-3)^2=0$

교점: $(0,\,0)$과 $(3,\,-3)$ — **서로 다른 두 점** ✓

($x=3$에서 $f'(3)=27-36+8=-1=$ 직선 기울기 → 접점)

**경우 B ($s=-6$) 검증:**

$\beta=-4,\;\gamma=-2$. $f(x)=x(x+4)(x+2)=x^3+6x^2+8x$.

$g(x)=x^2(x+6)$, $k=-4(-6)^3/27=32$. $x^2(x+6)-32=(x-2)(x+4)^2=0$.

교점: $x_1=2$, $x_2=-4$. 곱 $=2\cdot(-4)=-8$ ✓. **조건 (가) 충족.**

$f(x)+x=x(x^2+6x+9)=x(x+3)^2=0$. 교점: $(0,0)$, $(-3,3)$ → 2개 ✓. **조건 (나) 충족.**

$f(1)=1\cdot 5\cdot 3=15\neq 3$. **양수 전제 없이는 답 미확정.** ✅

**발문:**

$$f(1)=(1)(1-2)(1-4)=(1)(-1)(-3)=\boxed{3}$$
