### [STEP 1] 독립 풀이

#### 1-1. 조건별 독립 전개 + 자유도 분석

**조건 (가) 단독:**

$f'(x) = 3x^2 + a = 0$이 서로 다른 두 실근을 가져야 함.
$$x^2 = -\frac{a}{3} \quad \Longrightarrow \quad a < 0$$
극대점 $\alpha = -\sqrt{-a/3}$, 극소점 $\beta = \sqrt{-a/3}$.

- 결정되는 파라미터: $a < 0$ (부등식)
- 미결 파라미터: $a$의 구체적 값, $b$
- **자유도: 2**

---

**조건 (나) 단독:**

$y = f(2)$는 수평선. $f(2) = 8 + 2a + b$.
$$f(x) - f(2) = x^3 + ax + b - (8 + 2a + b) = x^3 - 8 + a(x - 2) = (x-2)(x^2 + 2x + 4 + a)$$

$x \neq 2$인 점에서 **접한다** = $x^2 + 2x + 4 + a = 0$이 중근을 가짐:
$$\Delta = 4 - 4(4+a) = -12 - 4a = 0 \quad \Longrightarrow \quad a = -3$$
중근 $t = -\frac{2}{2} = -1$ (접점).

검증: $f'(-1) = 3(-1)^2 + (-3) = 0$ ✓ (수평접선)

- 결정되는 파라미터: $a = -3$
- 미결 파라미터: $b$
- **자유도: 1**

---

**조건 (다) 단독:**

$f(0) = b > 0$이므로 $|f(x)| = b$.
이는 $f(x) = b$ 또는 $f(x) = -b$.

**$f(x) = b$:** $x^3 + ax + b = b \Rightarrow x(x^2 + a) = 0$.
$a < 0$이면 근 $x = 0, \pm\sqrt{-a}$으로 **3개**.

**$f(x) = -b$:** $x^3 + ax + 2b = 0$.
총 5근이려면 이 방정식이 **정확히 2개의 서로 다른 실근**을 가져야 함(3개 근과 겹치지 않을 때).
삼차방정식이 2개의 서로 다른 실근 → 중근 존재 → 판별식 $= 0$:
$$-4a^3 - 27(2b)^2 = 0 \quad \Longrightarrow \quad -4a^3 - 108b^2 = 0 \quad \Longrightarrow \quad b^2 = -\frac{a^3}{27}$$

- 결정되는 파라미터: $b^2 = -a^3/27$ (등식 1개), $a < 0$ (부등식)
- 미결 파라미터: $a$ (하나의 등식에 두 미지수)
- **자유도: 1**

#### 1-2. 얽힘 검증

| 검증 항목 | 결과 | 근거 |
|----------|------|------|
| (가)의 미결 파라미터가 (나)의 등식에 등장하는가 | O | $a$가 (나)의 $\Delta = -12-4a=0$에 등장 |
| (나)의 미결 파라미터가 (다)의 등식에 등장하는가 | O | $b$가 (다)의 $b^2 = -a^3/27$에 등장 |
| (가)+(나)의 자유도가 각 단독보다 2 이상 줄어드는가 | **X** | (가): 2, (나): 1, (가)+(나): 1. 감소폭은 (가)에서 1, (나)에서 0 |
| 연립 과정에서 단독 전개에 없던 새로운 등식이 나타나는가 | **X** | (나)→$a=-3$ 대입 → $b^2=1$은 단순 치환이며 구조적 신규 등식 아님 |

**얽힘 등급: 2개 O → 약한 얽힘 또는 독립 구조**

본질적으로 **(나)가 $a$를 완전 결정 → (다)가 $b$를 완전 결정**하는 **순차적 체인 구조**.

#### 1-3. 조건 연립 + 풀이

**연립 순서:** (가) → (나) → (다) (자연스러운 순차 진행)

**Step 1 [(가)]:** $a < 0$.

**Step 2 [(나)]:**
$$f(x) - f(2) = (x-2)(x^2 + 2x + 4 + a)$$
$x \neq 2$에서 접선 조건: $x^2+2x+4+a=0$의 판별식 $=0$
$$a = -3, \quad t = -1$$

**Step 3 [(다)]:**
$f(x) = x^3 - 3x + b$, $f(0) = b$.

$f(x) = b$: $x(x^2-3) = 0$ → 근 $0, \pm\sqrt{3}$ (3개)

$f(x) = -b$: $h(x) = x^3 - 3x + 2b = 0$.
$h'(x) = 3x^2 - 3 = 0 \Rightarrow x = \pm 1$.
- 극소 $h(1) = 2b - 2$
- 극대 $h(-1) = 2b + 2$

정확히 2근 $\Rightarrow$ 극솟값 $= 0$:
$$2b - 2 = 0 \quad \Longrightarrow \quad b = 1$$
($h(-1) = 4 > 0$이므로 극대 $> 0$, 극소 $= 0$: 중근 + 단근 = 2개 실근 ✓)

$h(x) = x^3 - 3x + 2 = (x-1)^2(x+2)$: 근 $x=1$(중근), $x=-2$

겹침 확인: $\{1, -2\} \cap \{0, \sqrt{3}, -\sqrt{3}\} = \varnothing$ ✓ → 총 5근 ✓

$$f(2) = 8 - 6 + 1 = \boxed{3}$$

**전환 지점:**

- **전환 1:** (나)에서 $f(x)-f(2)$를 $(x-2)$로 인수분해하여 접선 조건을 이차식의 판별식 문제로 변환. [직접적인 연립이 아닌 다항식 구조 활용]
- **전환 2:** (다)에서 $|f(x)|=b$를 두 방정식으로 분리한 뒤, 5근 조건을 "$f(x)=-b$가 정확히 2근"으로 번역하고, 이를 극솟값$=0$ 조건으로 전환. [근의 개수 → 접선/극값 기하 조건]

- $a = -3$: (나)에서 유일 결정
- $b = 1$: (다)에서 $b>0$ 조건으로 $b=-1$ 배제 → 유일
- $f(2) = 3$: 유일

**→ O (유일해)**

#### 1-5. 조건 과잉/부족 체크

- **조건 (가) 제거:** (나)의 접선 조건에서 $x^2+2x+4+a=0$이 중근 → $a=-3 < 0$ → 자동으로 극대·극소 존재. **(가) 없이도 답 동일. (가)는 과잉.**

- **조건 (나) 제거:** $a < 0$, $b^2 = -a^3/27$, $b>0$만 남음. $a$가 미결정 → 답 부정. **필요.**

- **조건 (다) 제거:** $a = -3$이지만 $b$ 미결정 → 답 부정. **필요.**

### [STEP 2] 설계 의도 대조

설계 정보에 따르면:
1. (나) → $a = -3k^2/4$ (극값을 활용한 Vieta 분석)
2. 막힘: $b$ 결정 불가
3. (다) → $b^2 = -a^3/27$ 연립 → $b = 1$
4. 숨은 사실: $-f(0) = $ 극솟값

**설계 의도의 "강제 결과/숨은 사실"이 STEP 1-3에서 나타났는가?** → **O**

나의 풀이 전환 지점 2에서: $h(1) = 2b-2 = 0$, 즉 $f(1) = b-2$. $b=1$일 때 $f(1) = -1 = -b = -f(0)$. 극솟값 $= -f(0)$이라는 사실이 정확히 나타남.

그러나 이 "발견"은 풀이의 **필수적 난관**이 아니라, $h(1) = 0$이라는 단순 계산의 **해석**에 불과. 극솟값$=0$ 조건을 세우는 것은 삼차 근 개수 분석의 표준 기법.

**핵심 차이:** 설계는 (나)+(다)의 연립에서 진정한 얽힘을 의도했지만, 실제로 (나)가 $a$를 완전히 결정하므로 (다)는 $a=-3$이 주어진 상태에서 $b$만 결정하는 독립적 역할. 숨은 사실은 풀이에 나타나지만 **풀이를 전환시키는 핵심 역할**이라기보다 계산 결과의 부산물.
