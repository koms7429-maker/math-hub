## 이벨류에이터

### 원본 문제 A
## Problem

**4.**

세 상수 $a(a \neq 0)$, $b$, $c$에 대하여 함수

$$f(x) = ax^3 + bx^2 - cx + c$$

가 다음 조건을 만족시킬 때, $f(3)$의 값을 구하시오.

> (가) $f(1) = 0$, $f'(1) = 1$
> (나) $f(k) = 0$, $f'(k) = -\dfrac{f(-1)}{2}$인 실수 $k(k \neq 1)$이
> 존재한다.

### 원본 문제 B
## Problem

**13.**

$0 < a < 6 < b$인 두 상수 $a$, $b$와 최고차항의 계수가 $-1$인 삼차함수 $f(x)$에 대하여 $f(0) = 20$이고 부등식

$$f(a) \leq f(x) \leq f(6)$$

의 해가 $0 \leq x \leq b$일 때, $f\left(\frac{a+b}{2}\right)$의 값을 구하시오.
