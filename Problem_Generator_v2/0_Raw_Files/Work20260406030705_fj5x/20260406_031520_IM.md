따라서 f(x) = (x − x₀)²(x − r) (삼차, 최고차 계수 1).

- **경우 ①**: 이중근이 x = 1 자체 → f = (x−1)²(x−r), f(1) = 0 자동 충족. 자유 파라미터: r.
- **경우 ②**: 이중근이 x₀ ≠ 1 → f = (x−x₀)²(x−1), f(1) = 0 충족. 자유 파라미터: x₀.

어느 경우든 a = −(2x₀ + r) 또는 a = −(2x₀ + 1)이 x₀ 또는 r에 종속. **a는 미결**.

- 결정: b, c가 하나의 자유 파라미터(r 또는 x₀)로 표현됨
- 미결: a (자유 파라미터 1개에 의존)
- **자유도 = 1**

**제약 A+B 동시**:

A에서 a = 1. 이를 B의 두 경우에 대입:

- **경우 ①**: f = (x−1)²(x−r). a = −(2·1 + r) = −(2+r) = 1이므로 **r = −3**.
  극소점 x₀ = 1에서 f″(1) = 6(1) + 2(1) = 8 > 0. ✓ 극소 확인.
  → f(x) = (x−1)²(x+3) = x³ + x² − 5x + 3.

- **경우 ②**: f = (x−x₀)²(x−1). a = −(2x₀+1) = 1이므로 **x₀ = −1**.
  극소점 x₀ = −1에서 f″(−1) = 6(−1) + 2(1) = −4 < 0. ✗ **극대**이므로 탈락!

유일한 해: **f(x) = (x−1)²(x+3), b = −5, c = 3. 자유도 = 0.**
