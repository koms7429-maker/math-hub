## 아이디어 셀렉터

### 수학적 상황
- **수학적 객체**: 최고차항의 계수가 1인 사차함수 f(x)

- **제약 A가 부과하는 것**: f(−1) = 0이고 f(3) = 0이다. 이는 f(x) = (x+1)(x−3)Q(x) (Q는 이차식)로의 인수분해를 허용하지만, 이차인수 Q(x) = x² + ex + g의 두 계수 e, g가 미결이다. **자유도 2.**

- **제약 B가 부과하는 것**: f′(1) = 0이고 f(1) = −16이다. 이는 x = 1에서 수평접선을 가지며 그 접선의 y좌표가 −16임을 뜻한다. 사차함수의 4개 계수 중 c, d를 a, b로 표현하지만, a와 b는 미결이다. **자유도 2.**

- **공유 파라미터**: a와 b (또는 동치적으로, 이차인수 Q의 계수 e와 g). 제약 A는 (c, d)를 (a, b)의 일차식으로 결정하고, 제약 B도 (c, d)를 (a, b)의 **다른** 일차식으로 결정한다. 두 결정이 양립하려면 a와 b에 대한 연립일차방정식이 성립해야 한다.

- **강제 결과**: A+B를 동시에 적용하면 a = −4, b = 6, c = −4, d = −15가 유일하게 결정되어 **f(x) = (x−1)⁴ − 16**이 된다. 두 근 −1과 3은 x = 1에 대해 대칭이고, 나머지 이차인수 x² − 2x + 5 = (x−1)² + 4도 x = 1에 대해 대칭이므로, f(x) 전체가 x = 1을 축으로 하는 완전한 짝함수 구조(평행이동된 네제곱)를 가진다. 이는 어느 제약 단독으로도 예측할 수 없는 결과이다.

### 수험생의 발견 순간
1. **첫 시도**: 제약 A를 전개한다. f(−1) = 0, f(3) = 0으로부터 c = −20 − 7a − 2b, d = −21 − 6a − 3b를 얻는다. 또는 f(x) = (x+1)(x−3)(x² + ex + g)로 놓고 전개하여 a = e−3, b = g−3e, c = −3g, d = 0… 이 아니라 상수항 확인이 필요하다. 어느 쪽이든 자유 파라미터 2개가 남는다.

2. **막히는 순간**: "a, b가 자유라서 f가 하나로 결정되지 않는다. 극점 조건을 써야 하는데, f′(1) = 0이 a와 b를 어떻게 동시에 잡아줄 수 있지?"

3. **제약 B 적용**: f′(1) = 4 + 3a + 2b + c에 제약 A의 결과 c = −20 − 7a − 2b를 대입한다.
   f′(1) = 4 + 3a + 2b + (−20 − 7a − 2b) = **−16 − 4a**.
   여기서 결정적 발견: **b가 소거된다!** f′(중점)을 계산할 때 b의 기여가 정확히 상쇄되어, f′(1) = 0은 오직 **a = −4**만을 강제한다.

4. **발견**: "b가 사라진 이유는 x = 1이 두 근 −1, 3의 정확한 중점이기 때문이다. 중점에서 (x+1)(x−3)의 기울기 기여가 대칭적으로 상쇄되어, 도함수가 이차인수 Q의 정보만 남긴다. a = −4를 얻은 뒤 f(1) = −16에 대입하면 b = 6이 즉시 결정되고, **f(x) = (x−1)⁴ − 16**이라는 완전네제곱 구조가 드러난다. 네 조건이 모두 이 하나의 대칭 형태를 가리키고 있었다!"
