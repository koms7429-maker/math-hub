## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: 원점을 지나는 삼차함수 $f(x) = x^3 + ax^2 + bx$와 원점을 지나는 직선 $y = mx$.

**사실 A가 부과하는 제약 (P026)**:
모든 실수 $x$에서 $x \cdot (f(x) - mx) \geq 0$이 성립해야 한다. 부호인수 $x$에 의해 양의 영역($x > 0$)에서는 $f(x) \geq mx$, 음의 영역($x < 0$)에서는 $f(x) \leq mx$라는 이중 경계가 동시에 부과된다. $f(x) - mx = x(x^2 + ax + (b-m))$이므로, 이 조건은 이차식 $x^2 + ax + (b-m)$이 항상 비음수일 것을 요구하며, 이는 판별식 부등식 $a^2 \leq 4(b-m)$으로 환원된다.

**사실 B가 부과하는 제약 (P010)**:
원점을 지나는 직선이 삼차곡선에 접하는 조건은 $f(t) = mt$와 $f'(t) = m$의 연립이며, 소거하면 $f(t) - tf'(t) = -t^2(2t+a) = 0$. 비자명 접점은 $t = -a/2$이고, 접선 기울기는 $m = b - a^2/4$이다. 이 접점은 삼차함수의 변곡점($x = -a/3$)과 원점 사이에서 고정된 비율로 위치한다.

**숨은 사실 (A+B 동시 성립 시 강제되는 것)**:
부등식의 임계 기울기(판별식 $= 0$이 되는 $m$의 최댓값)와 접선의 기울기가 모두 $b - a^2/4$로 일치한다. 따라서 **부등식 $x(f(x)-mx) \geq 0$이 모든 $x$에서 성립하는 가장 가파른 직선 $y = mx$는, 원점에서 삼차곡선으로 그은 접선 그 자체이다.** 등호가 성립하는 점 $x = -a/2$는 접점이며, 변곡점 $x = -a/3$은 원점과 이 접점을 항상 $1:2$로 내분한다 — 이 기하적 관계는 매개변수 $b$에 전혀 의존하지 않는다.

---

### 수험생의 발견 순간
1. **처음 시도**: $x(f(x) - mx) \geq 0$을 정리하여 $x^2(x^2 + ax + (b-m)) \geq 0$을 얻고, "$x^2 \geq 0$이니까 이차식만 비음수이면 되겠구나"라고 판별식 $D = a^2 - 4(b-m) \leq 0$을 세워 $m$의 범위를 구한다.

2. **막히는 순간**: 최대 $m = b - a^2/4$를 구했지만, 문제가 이 값과 삼차함수의 기하적 성질(접선, 변곡점 등)을 연결하는 후속 조건을 제시한다. 판별식에서 나온 대수적 값 $b - a^2/4$가 기하적으로 무엇을 의미하는지 알 수 없어 막힌다.

3. **발견**: "$b - a^2/4$가 혹시 어디서 본 값 아닌가?" → 원점에서 곡선에 접선을 그어 보면, 접점 $t = -a/2$에서의 기울기 $f'(-a/2) = 3(a/2)^2 - 2a(a/2) + b = b - a^2/4$. **"판별식의 등호 조건이 곧 접선 조건이었다!"** 그리고 등호에서 $x^2 + ax + (b-m) = (x + a/2)^2$이 되어, 이중근 $x = -a/2$이 바로 접점임을 확인한다.

4. **대응**: 이 발견은 STEP 2-2의 숨은 사실 — "부등식의 경계 직선 = 원점에서의 접선" — 과 정확히 대응한다. 수험생은 부등식의 대수적 처리(부호 분리 → 판별식)와 미분 기하적 해석(접선 기울기 → 접점 위치)이 동일한 결과로 합류하는 순간을 경험하며, 이로부터 변곡점의 내분 비율 등 후속 계산이 자연스럽게 열린다.

## 2. 프라블럼 메이커 (Problem Maker)

### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$와 실수 $m$이 다음 조건을 만족시킨다.
>
> **(가)** $f(0) = 0$
>
> **(나)** 모든 실수 $x$에 대하여 $\; x\!\left\{f(x) - mx\right\} \ge 0\;$이다.
>
> **(다)** 곡선 $y = f(x)$와 직선 $y = mx$는 원점이 아닌 점 $\mathrm{P}$에서 만난다.
>
> **(라)** 곡선 $y = f(x)$의 변곡점의 $x$좌표가 $\; p \; $이고 $\; f(p) = \alpha\;$이다.
>
> 점 $\mathrm{P}$의 $x$좌표를 $k$라 할 때, $\; f(k)$의 값을 구하시오.
>
> *(수치 $\alpha,\; p$ 등은 수치 확정 단계에서 결정)*

### 발문 변형 옵션 (수치 확정 시 선택)

| 변형 | 발문 | 비고 |
|---|---|---|
| A | $m$의 값을 구하시오. | (라)에서 $a, b$를 결정한 뒤 $m = b - a^2/4$ 계산 |
| B | $f(k)$의 값을 구하시오. | 접점의 함수값을 묻는 형태 |
| C | $k/p$의 값을 구하시오. | 매개변수 무관 불변량 $\dfrac{3}{2}$ — (라) 없이도 출제 가능 |

| **(라)** 변곡점 좌표 조건 | 수치 확정용 보조 조건 | $a, b$ 결정에 사용. 내분 비율 $1{:}2$로의 연결 다리 |
