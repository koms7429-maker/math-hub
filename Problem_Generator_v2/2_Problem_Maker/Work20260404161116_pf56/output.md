## 프라블럼 메이커

### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** $f(0) = 0$이고 $f'(0) = 9$이다.
>
> **(나)** $x \geq 0$인 모든 실수 $x$에 대하여 $f(x) \geq 0$이다.
>
> **(다)** $f(t) = 0$인 양의 실수 $t$가 존재한다.
>
> $f(5)$의 값을 구하시오.

---

##

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 역할 |
|---|---|---|
| **(가)** $f(0)=0$이고 $f'(0)=9$ | 제약 A 전체 | $c=0$, $b=9$ 확정. $f(x)=x(x^2+ax+9)$로 축소 |
| **(나)** $x\ge0$이면 $f(x)\ge0$ | 제약 B의 $f'(t)=0$ 부분을 **부등식으로 포장** | $x^2+ax+9\ge0\;(x\ge0)$을 강제. 단독으로는 $a$를 유일하게 결정 못 함 |
| **(다)** $f(t)=0$인 양의 실수 $t$ 존재 | 제약 B의 $f(t)=0$ 부분 | (나)와 합쳐져 "비음수이면서 양의 영점" → 이중근 → $a=-6$, $t=3$ 강제 |
