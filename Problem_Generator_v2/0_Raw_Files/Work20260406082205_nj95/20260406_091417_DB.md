### 1-2. 분기 분석

**"$0$이 아닌 중근"의 해석 분기:**

| 경우 | $x(x^2+ax+b) = 0$의 근 구조 | (가) 충족? |
|---|---|---|
| $x^2+ax+b$가 이중근 $r = -a/2 \neq 0$ | 근: $0,\;r,\;r$ | ✓ — 이중근 $r \neq 0$ |
| $x^2+ax+b$가 근 $0$을 포함 ($b=0$) | $x^2(x+a)=0$, 이중근은 $0$ | ✗ — 이중근이 $0$ |
| $x^2+ax+b$가 서로 다른 두 비영근 | 중근 없음 | ✗ |

**결론: "$0$이 아닌 중근" 조건은 $a^2 - 4b = 0$, $a \neq 0$을 유일하게 강제합니다.**

> **판정 1-2**: 분기가 유일하게 결정됨 → **CONTINUE**

### 1-3. 자유도 계산

| 단계 | 자유 파라미터 | 조건 | 잔여 자유도 |
|---|---|---|---|
| 초기 | $(a, b, c)$ | — | 3 |
| (나) 이중근 | — | $b = a^2/4$, $a \neq 0$ | 2 |
| (다) 고정점 | — | $c^2 + ac + b = 0$ → $c^2 + ac + a^2/4 = 0$ → $(c + a/2)^2 = 0$ → $c = -a/2$ | **1** |
| (라) 수치 | — | $c = -N$ (구체값) → $a = 2N$ | **0** |

**핵심 발견**: (나)에서 $b = a^2/4$를 대입하면 (다)의 이차방정식 $c^2 + ac + b = 0$의 판별식이 $a^2 - 4b = 0$으로 정확히 소멸하여 $c = -a/2$가 유일 결정됩니다. 이는 어느 조건에서도 직접 읽을 수 없고 **연립 과정에서만 나타나는 숨은 사실**입니다.

> **판정 1-3**: 자유도가 조건 (라)의 수치 하나로 0에 도달 → **CONTINUE**

### 1-4. 탐색 전략

- 조건 (가) "극댓값·극솟값 존재"의 필요성 검증 → $a = 2N \neq 0$일 때 $f'$의 판별식 $= 4a^2 - 12b = 4(2N)^2 - 12N^2 = 4N^2 > 0$이므로 **자동 충족** → **(가)는 불필요, 제거**
- (라)의 형태: $f(0) = -N$ (직접), 또는 간접 수치 조건 검토
- 발문: $f(M) = M(M+N)^2 - N$ 형태에서 정수 답이 나오는 $(N, M)$ 탐색

> **판정 1-4**: 정수화 가능, 전략 수립 완료 → **CONTINUE**

### 후보 생성

확정된 함수: $f(x) = x^3 + 2Nx^2 + N^2 x - N = x(x+N)^2 - N$

| 후보 | $N$ | $M$ | $f(M) = M(M+N)^2 - N$ | 비고 |
|---|---|---|---|---|
| ① | $2$ | $5$ | $5 \times 49 - 2 = 243$ | $243 = 3^5$, 깔끔 |
| ② | $3$ | $3$ | $3 \times 36 - 3 = 105$ | 무난 |

| ④ | $4$ | $6$ | $6 \times 100 - 4 = 596$ | 큰 수 |

### 후보 평가

**후보 ①** ($N=2$, $M=5$, 답 $243$):
- 손계산: $f(5) = 125 + 100 + 20 - 2 = 243$ — 4단계 덧셈, 매우 간결
- 유일성: $(a-4)^2 = 0$ → $a=4$ 유일 ✓
- 정수성: 모든 계수 정수, 답 정수 ✓
- 난이도: 풀이 과정에 완전제곱식 등장 → 적절한 깊이

**후보 ② ③ ④**: 답이 덜 깔끔하거나 계산 부담이 큼. 패스.

### **채택: 후보 ①** ($N=2$, $M=5$)

### 풀이 경로 검증

$$c = -2 \quad\text{(라)}$$

$$b = \tfrac{a^2}{4} \quad\text{(가): 이중근 조건}$$

$$f(-2) = -2 \;\text{(나)}: \quad -8 + 4a - 2b - 2 = -2 \;\Rightarrow\; 2a - b = 4$$

$$2a - \tfrac{a^2}{4} = 4 \;\Rightarrow\; a^2 - 8a + 16 = 0 \;\Rightarrow\; (a-4)^2 = 0 \;\Rightarrow\; a = 4$$

$$\therefore\; f(x) = x^3 + 4x^2 + 4x - 2 = x(x+2)^2 - 2$$

$$f(5) = 5 \cdot 7^2 - 2 = 245 - 2 = \boxed{243}$$
