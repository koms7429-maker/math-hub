## [2] 각 아이디어가 만드는 계수 관계식 (기호 수준)

**함수 설정**: $f(x) = (x - r_1)(x - r_2)(x - r_3)$, 모닉 삼차다항식, 세 실근 $r_1, r_2, r_3$ (모두 다름).

### 관계식 R1 — P004로부터

> "삼차 $f$의 두 서로 다른 근 $r_1, r_2$에서 $f'(r_1) = f'(r_2)$가 성립한다."

$f'$는 이차식이므로 $f'(r_1) = f'(r_2)$이면 $r_1 + r_2$는 $f'$의 대칭축의 2배, 즉

$$r_1 + r_2 = \frac{2(r_1 + r_2 + r_3)}{3}$$

정리하면

$$\boxed{r_3 = \frac{r_1 + r_2}{2}}$$

즉 **제3근 $r_3$가 $r_1$과 $r_2$의 중점 = 변곡점의 $x$좌표**에 위치한다. $r_1 = r_3 - d$, $r_2 = r_3 + d$로 놓으면 $f$는 변곡점 $r_3$ 중심의 **대칭 1-파라미터족**으로 환원된다:

$$f(x) = (x - r_3)^3 - d^2(x - r_3) \quad (t = x - r_3 \text{로 치환하면 } g(t) = t^3 - d^2 t)$$

R1은 **근의 배치를 $(r_3, d)$ 2-파라미터로 축약**하되, $r_3$와 $d$의 값 자체는 결정하지 못한다.

### 관계식 R2 — P001로부터

> "$f(x - f(x)) = 0$의 실근이 정확히 $N$개이다."

$f(t) = 0$의 근이 $r_1, r_2, r_3$이므로, 방정식은 세 하위 문제로 분해된다:

$$f(x) = x - r_i \quad (i = 1, 2, 3)$$

각각은 기울기 1인 직선 $y = x - r_i$와 삼차 곡선 $y = f(x)$의 교점 문제이다. 총 실근 수 $N$은 세 하위 문제의 실근 수의 합이며, 각 하위 문제의 근의 수는 **직선이 삼차곡선의 극값 수준에 대해 어디에 위치하느냐**(임계값과의 비교)에 의존한다.

$t = x - r_3$로 치환하면:

| 분기 | 방정식 | 비고 |
|---|---|---|
| $i = 3$ | $t^3 - (d^2+1)t = 0$ | 항상 3실근: $t = 0, \pm\sqrt{d^2+1}$ |
| $i = 1$ | $t^3 - (d^2+1)t - d = 0$ | $g(t) = d$ 형태 |
| $i = 2$ | $t^3 - (d^2+1)t + d = 0$ | $g(t) = -d$ 형태, $g$가 기함수이므로 $i=1$의 근의 부호 반전 |

여기서 $g(t) = t^3 - (d^2+1)t$의 극값은 $t = \pm\sqrt{(d^2+1)/3}$에서

$$g_{\text{극대}} = \frac{2}{3}\left(\frac{d^2+1}{3}\right)^{3/2}, \quad g_{\text{극소}} = -\frac{2}{3}\left(\frac{d^2+1}{3}\right)^{3/2}$$

$i=1$ 분기가 **이중근(접선 조건)**을 가지려면:

$$\boxed{d = \frac{2}{3}\left(\frac{d^2+1}{3}\right)^{3/2}} \quad \cdots \text{(R2: 접선 임계 방정식)}$$

이 방정식은 **$d$ 하나에 대한 비자명한 대수 방정식**으로, $g$의 기함수 대칭에 의해 $i=2$ 분기도 동시에 이중근을 갖는다.

R2는 **$d$에 대한 방정식을 부여**하되, R1 없이는 삼차함수의 근 배치가 대칭 구조인지 알 수 없으므로 이 형태로 환원되지 않는다.

## [3] 파라미터 공유 확인

| | R1이 구속하는 것 | R2가 구속하는 것 |
|---|---|---|
| 파라미터 $d$ (근 간격) | 구속하지 않음 — $d$를 자유 파라미터로 남김 | $d$에 대한 비자명한 방정식 제공 |
| 파라미터 $r_3$ (변곡점 위치) | 구속하지 않음 — $r_3$를 자유 파라미터로 남김 | 구속하지 않음 — $r_3$가 치환으로 소거됨 |
| 근의 배치 구조 | **대칭 2-파라미터족 $(r_3, d)$로 환원** | 이 대칭 구조를 **전제**해야 $d$의 단일 방정식으로 환원 가능 |

**공유 파라미터**: $d$ (근 간격 파라미터)

- R1 단독: $d$를 결정하지 못하나, $d$가 의미를 갖는 **구조를 생성**
- R2 단독: 원래 삼차의 자유도가 $(r_1, r_2, r_3)$ 3개여서 접선 조건 하나로는 부족
- **R1 + R2 연립**: R1이 자유도를 $(r_3, d)$로 축소 → R2가 $d$를 결정 → $r_3$만 잔여 자유도로 남음 (제3조건이 확정)

핵심: **R1이 없으면 R2는 3-변수 접선 조건이 되어 닫히지 않고, R2가 없으면 R1은 구조만 주고 값을 결정하지 못한다.** 두 조건이 $d$라는 공유 파라미터 위에서 **비로소 교차**한다.

## [4] 수학적 상황 서술

모닉 삼차다항식 $f$가 세 서로 다른 실근을 갖고, 다음 두 조건이 동시에 부여된다:

1. **(P004 조건)** $f$의 두 서로 다른 근에서 도함수값이 같다.
2. **(P001 조건)** $f(x - f(x)) = 0$의 실근의 총 개수가 특정 값 $N$이다.

**조건 1만 적용했을 때**: 두 근의 중점이 $f'$의 대칭축과 일치해야 하므로, 제3근이 정확히 변곡점에 놓인다는 **구조적 사실**이 드러난다. 이로써 $f$는 변곡점 중심 좌표로 이동하면 $g(t) = t^3 - d^2 t$ (기함수)로 정규화되어, 3-파라미터 → 2-파라미터 $(r_3, d)$로 축소된다.

**조건 2만 적용했을 때**: 자기합성 방정식이 세 분기로 분해되지만, 일반 삼차의 세 근 $r_1, r_2, r_3$의 상대 위치가 자유로워, 각 분기의 근의 수를 제어하는 데 변수가 너무 많다.

**두 조건의 연립**: 조건 1에 의해 $f$가 기함수 $g(t) = t^3 - d^2 t$로 환원되면, 세 분기가 다음과 같이 **극적으로 단순화**된다:

- **$i=3$ 분기**: $t(t^2 - (d^2+1)) = 0$ — 항상 정확히 3실근.
- **$i=1, 2$ 분기**: $g(t) = \pm d$ — 기함수 대칭에 의해 두 분기의 근 구조가 **완전히 거울상**. 하나가 3실근이면 다른 것도 3실근, 하나가 접선(이중근+단근)이면 다른 것도 접선.

따라서 총 근의 수 $N$의 가능한 값은:

| $i=1, 2$ 각 분기 근 수 | 총 $N$ |
|---|---|
| 각 3개 | $3 + 3 + 3 = 9$ |
| 각 2개 (접선, 이중근) | $3 + 2 + 2 = 7$ |
| 각 1개 | $3 + 1 + 1 = 5$ |

$N = 7$을 요구하면 $i=1, 2$ 분기 **각각에서 접선 조건**(이중근)이 강제되고, 이것이 $d$에 대한 방정식 $d = \frac{2}{3}\!\left(\frac{d^2+1}{3}\right)^{3/2}$를 생산한다.

**드러나는 구조적 조건**: 이 방정식을 정리하면 $d^2$에 대한 유한 차수 다항 방정식으로 환원되며($d > 0$으로 제한하면), $d$의 값이 유한 개의 이산적 후보로 결정된다. 이것은 원래 삼차함수의 **근 간격이 단 한 가지(또는 유한 개)**로 확정됨을 의미한다.

잔여 자유도 $r_3$(변곡점 위치 = 평행이동)는 제3조건(예: 특정 점 통과, 특정 근의 값)으로 확정하면 $f$가 유일하게 결정된다.

## [5] 수험생의 발견 지점

수험생은 다음 두 순간에 예상치 못한 구조를 발견한다:

**첫 번째 발견 — "제3근이 변곡점이다"**:
두 근에서 도함수값이 같다는 조건(P004)을 $f'(r_1) = f'(r_2)$로 풀면, $r_1 + r_2$가 $f'$의 대칭축에 의해 결정되고, 비에타 공식과 비교하는 순간 **제3근 $r_3$가 변곡점의 $x$좌표와 정확히 일치**한다는 사실을 발견한다. 이것은 조건에 명시되지 않은 숨겨진 결과로, $f$를 변곡점 중심의 **기함수로 정규화**할 수 있게 하는 열쇠이다.

**두 번째 발견 — "기함수 대칭이 거울상 분기를 만든다"**:
자기합성 $f(x - f(x)) = 0$을 세 분기로 분해한 뒤, 첫 번째 발견에 의해 정규화된 $g(t) = t^3 - d^2 t$가 기함수임을 이용하면, $i=1$과 $i=2$ 분기가 $g(t) = d$와 $g(t) = -d$로 **정확한 거울상**이 되어 근의 수가 동시에 같아진다는 것을 발견한다. 이 대칭에 의해 $N = 7$이라는 조건이 "두 분기 각각에서 접선"이라는 **단일 사건**으로 귀결되고, $d$에 대한 하나의 방정식으로 닫히는 순간을 경험한다. 이 두 번째 발견은 첫 번째 발견(기함수 구조) 없이는 도달할 수 없으므로, **두 아이디어의 연동이 필수적**이다.

## [무대 설정]

최고차항의 계수가 1이고 서로 다른 세 실근을 갖는 삼차함수 $f(x)$, 세 근을 크기순으로 $\alpha < \beta < \gamma$로 명명. 파라미터: 변곡점 위치 $r_3$, 근 간격 $d$.

## [조건 배치 계획]

| 수학적 상황 요소 | 번역 대상 조건 | 설계 의도 |
|---|---|---|
| P004: $f'(\alpha) = f'(\gamma)$ | **(가)** | 무해한 접선 조건처럼 보이나, 처리하면 $\beta$가 변곡점 = $(\alpha+\gamma)/2$라는 숨겨진 대칭 구조가 드러남. **"제3근이 변곡점"이라는 사실을 직접 알려주지 않음.** |
| P001: $f(x-f(x))=0$의 실근 수 $= N$ | **(나)** | (가)와 완전히 독립적으로 보이는 합성방정식 조건. 단독으로는 3변수 문제라 손댈 수 없으나, (가)의 대칭 구조를 대입하면 단일 파라미터 $d$의 방정식으로 환원됨. **두 조건의 교차점에서 $d$가 결정됨.** |
| 잔여 자유도 $r_3$ 고정 | **(다)** | $\alpha + \gamma = s$ 형태로 제시. 표면적으로는 양 끝 근의 합이라는 별개 정보이나, (가)에서 발견한 $\beta = (\alpha+\gamma)/2$와 결합하면 변곡점 위치 $r_3 = s/2$가 확정됨. **단독으로는 무해, (가)의 결과와 합쳐질 때만 앵커 역할.** |

**충돌 은닉 확인:**
- (가)만 보면: 양 끝 근에서 접선 기울기가 같다는 단순 조건 → 학생은 $f'(\alpha) - f'(\gamma) = 0$을 전개해야 비로소 $\beta$가 중점이라는 사실을 발견
- (나)만 보면: 자기합성 방정식의 근 개수라는 독립적 과제 → (가)의 결과 없이는 세 분기의 근 구조를 제어할 변수가 과다
- (다)만 보면: 근의 합이라는 평범한 비에타 정보
- **(가)+(나) 연립**: 대칭 구조에서 기함수의 거울상 분기가 드러나고, $N$ 값이 접선 임계 조건으로 환원되어 $d$가 결정됨 — **이것이 핵심 충돌**
- **(가)+(나)+(다)**: $d$와 $r_3$가 모두 확정되어 $f$가 유일하게 결정됨

## [문제 초안]

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 서로 다른 세 실근을 가지며, 세 근을 크기순으로 $\alpha, \beta, \gamma$ $(\alpha < \beta < \gamma)$라 하자.
>
> **(가)** $\; f'(\alpha) = f'(\gamma)$
>
> **(나)** $\; $ 방정식 $f(x - f(x)) = 0$의 서로 다른 실근의 개수는 $N$이다.
>
> **(다)** $\; \alpha + \gamma = s$
>
> $f(q)$의 값을 구하시오.
>
> *(여기서 $N$, $s$, $q$는 수치 결정 단계에서 확정)*

- **발문 형태**: $f(6)$ 값 계산 (정수 답 62)

## STEP 1. 파라미터 결정 가능성 검토

### (가)에서 얻는 정보

$f(x) = (x-\alpha)(x-\beta)(x-\gamma)$이므로

$$f'(\alpha) - f'(\gamma) = (\alpha - \gamma)\bigl[\alpha + \gamma - 2\beta\bigr] = 0$$

$\alpha \neq \gamma$이므로 $\beta = \dfrac{\alpha + \gamma}{2}$. 즉 **$\beta$가 변곡점**이며, 세 근은 대칭 배치된다.

파라미터화: $\alpha = m-d,\; \beta = m,\; \gamma = m+d$ ($d>0$)로 놓으면

$$f(x) = (x-m)^3 - d^2(x-m)$$

**자유 파라미터: $m$, $d$ (2개)**

### (다)에서 얻는 정보

$\alpha + \gamma = 2m = s$ → $m = s/2$ 확정. **잔여 파라미터: $d$ (1개)**

### (나)에서 얻는 정보

$f(x-f(x))=0$ ⟺ $x - f(x) \in \{\alpha, \beta, \gamma\}$ ⟺ 세 분기 $f(x) = x - r$ ($r = m-d,\, m,\, m+d$)

$t = x - m$으로 치환하면 각 분기는:

| 분기 | 방정식 | 비고 |
|---|---|---|
| $r = m-d$ | $h_1(t) = t^3 - (d^2+1)t - d = 0$ | |
| $r = m$ | $h_2(t) = t^3 - (d^2+1)t = 0$ | 항상 3근: $t=0,\,\pm\sqrt{d^2+1}$ |
| $r = m+d$ | $h_3(t) = t^3 - (d^2+1)t + d = 0$ | $h_3(-t) = -h_1(t)$ (거울상) |

**분기 간 근 겹침 없음** (차이 = $\pm 2d \neq 0$).

$h_1$과 $h_3$은 거울 대칭이므로 실근 수가 동일. $h_2$는 항상 3개.

- $h_1$ 3근 + $h_3$ 3근 → $N = 3+3+3=9$
- $h_1$ 2근(접선) + $h_3$ 2근 → $N = 3+2+2=7$
- $h_1$ 1근 + $h_3$ 1근 → $N = 3+1+1=5$

**$N = 7$만이 등호 조건으로 $d$를 유일하게 결정한다.**

$N = 9$나 $N = 5$는 부등식이므로 $d$ 미결정 → **$N = 7$ 선택 필수**.

### 결론: 3조건으로 파라미터 $(m, d)$ 유일 결정. 추가 조건 불필요.

## STEP 2. 수치 결정

### $d$ 결정 ($N=7$ 접선 조건)

$h_1(t) = t^3 - (d^2+1)t - d$의 극대 = 0:

$$h_1'\!\left(-\sqrt{\tfrac{d^2+1}{3}}\right) = 0 \implies \frac{2}{3\sqrt{3}}(d^2+1)^{3/2} = d$$

양변 제곱: $\dfrac{4}{27}(d^2+1)^3 = d^2$. $u = d^2 + 1$로 놓으면

$$4u^3 = 27(u - 1) \implies 4u^3 - 27u + 27 = 0$$

$u = \tfrac{3}{2}$ 시도: $4\cdot\tfrac{27}{8} - 27\cdot\tfrac{3}{2} + 27 = \tfrac{27}{2} - \tfrac{81}{2} + 27 = 0$ ✓

인수분해: $(u - \tfrac{3}{2})^2(u + 3) = 0$ → $u = \tfrac{3}{2}$ (이중근), $u = -3$ (기각)

$$d^2 = u - 1 = \frac{1}{2}, \quad d = \frac{\sqrt{2}}{2}$$

### $m$ 결정

$s = 4$로 선택 → $m = 2$

### 함수 확정

$$f(x) = (x-2)^3 - \frac{1}{2}(x-2)$$

세 근: $\alpha = 2 - \frac{\sqrt{2}}{2},\;\; \beta = 2,\;\; \gamma = 2 + \frac{\sqrt{2}}{2}$

### 발문 수치

$$f(6) = 4^3 - \frac{1}{2}\cdot 4 = 64 - 2 = \boxed{62}$$
