## 이벨류에이터

### 원본 문제 A
## Problem

**9.**

자연수 $k$에 대하여 함수 $f(x) = 4x^3 - 12k^2x$가 있다. $x$에 대한 방정식 $f(x) = t$가 세 실근 $\alpha$, $\beta$, $\gamma$ ($\alpha < \beta < \gamma$)를 갖고,

$$(\alpha + k^2) \times \gamma \geq \beta^2$$

을 만족하도록 하는 실수 $t$의 값의 범위가 $p \leq t < q$일 때, $p + q$의 값을 구하시오. (단, $p < q$이다.)

### 원본 문제 B
## Problem

**22.** 2025학년도 9월 평가원

최고차항의 계수가 1인 삼차함수 $f(x)$가 모든 정수 $k$에 대하여

$$2k - 8 \leq \frac{f(k+2) - f(k)}{2} \leq 4k^2 + 14k$$

를 만족시킬 때, $f'(3)$의 값을 구하시오.
