## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 상황 서술**

**수학적 객체**: 모닉 삼차함수 $f(x) = x^3 + bx + c$ ($b, c$는 실수, 서로 다른 세 실근을 가짐)

**사실 A (P006)가 부과하는 제약**:
삼차곡선 위의 한 근 $r$에서 접선을 그으면, 접선 방정식과 곡선의 차이가 $(x-r)^2$을 인수로 가지므로 (이중근), 제2교점의 $x$좌표가 $-2r$로 결정된다. 이는 이차항이 없는 삼차함수의 보편적 성질이다. 이 제2교점을 기준으로 추가 기하 조건(제2교점을 지나는 특정 기울기의 직선과 곡선의 교점 등)이 부과될 수 있다.

**사실 B (P003)가 부과하는 제약**:
세 근 $r_1, r_2, r_3$의 합이 0이므로, 부호가 유일한 근 하나의 절댓값이 나머지 두 근의 절댓값 합과 같다. 따라서 $|r_1|+|r_2|+|r_3|$라는 양은 그 유일한 근의 절댓값의 **정확히 2배**가 된다. 이 환원 덕분에, 세 개의 미지수가 관여하는 절댓값 합 문제가 단일 변수 문제로 축소된다.

**숨은 사실 (A+B 동시 성립 시 강제되는 것)**:
부호가 유일한 근 $r_{\text{lone}}$에서 접선을 긋는 기하적 작도를 수행하면, 제2교점의 $|x|$좌표 $= 2|r_{\text{lone}}|$가 바로 세 근의 절댓값 합 $|r_1|+|r_2|+|r_3|$와 **항등적으로** 일치한다. 수험생이 접선의 제2교점 좌표를 구한 순간, 근의 절댓값 합이라는 전혀 별개로 보이던 양이 이미 손에 들려 있다.

---

### 수험생의 발견 순간
1. **수험생이 처음 시도하는 것**
   $f(x) = x^3 + bx + c$의 세 근의 절댓값 합 $|r_1|+|r_2|+|r_3|$을 구하기 위해, 근과 계수의 관계($r_1+r_2+r_3=0$, $r_1r_2+r_2r_3+r_3r_1=b$, $r_1r_2r_3=-c$)를 쓰고 절댓값을 직접 처리하려 한다. 그러나 절댓값 내부의 부호 분류가 경우로 나뉘어 막힌다.

2. **막히는 순간**
   근의 합 = 0에서 $|r_1|+|r_2|+|r_3|$를 $b, c$로 표현하려면 각 근의 부호를 알아야 하는데, 근을 직접 구하지 않고서는 부호를 확정할 수 없다. 한편, 문제에서 주어진 접선 조건의 기하적 계산은 이미 끝나서 제2교점 $-2r_{\text{lone}}$이 구해져 있지만, 이것이 절댓값 합과 무슨 관련인지 아직 보이지 않는다.

3. **발견**
   근의 합 = 0이므로 $|r_{\text{lone}}| = |r_2| + |r_3|$ (나머지 두 근은 같은 부호), 따라서 $|r_1|+|r_2|+|r_3| = 2|r_{\text{lone}}|$. 그런데 이것은 이미 구한 **접선 제2교점의 $|x|$좌표** $|-2r_{\text{lone}}| = 2|r_{\text{lone}}|$와 정확히 같다! 즉, 세 근의 절댓값 합을 따로 계산할 필요 없이, 접선의 기하적 결과물이 답 그 자체였다.

4. **STEP 2-2의 숨은 사실과의 대응**
   이 발견은 "접선 제2교점의 $|x|$ = 세 근의 절댓값 합"이라는 숨은 항등식과 정확히 대응한다. 수험생은 접선 계산(P006)과 절댓값 합 환원(P003)을 별개로 수행하지만, 둘이 $2|r_{\text{lone}}|$이라는 동일한 양으로 수렴하는 순간, 복잡해 보이던 문제가 이미 풀려 있었음을 깨닫게 된다.

## 2. 프라블럼 메이커 (Problem Maker)

### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x) = x^3 + ax + b$ ($a$, $b$는 실수)가 서로 다른 세 실근 $\alpha$, $\beta$, $\gamma$를 가진다. 다음 조건을 만족시킬 때, $|\alpha| + |\beta| + |\gamma|$의 값을 구하시오.
>
> **(가)** $f(p) = 0$, $\,p > 0$인 실수 $p$에 대하여, 곡선 $y = f(x)$ 위의 점 $(p,\; 0)$에서의 접선이 곡선과 만나는 $(p,\; 0)$이 아닌 점의 $x$좌표는 $-d$이다.
>
> **(나)** $f(0) < 0$

*(여기서 $d$는 확정할 양의 실수. 예: $d = 6$이면 답은 $6$.)*

---

##

### 조건-상황 대응표
| 조건 | 수학적 상황의 요소 | 역할 |
|---|---|---|
| $f(x)=x^3+ax+b$ (함수 형태) | 사실 B의 토대: $\alpha+\beta+\gamma=0$ | 이차항 부재가 근의 합 = 0을 자동으로 강제. 별도 조건 없이 숨겨짐 |
| **(가)** 양의 근 $p$에서 접선의 제2교점 $x$좌표 $= -d$ | 사실 A (P006): 접선 제2교점 $x = -2p$ | 학생이 $(x-p)^2(x+2p)$ 인수분해를 유도하여 $p = d/2$를 결정 |
| **(나)** $f(0) < 0$ | 사실 B (P003): $p$가 유일한 양의 근 | 세 근의 곱 > 0과 $p > 0$으로부터 나머지 두 근이 모두 음수임을 강제. $\|\alpha\|+\|\beta\|+\|\gamma\| = 2p$로 환원 가능하게 함 |
| **발문**: $\|\alpha\|+\|\beta\|+\|\gamma\|$ | 숨은 사실 (A+B 동시 성립) | (가)의 $2p$와 (나)의 $2\|r_{\text{lone}}\|$이 동일한 양 $d$로 수렴. 접선 계산 결과가 곧 답 |
