## 완성된 문제
# STEP 0: 피드백 반영 선언

| 피드백 지적 사항 | 수정 계획 | 수정 이유 |
|---|---|---|
| (가)+(다)만으로 답 256이 유일하게 결정됨 — (나) 과잉 | **(나)를 $t_0+t_2$ (일차식) → $t_0 \cdot t_2$ (이차식)로 변경**, **(다)를 $c_0=\gamma$ → $c_0^2=\gamma$로 변경** | (가)가 $\alpha$를 확정하면 $d_0$가 1개의 자유도. 합 $t_0+t_2$는 $d_0$에 대해 일차이므로 (가)와 결합 시 $d_0$가 즉시 확정 → (다)가 불필요. **곱** $t_0 \cdot t_2$는 $d_0$에 대해 **이차**이므로 2후보를 만들고, $c_0^2$은 $|d_0|$만 고정하여 부호 자유도를 남기므로, 세 조건이 모두 필요해짐 |

# STEP 1: 구조 분석 (기호만)

## 1-1. 조건이 만드는 방정식 체계

**기본 설정:** $f(x) = x^3 + ax^2 + bx + c$, 변곡점 $x$좌표 $= -a/3$.

**사실 A (접선 연쇄):** 접점 $t_n$에서의 잔차
$$f(x) - \ell_n(x) = (x-t_n)^2(x-t_{n+1})$$
$x^2$ 계수 비교: $a = -(2t_n + t_{n+1})$ → $t_{n+1} = -a - 2t_n$

간격 $d_n = t_{n+1} - t_n$에 대해:
$$d_{n+1} = t_{n+2} - t_{n+1} = (-a - 2t_{n+1}) - t_{n+1} = -a - 3t_{n+1}$$
한편 $d_n = -a - 3t_n$이므로 $d_{n+1} = -a - 3(t_n + d_n) = d_n - 3d_n = -2d_n$

$$\boxed{d_n = d_0 \cdot (-2)^n}$$

**사실 B (잔차 극값):** $g(x) = (x-t_n)^2(x-t_{n+1})$의 비자명 극점은 $x = \frac{2t_{n+1}+t_n}{3}$
$$c_n = g\!\left(\frac{2t_{n+1}+t_n}{3}\right) = -\frac{4}{27}d_n^3 = -\frac{4}{27}d_0^3 \cdot (-8)^n$$

$$\boxed{c_n = c_0 \cdot (-8)^n}, \quad c_0 = -\frac{4}{27}d_0^3$$

**좌표 표현:** 변곡점 $x$좌표 $\alpha = -a/3$이면 $a = -3\alpha$, $d_0 = 3(\alpha - t_0)$, $t_0 = \alpha - d_0/3$.
$$t_0 = \alpha - \frac{d_0}{3}, \quad t_2 = t_1 + d_1 = \left(\alpha + \frac{2d_0}{3}\right) + (-2d_0) = \alpha - \frac{4d_0}{3}$$

**조건 번역 (수정된 구조):**

| 조건 | 방정식 |
|---|---|
| (가) 변곡점 $x$좌표 $= \alpha$ | $\alpha$ 확정 |
| (나) $t_0 \cdot t_2 = \beta$ | $\left(\alpha - \frac{d_0}{3}\right)\!\left(\alpha - \frac{4d_0}{3}\right) = \beta$ — $d_0$에 대해 **이차** |
| (다) $c_0^2 = \gamma$ | $\frac{16}{729}d_0^6 = \gamma$ — $|d_0|$ 확정, 부호 미정 ($d_0 = \pm|d_0|$) |

## 1-2. 분기 분석

(가) 적용 후 자유 파라미터: $d_0$ (1개).

| 조건 쌍 | $d_0$ 후보 수 | 유일 결정? |
|---|---|---|
| (가)+(나) | 이차 방정식 → **최대 2개** | ✗ |
| (가)+(다) | $d_0 = +k$ 또는 $-k$ → **2개** | ✗ |
| (나)+(다) | $\alpha$ 미정 → $(\alpha, d_0)$ 다수 | ✗ |
| (가)+(나)+(다) | 교집합으로 **1개** 선별 목표 | → 수치 조정으로 달성 |

**핵심:** 어떤 두 조건 쌍도 $d_0$를 유일하게 결정하지 못하고, 세 조건 모두 결합해야 유일한 $d_0$가 결정되도록 수치를 설정하면 **조건 과잉 없이 세 조건 모두 필수**.

## 1-3. 자유도 계산

| 항목 | 수 |
|---|---|
| 체인 결정 파라미터 | $\alpha, d_0$ — 2개 |
| 방정식 | (가) 1개 + (나) 1개 + (다) 1개 = 3개 |
| 유효 자유도 | (가)로 $\alpha$ 제거 → $d_0$ 1개에 대해 (나)(이차)+(다)(부호 2분기) 합류 |

→ $d_0$의 2후보 중 하나만 (나)를 만족하도록 수치 설정 필요.

## 1-4. 탐색 전략 수립

**자유 파라미터:** $\alpha$ (정수), $d_0$ (정수, 비영)
**종속:** $\beta = t_0 \cdot t_2$, $\gamma = c_0^2$, $c_m = c_0 \cdot (-8)^m$

$c_0 = -\frac{4}{27}d_0^3$이 정수가 되려면 $d_0^3$이 27의 배수 → $d_0$가 3의 배수.

**탐색 후보:**

| $\alpha$ | $d_0$ 후보 | 비고 |
|---|---|---|
| $1$ | $\pm 3$ | $c_0 = \mp 4$, 간결 |
| $2$ | $\pm 3$ | $c_0 = \mp 4$ |
| $0$ | $\pm 3$ | $c_0 = \mp 4$ |

**탐색 순서:** $(\alpha, d_0) = (1, -3)$을 "통과 후보"로 잡고, 반대 부호 $d_0 = +3$이 (나)를 **불만족**하도록 $\beta$ 설정.

발문: $c_m$, $m = 2$ → $c_2 = c_0 \cdot 64$. $c_0 = 4$이면 $c_2 = 256 \leq 999$. ✓

# STEP 2: 수치 탐색

## 시도 1: $\alpha = 1$, $d_0 = -3$

**종속 파라미터 계산:**

$$t_0 = 1 - \frac{-3}{3} = 1 + 1 = 2$$
$$t_1 = -(-3) - 2(2) = 3 - 4 = -1 \quad (a = -3, \ t_1 = -a - 2t_0 = 3-4=-1)$$
$$t_2 = t_1 + d_1 = -1 + 6 = 5 \quad (d_1 = -2(-3) = 6)$$

조건값 결정:
$$\beta = t_0 \cdot t_2 = 2 \times 5 = 10$$
$$c_0 = -\frac{4}{27}(-3)^3 = -\frac{4}{27}(-27) = 4$$
$$\gamma = c_0^2 = 16$$

**반대 분기 검증 ($d_0 = +3$):**

$$t_0 = 1 - \frac{3}{3} = 0, \quad t_2 = 1 - \frac{4(3)}{3} = 1 - 4 = -3$$
$$t_0 \cdot t_2 = 0 \times (-3) = 0 \neq 10$$

→ $d_0 = +3$은 (나)를 **불만족**. ✓ 세 조건 합류 시 $d_0 = -3$만 생존.

**조건 독립성 (세 조건 각각 제거 시 답이 유일하지 않음을 확인):**

### (가) 제거 — $\alpha$ 미지

(다): $d_0 = \pm 3$. (나): $\left(\alpha + 1\right)\left(\alpha + 4\right) = 10$ 또는 $\left(\alpha - 1\right)\left(\alpha - 4\right) = 10$.

$d_0 = -3$: $\alpha^2 + 5\alpha + 4 = 10$ → $\alpha^2 + 5\alpha - 6 = 0$ → $(\alpha+6)(\alpha-1) = 0$ → $\alpha = 1$ 또는 $\alpha = -6$

$d_0 = 3$: $\alpha^2 - 5\alpha + 4 = 10$ → $\alpha^2 - 5\alpha - 6 = 0$ → $(\alpha-6)(\alpha+1) = 0$ → $\alpha = 6$ 또는 $\alpha = -1$

$(\alpha, d_0) \in \{(1,-3),\ (-6,-3),\ (6,3),\ (-1,3)\}$
$$c_0 \in \{4, \ 4, \ -4, \ -4\}, \quad c_2 \in \{256, \ 256, \ -256, \ -256\}$$

**답이 $256$ 또는 $-256$ → 유일하지 않음.** ✓ (가) 필수.

### (나) 제거

(가): $\alpha = 1$. (다): $d_0 = \pm 3$.

| $d_0$ | $c_0$ | $c_2 = c_0 \cdot 64$ |
|---|---|---|
| $-3$ | $4$ | $256$ |
| $+3$ | $-4$ | $-256$ |

**답 2개 → 유일하지 않음.** ✓ (나) 필수.

### (다) 제거

(가): $\alpha = 1$. (나): $(1 - d_0/3)(1 - 4d_0/3) = 10$.

전개: $1 - \frac{5d_0}{3} + \frac{4d_0^2}{9} = 10$

$\frac{4d_0^2}{9} - \frac{5d_0}{3} - 9 = 0$ → $4d_0^2 - 15d_0 - 81 = 0$

$d_0 = \frac{15 \pm \sqrt{225 + 1296}}{8} = \frac{15 \pm \sqrt{1521}}{8} = \frac{15 \pm 39}{8}$

| 근 | $d_0$ | $c_0 = -4d_0^3/27$ | $c_2 = 64 \cdot c_0$ |
|---|---|---|---|
| $+$ | $54/8 = 27/4$ | $-4(27/4)^3/27 = -4 \cdot 729/64 = -729/16$ | $-729 \cdot 4 = -2916$ |
| $-$ | $-24/8 = -3$ | $4$ | $256$ |

**답 $256$ 또는 $-2916$ → 유일하지 않음.** ✓ (다) 필수.

### 탐색 결과 표

| 시도 | $\alpha$ | $d_0$ | $\beta$ | $\gamma$ | $m$ | 답 $c_m$ | 유일성 | 손계산 | 조건 독립 | 판정 |
|---|---|---|---|---|---|---|---|---|---|---|
| 1 | $1$ | $-3$ | $10$ | $16$ | $2$ | $256$ | ✅ | ✅ | ✅ 3조건 모두 필수 | **채택** |

## 완성된 문제

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여 곡선 $y = f(x)$ 위의 점 $\mathrm{P}$에서의 접선이 이 곡선과 만나는 $\mathrm{P}$가 아닌 점을 $T(\mathrm{P})$라 하자.
>
> 곡선 $y = f(x)$ 위의 점 $\mathrm{P}_0$에 대하여
> $$\mathrm{P}_{n+1} = T(\mathrm{P}_n) \quad (n = 0,\, 1,\, 2,\, \cdots)$$
> 으로 점열 $\{\mathrm{P}_n\}$을 정의하고, 점 $\mathrm{P}_n$의 $x$좌표를 $t_n$이라 하자. 점 $\mathrm{P}_n$에서 곡선 $y = f(x)$에 접하는 직선을 $\ell_n$이라 할 때, 함수 $f(x) - \ell_n(x)$의 극값 중 $0$이 아닌 것을 $c_n$이라 하자.
>
> 다음 조건을 만족시킨다.
>
> (가) 곡선 $y = f(x)$의 변곡점의 $x$좌표는 $1$이다.
>
> (나) $t_0 \times t_2 = 10$
>
> (다) ${c_0}^2 = 16$
>
> $c_2$의 값을 구하시오. $\quad$ [4점]
>
> **정답: $256$**

## 출처 아이디어
**수학적 객체**: 최고차항의 계수가 1인 삼차함수 $f(x)$와 그 위의 한 점 $P_0 = (a, f(a))$.

**사실 A가 부과하는 제약 (P006 — 접선 연쇄의 제3교점 결정)**:
$P_0$에서 곡선에 접선을 그으면, 접선과 곡선의 교점 방정식이 접점에서 이중근을 가지므로 제3교점 $P_1$이 유일하게 결정된다. $P_1$에서 다시 접선을 그으면 $P_2$가, 이를 반복하면 $P_3, P_4, \ldots$가 연쇄적으로 결정된다. 연속하는 제3교점 사이의 $x$좌표 간격 $d_k = x_{P_{k+1}} - x_{P_k}$는 매 단계 $(-2)$배로 증폭된다.

**사실 B가 부과하는 제약 (P012 — 잔차의 극값 정량화)**:
$k$번째 단계에서 접선을 차감한 잔차 함수 $g_k(x) = f(x) - \ell_k(x)$는 접점에서 이중근, 제3교점에서 단근을 가져 $(x - x_{P_k})^2(x - x_{P_{k+1}})$로 인수분해된다. 이 잔차 삼차식의 유일한 비자명 극값 $E_k$는 두 근의 간격 $d_k$의 세제곱에 비례하는 닫힌 식 $E_k = -\frac{4}{27}d_k^3$이다.

**숨은 사실 (A + B 동시 성립 시 강제)**:
두 사실을 결합하면 $E_{k+1}/E_k = (-2)^3 = -8$이 매 단계 성립한다. 즉, **삼차함수의 접선 연쇄에서 잔차 극값은 공비 $-8$의 등비수열**을 이룬다. 이 비율 $-8$은 삼차함수의 구체적 계수에도, 시작점의 선택에도 전혀 의존하지 않는 보편 상수이다. 부호가 매번 반전되므로 극대 → 극소 → 극대 → ⋯ 의 교대가 동반된다.

## 수험생의 발견 순간
1. **수험생이 처음 시도하는 것**: 주어진 점에서 접선을 구하고, $f(x) - \ell(x) = 0$을 풀어 제3교점 $P_1$을 찾는다. 이중근 인수분해 $(x-a)^2(x-\beta)$를 수행하고, 비에타 관계로 $\beta$를 구한다. 이어서 $P_1$에서 같은 과정을 반복하여 $P_2$를 구한다.

2. **막히는 순간**: 문제가 $n$번째 잔차의 극값에 관한 조건(예: 극값의 절댓값이 특정 값 이하가 되는 최소 $n$, 또는 극값의 합)을 요구하는데, 매 단계마다 새로운 접선을 구하고 잔차를 계산하는 것은 계산이 폭발적으로 증가하여 직접 추적이 불가능해 보인다.

3. **발견**: 첫 단계의 잔차 극값 $E_0$와 두 번째 단계의 잔차 극값 $E_1$을 직접 계산해보니 $E_1 = -8 E_0$이다. 세 번째를 계산하면 $E_2 = -8 E_1 = 64 E_0$. "어? 매번 $-8$배가 되네?" — 제3교점 간격이 $(-2)$배씩 늘어나고, 극값 공식이 간격의 **세제곱**에 비례하므로 $(-2)^3 = -8$이 보편 공비가 된다는 것을 깨닫는다.

4. **숨은 사실과의 대응**: 이 발견은 STEP 2-2의 숨은 사실 — *거리 증폭 비율의 세제곱이 극값 비율이 된다*는 구조 — 와 정확히 대응한다. 일단 공비 $-8$을 인식하면, $E_n = E_0 \cdot (-8)^n$으로 일반항이 즉시 확정되어 등비수열의 도구(합, 부등식 등)로 문제가 마무리된다.
