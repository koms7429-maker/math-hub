# STEP 1. 구조 분석

## 1-1. 조건이 만드는 방정식 체계

$f(x) = (x-\alpha)(x-\beta)(x-\gamma)$로 놓으면

$$f'(\text{root}_i) = \prod_{j \neq i}(\text{root}_i - \text{root}_j)$$

$a = \beta - \alpha > 0$, $b = \gamma - \beta > 0$으로 놓으면:

| 조건 | 기호 방정식 |
|------|-----------|
| (가) | $f'(\alpha)+f'(\beta)+f'(\gamma)= a^2+ab+b^2 = k$ |
| (나) | $f'(\alpha)\cdot f'(\gamma)= ab(a+b)^2 = m$ |
| (다) | $f''(s)=0 \;\Rightarrow\; s = \frac{\alpha+\beta+\gamma}{3}$, 즉 $\beta = s - \frac{b-a}{3}$ (위치 앵커) |
| 관계식 | $m = \dfrac{4k^2}{9}$ (PM이 제시한 수치 관계) |

**핵심 연립:** (가)+(나)의 비율 조건
$$\frac{ab(a+b)^2}{(a^2+ab+b^2)^2} = \frac{4}{9}$$

$t = a/b$로 치환하면 $4t^4 - t^3 - 6t^2 - t + 4 = 0$이 되고, 이것은 $(t-1)^2(4t^2+7t+4)=0$으로 인수분해됩니다. $4t^2+7t+4$의 판별식 $= -15 < 0$이므로 **유일한 양의 실수 해는 $t = 1$, 즉 $a = b$ (등차수열).**

## 1-4. 탐색 전략

1. $a = b := d$이므로 $k = 3d^2$. 정수 $d$를 선택하면 $k$ 결정.
2. $m = 4d^4 = \frac{4k^2}{9}$ 자동 만족.
3. $\beta = s$ (정수 선택).
4. 답: $f'(s) = -d^2$.

**정수해 탐색 범위:** $d \in \{1, 2, 3, \ldots\}$. $k, m$이 너무 크지 않고 깔끔한 후보: $d = 1, 2, 3$.

## 완성된 문제

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 서로 다른 세 실근 $\alpha,\;\beta,\;\gamma$ $(\alpha < \beta < \gamma)$를 갖고 다음 조건을 만족시킨다.
>
> (가) $f'(\alpha) + f'(\beta) + f'(\gamma) = 12$
>
> (나) $f'(\alpha) \times f'(\gamma) = 64$
>
> (다) 곡선 $y = f(x)$의 변곡점의 $x$좌표는 $3$이다.
>
> $-f'(3)$의 값을 구하시오. $\quad$ [4점]

**정답: $4$**

## 출처 아이디어
**수학적 객체**: 매개변수를 포함한 모닉 삼차함수 $f(x) = x^3 + ax^2 + bx + c$로, 세 서로 다른 실근 $r_1 < r_2 < r_3$을 갖는다.

**사실 A가 부과하는 제약 (P011의 항등식)**:
세 근에서의 도함수 값의 합 $f'(r_1) + f'(r_2) + f'(r_3)$이 특정 상수로 주어진다. 이는 곧 $a^2 - 3b$의 값을 고정하여, 계수 $a$와 $b$ 사이에 하나의 관계식을 부여한다.

**사실 B가 부과하는 제약 (P009의 축약)**:
양 끝 근에서의 도함수에 대한 조건 — 예를 들어 $f'(r_1) = f'(r_3)$, 또는 $f'(r_1) \leq f'(r_3)$ — 이 부과된다. 이 조건은 비에타 대입 후 극적으로 축약되어 $r_2$와 $\frac{r_1+r_3}{2}$의 대소 관계로 환원된다.

**숨은 사실 (A+B 동시 작용 시 강제되는 것)**:
$f'(r_1) = f'(r_3)$일 때, 비에타 축약에 의해 세 근이 등차수열 $r_2 - d, \; r_2, \; r_2 + d$로 강제된다. 이 순간 각 근에서의 도함수 값은 $f'(r_1) = f'(r_3) = 2d^2$, $f'(r_2) = -d^2$로 확정되며, A의 항등식 $f'(r_1) + f'(r_2) + f'(r_3) = 3d^2 = a^2 - 3b$가 공차 $d$를 계수와 직접 연결한다. 결과적으로, 도함수 등가 조건 하나가 함수의 근 구조를 거의 완전히 결정한다.

---

## 수험생의 발견 순간
1. **수험생이 처음 시도하는 것**:
   각 근 $r_i$에서 $f'(r_i)$를 직접 계산하려 한다. $f'(x) = 3x^2 + 2ax + b$에 각 근을 대입하여 $f'(r_1) = f'(r_3)$을 정리하면 $3(r_1^2 - r_3^2) + 2a(r_1 - r_3) = 0$이 되고, $r_1 \neq r_3$으로 나누어 $3(r_1 + r_3) + 2a = 0$, 즉 $r_1 + r_3 = -\frac{2a}{3}$를 얻는다. 한편 비에타에서 $r_1 + r_2 + r_3 = -a$이므로 $r_2 = -a - (r_1+r_3) = -a + \frac{2a}{3} = -\frac{a}{3}$을 구한다.

2. **막히는 순간**:
   $r_2 = -\frac{a}{3}$을 얻었지만, 이것이 근의 배치에 어떤 의미인지 즉시 보이지 않는다. $r_1, r_3$은 여전히 자유로워 보이고, "도함수가 같다"는 조건이 근의 구조를 얼마나 제약하는지 감이 안 온다. 매개변수의 범위나 부등식 조건을 다루기 위해 근의 배치를 알아야 하는데, 정보가 부족해 보인다.

3. **발견**:
   $r_1 + r_3 = -\frac{2a}{3}$이고 $r_2 = -\frac{a}{3}$이므로, $r_1 + r_3 = 2r_2$임을 깨닫는다. **"세 근이 등차수열이다!"** 이 순간, $r_1 = r_2 - d$, $r_3 = r_2 + d$로 놓을 수 있고, 함수가 $f(x) = (x - r_2 + d)(x - r_2)(x - r_2 - d) = (x-r_2)^3 - d^2(x - r_2)$, 즉 변곡점 $r_2$를 중심으로 한 기본형 $t^3 - d^2 t$로 완전히 단순화된다. 도함수 값도 $f'(r_1) = f'(r_3) = 2d^2$, $f'(r_2) = -d^2$으로 공차 하나에 종속되어, 이후의 모든 조건이 일사천리로 풀린다.

4. **STEP 2-2의 숨은 사실과의 대응**:
   발견 "세 근이 등차수열"은 정확히 STEP 2-2의 숨은 사실 — *양 끝 근에서의 도함수 값이 같으면 세 근은 등차수열이고, 도함수 값이 공차의 제곱으로 완전 결정된다* — 에 대응한다. 수험생은 도함수 등가라는 **미분적** 조건에서 출발하여, 비에타 축약이라는 **대수적** 경로를 거쳐, 등차 배치라는 **기하적** 결론에 도달한다. 이 범주 전환의 놀라움이 문제의 핵심 발견 순간이다.

## 문제 구조
### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 서로 다른 세 실근 $\alpha,\;\beta,\;\gamma$ $(\alpha < \beta < \gamma)$를 갖고 다음 조건을 만족시킨다.
>
> (가) $f'(\alpha) + f'(\beta) + f'(\gamma) = k$
>
> (나) $f'(\alpha) \times f'(\gamma) = m$
>
> (다) 곡선 $y = f(x)$의 변곡점의 $x$좌표는 $s$이다.
>
> $f'(s)$의 값을 구하시오.

*(여기서 $k,\; m,\; s$는 $m = \dfrac{4k^2}{9}$을 만족하는 구체적 수치로 결정될 것이며, 발문 $f'(s)$의 답은 $-\dfrac{k}{3}$이 된다.)*
