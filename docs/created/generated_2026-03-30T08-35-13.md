## [2] 각 아이디어가 만드는 계수 관계식 (기호 수준)

**설정**: $f(x) = x^3 + ax^2 + bx + c$를 세 실근 $r_1, r_2, r_3$을 가진 모닉 삼차다항식으로 놓고, $g(x) = f(x) - x = x^3 + ax^2 + (b-1)x + c$를 정의한다.

### 아이디어 P004가 만드는 관계식 $R_1(a, b, c)$

$f$의 두 서로 다른 근 $r_i, r_j$에서 $f'(r_i) = f'(r_j)$가 성립하면:

$$3(r_i + r_j) + 2a = 0 \quad \Longrightarrow \quad r_i + r_j = -\tfrac{2a}{3}$$

Vieta에 의해 $r_i + r_j + r_k = -a$이므로, 나머지 근은

$$r_k = -a + \tfrac{2a}{3} = -\tfrac{a}{3}$$

즉 **$f$의 한 근이 변곡점의 $x$좌표 $-a/3$과 정확히 일치**해야 한다.
이것은 $f(-a/3) = 0$, 즉:

$$R_1:\quad \left(-\tfrac{a}{3}\right)^3 + a\left(-\tfrac{a}{3}\right)^2 + b\left(-\tfrac{a}{3}\right) + c = 0$$

정리하면 $\dfrac{2a^3}{27} - \dfrac{ab}{3} + c = 0$, 즉 $a, b, c$ 사이의 **하나의 관계식**.

> $R_1$은 $b$나 $c$를 단독으로 결정하지 못한다. $a$가 주어져도 $b, c$에 대한 1차 관계식 하나만 남는다.

---

### 아이디어 P001이 만드는 관계식 $R_2(a, b, c)$

$f(x - f(x)) = 0$은 각 근 $r_i$에 대해 $f(x) = x - r_i$, 즉 $g(x) = -r_i$로 분해된다.

$g(x)$는 삼차이므로 임계점 $p, p'$ ($g'(p)=0, g'(p')=0$)에서 극대 $M = g(p)$, 극소 $m = g(p')$를 가진다.

$g(x) = -r_i$의 실근 개수는 $-r_i$가 $m$과 $M$ 사이에 있는지에 따라 3개 또는 1개이고, $-r_i = M$ 또는 $-r_i = m$이면 2개(이중근)이다.

**총 근의 개수를 특정 값으로 고정하면**, 세 수 $\{-r_1, -r_2, -r_3\}$ 중 최소 하나가 $g$의 임계값 $M$ 또는 $m$과 **정확히 일치**해야 한다.

가령 $-r_k = M = g(p)$라 하면, $g(x) - g(p) = (x-p)^2(x-q)$이므로 Vieta에 의해:

$$2p + q = -a, \quad p^2 + 2pq = b-1, \quad -p^2 q = c + r_k$$

여기서 $r_k = -g(p) = -(p^3 + ap^2 + (b-1)p + c)$이고, $p$는 $g'(p) = 3p^2 + 2ap + (b-1) = 0$의 근이다.

이 조건들을 정리하면 **$a, b, c, p$ 사이의 연립 관계식** $R_2$가 생성된다.

> $R_2$ 역시 단독으로는 자유도가 남는다. 임계점 위치 $p$가 미지수로 추가되어, 방정식 수보다 미지수가 많다.

## [3] 파라미터 공유 확인

| | $a$ | $b$ | $c$ |
|---|---|---|---|
| $R_1$ (P004) | ✓ | ✓ | ✓ |
| $R_2$ (P001) | ✓ | ✓ | ✓ |

**세 파라미터 $a, b, c$ 모두 공유한다.**

더 중요하게:
- $R_1$은 $r_k = -a/3$을 강제하여 **$c$를 $a, b$의 함수로 축약**.
- $R_2$는 임계값-근 정렬 조건으로 **$b$를 $a, p$의 함수로 축약**하면서 동시에 $p$도 $a, b$에 종속.
- $R_1$이 제공하는 **$r_k = -a/3$ 구조**가 $R_2$의 "$-r_k = M$" 조건에 대입되면, $g(-a/3)$의 값이 $g$의 임계값과 일치해야 하는 **$a$ 하나에 대한 고차 방정식**이 남는다.
- 이 고차 방정식은 $P004$가 부여한 대칭 구조 때문에 **겉보기 차수보다 낮게 인수분해**된다.

## [4] 수학적 상황 서술

$f(x) = x^3 + ax^2 + bx + c$가 세 실근을 가진다고 하자.

**조건 ①에서 P004가 작동**: $f$의 두 근에서 도함수 값이 같다.
→ 나머지 한 근이 $r_k = -a/3$(변곡점 $x$좌표)으로 고정.
→ $f(-a/3) = 0$에서 $c$가 $a, b$의 함수로 결정: $c = \tfrac{ab}{3} - \tfrac{2a^3}{27}$.
→ 동시에 대칭인 두 근 $r_i, r_j$는 $r_i + r_j = -2a/3$, $r_i r_j = \tfrac{b}{1}$... 등으로 $a, b$에 종속.

**조건 ②에서 P001이 작동**: $f(x - f(x)) = 0$의 총 실근 개수가 특정 값으로 주어진다.
→ $g(x) = f(x) - x$의 임계점 $p$에서 $g(p) = -r_k = a/3$이 성립해야 함.
→ $g'(p) = 0$과 $g(p) = a/3$을 연립하면 $p$가 $a, b$에 종속.

**두 조건의 연립 (비자명한 구조 발현)**:

$R_1$에서 $c$를 소거한 뒤, $R_2$의 임계값 조건에 $r_k = -a/3$을 대입하면:

$$g\left(-\frac{a}{3}\right) = f\left(-\frac{a}{3}\right) - \left(-\frac{a}{3}\right) = 0 + \frac{a}{3} = \frac{a}{3}$$

이 값이 $g$의 극대(또는 극소)값과 일치해야 하므로, 임계점 $p$에서의 이중근 분해

$$g(x) - \frac{a}{3} = (x - p)^2(x - q)$$

와 원래 $g(x) - \frac{a}{3} = f(x) - x - \frac{a}{3}$의 계수 비교가 **$a$ 하나의 고차 다항식**을 생성한다.

핵심: 이 고차 다항식은 $P004$가 강제한 "$f(-a/3) = 0$" 대칭 구조 때문에, **변곡점에서의 함수값이 이미 0으로 고정**되어 있어 상수항과 중간항에서 대량 상쇄가 발생한다. 결과적으로 **6차로 예상되는 다항식이 (기약 이차) × (중근을 가진 이차)² 등으로 비자명하게 인수분해**되며, 실수 조건·근의 실수 조건·대소 관계를 만족하는 후보가 이산적으로(1~2개) 남는다.

**어느 한 조건만으로는 $a$가 결정되지 않는 구조**:
- $R_1$만: $c$를 소거하지만, $(a, b)$는 자유 → $a$ 미결정.
- $R_2$만: 임계값-근 정렬이지만, 어떤 근이 정렬되는지 모르고 $(a, b, c)$ 3개 미지수에 방정식 2개 → $a$ 미결정.
- **$R_1 + R_2$**: $R_1$이 "정렬되는 근이 $-a/3$"이라는 정체를 밝히고, 이 정보가 $R_2$에 대입되는 순간 $b$가 $a$의 함수로 축약되어 **$a$에 대한 단일 폐쇄 방정식**이 성립. 이것이 숨겨진 대칭으로 인수분해됨.

## [5] 수험생의 발견 지점

수험생은 **두 번째 조건(P001 계열)의 임계값-근 정렬 조건을 전개**하면서, 어떤 근 $r_k$가 임계값과 정렬되는지 특정하지 못해 일반적인 고차식에 직면한다.

이때 **첫 번째 조건(P004 계열)로 돌아가** $r_k = -a/3$이라는 사실을 상기하고, 이것이 $g(-a/3) = a/3$(변곡점에서의 $g$ 값이 자동으로 $a/3$)이라는 **놀라운 단순화**를 만들어냄을 발견한다.

이 값을 임계값 조건에 대입하는 순간, 이중근 분해의 계수 비교에서 **$b$가 $a$의 이차식으로 즉시 표현**되고, 최종적으로 $a$에 대한 고차 방정식이 남는다. 이 방정식은 외형상 5~6차이지만, 대칭 구조가 **중근 인수를 숨기고 있어** 인수분해하면 실질적으로 이차 이하의 방정식으로 환원된다. 

**발견의 핵심**: "도함수값이 같은 두 근의 대칭성"이 강제한 $r_k = -a/3$이라는 구조가, 자기합성 방정식의 임계값 정렬에서 **$g$의 변곡점값을 자동으로 확정**시켜, 원래라면 3자유도($a,b,c$)에서 시작하는 문제를 **단일 변수의 인수분해 문제로 압축**하는 순간이다.

## [0] 이번 시도의 차별성 선언

- **수학적 객체**: 삼차함수의 *영점*이 아닌 **고정점** ($f(x)=x$)을 중심 무대로 설정 → $g(x)=f(x)-x$가 보조함수가 아닌 구조의 중심축으로 자연스럽게 등장
- **조건 형태**: 등식(도함수값 일치) + 개수(합성방정식 실근 수) + 존재성(특정 고정점 값) — 세 조건이 각각 **다른 형태**로 제시되어 연결 경로가 즉시 보이지 않음
- **충돌 표현**: 간접적(위장) — "두 고정점에서 $f'$값 동치"라는 대수 조건이 **"한 고정점 = 변곡점"** 이라는 기하적 사실을 숨기고, 이것이 합성방정식의 임계값 조건에 대입되는 순간 비로소 폭발적 단순화 발생

# STEP 1. 파라미터 결정 가능성 검토

## 1-1. 조건 (나)의 구조적 귀결

$f(x) = x^3 + ax^2 + bx + c$, $g(x) = f(x) - x = x^3 + ax^2 + (b-1)x + c$.

세 고정점(근) $\alpha, \beta, \gamma$에 대해 $f'(\alpha) = f'(\beta)$로부터:

$$3(\alpha+\beta) + 2a = 0 \implies \alpha + \beta = -\frac{2a}{3}$$

비에타: $\alpha+\beta+\gamma = -a$이므로

$$\gamma = -a - \left(-\frac{2a}{3}\right) = -\frac{a}{3}$$

이는 $g''(x) = 6x+2a = 0$의 해, 즉 $g$의 **변곡점**이 고정점 $\gamma$와 일치함을 의미한다.

따라서 $\alpha + \beta = 2\gamma$이고, 세 고정점은 $\gamma$에 대해 대칭: $\gamma - s,\; \gamma,\; \gamma + s$ ($s > 0$).

**자유 파라미터**: $\gamma$(위치), $s$(간격) — 총 2개.

## 1-2. 조건 (가)의 역할

"세 근 중 하나가 $2$이다."

| 분기 | 의미 | 잔여 파라미터 |
|------|------|--------------|
| $\gamma = 2$ (중간 근) | $t=2$, 근: $2-s,\;2,\;2+s$ | $s$ 1개 |
| $\gamma + s = 2$ (오른쪽 근) | $s = 2-\gamma$, 근: $2\gamma-2,\;\gamma,\;2$ | $\gamma$ 1개 |
| $\gamma - s = 2$ (왼쪽 근) | $s = \gamma-2$, 근: $2,\;\gamma,\;2\gamma-2$ | $\gamma$ 1개 |

→ 각 분기에서 **1개의 자유 파라미터**가 남음.

## 1-3. 조건 (다)의 역할 — 나머지 1 파라미터 결정

$f(x-f(x))=0$의 실근 개수가 $k$개.

$f(r)=0$의 근을 $r_1, r_2, r_3$이라 하면, $x - f(x) = r_i$, 즉 $g(x) = -r_i$의 실근 수를 합산.

$g(x) = (x-\gamma)^3 - s^2(x-\gamma)$이므로 $u = x-\gamma$에 대해 $g = u^3 - s^2 u$.

- **극댓값** $M = \frac{2s^3}{3\sqrt{3}}$, **극솟값** $m = -M$.
- $|{-r_i}| < M$이면 3개, $= M$이면 2개, $> M$이면 1개.

$k = 6$이 되려면: 정확히 **2개의 서로 다른 수준값**(하나는 $0$, 다른 하나는 이중근에서 발생)이 모두 범위 내에 있어야 함. 이는 $f(x)=0$이 **이중근**을 가질 때 발생.

→ 이 조건이 나머지 1개 파라미터를 결정.

## 1-4. 분기별 검증

### 분기 A: $\gamma = 2$ (중간 근이 $2$)

근: $2-s,\;2,\;2+s$. $g(x) = (x-2)^3 - s^2(x-2)$.

$$f(x) = g(x)+x = x^3 - 6x^2 + (13-s^2)x + (2s^2 - 8)$$

**$f$의 영점 분석**: $f(x) = 0$.

$s=2$일 때: $f(x) = x^3 - 6x^2 + 9x = x(x-3)^2$ → **이중근 $3$, 단근 $0$**.

수준값: $-r_i \in \{0, -3\}$ (2개의 서로 다른 값).
- $g(x)=0$: 3개.
- $g(x)=-3$: $(x-2)^3 - 4(x-2) = -3$ → $u^3 - 4u + 3 = 0$ → $(u-1)(u^2+u-3)=0$ → 3개.
- 겹침 없음 확인: $\{0,2,4\} \cap \{3, \frac{3\pm\sqrt{13}}{2}\} = \varnothing$.

$$k = 3 + 3 = 6\;\checkmark$$

$s \neq 2$인 경우:
- $s < 2$: $f$가 실근 1개만 가짐 → $k=3$.
- $s > 2$: $f$가 서로 다른 실근 3개 → 모든 수준값이 범위 내 → $k=9$.

따라서 **$s=2$가 유일**.

### 분기 B: $2$가 오른쪽 근 ($\gamma + s = 2$, $\gamma < 2$)

근: $2\gamma - 2,\;\gamma,\;2$. 여러 $\gamma$ 값 검증:

| $\gamma$ | 근 | $f$의 실근 개수 | $k$ |
|---|---|---|---|
| $1$ | $0, 1, 2$ | 1개 ($f(x)=x^3-3x^2+3x$, 근 $x=0$만) | $3$ |
| $0$ | $-2, 0, 2$ | 3개 ($f(x)=x^3-3x$) | $9$ (대칭) |
| $-1$ | $-4, -1, 2$ | 복잡한 근 | $\neq 6$ (수치 확인) |

$f$가 이중근을 갖는 조건($\Delta_f = 0$)을 풀면 $\gamma$ 값이 비정수 무리수가 되어, 설사 $k=6$이 되더라도 깔끔한 함수가 나오지 않음.

### 분기 C: $2$가 왼쪽 근 ($\gamma - s = 2$, $\gamma > 2$)

근: $2,\;\gamma,\;2\gamma-2$. 검증:

| $\gamma$ | 근 | $f$의 구조 | $k$ |
|---|---|---|---|
| $3$ | $2, 3, 4$ | $(x-3)^3+3=0$, 실근 1개 | $3$ |
| $4$ | $2, 4, 6$ | 서로 다른 실근 3개 | $9$ |

역시 $k=6$을 만드는 정수 $\gamma$가 존재하지 않음.

### 결론

**세 조건을 모두 만족하는 함수는 $f(x) = x(x-3)^2 = x^3 - 6x^2 + 9x$로 유일.**
