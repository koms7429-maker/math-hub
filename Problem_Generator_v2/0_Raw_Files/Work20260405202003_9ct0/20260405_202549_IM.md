[STEP 0] 이번 시도의 차별성 선언

- **객체**: 삼차함수 f(x) = x³ + px + q (변곡점이 y축 위에 오도록 x² 항이 없는 표준형)
- **제약의 성격**: (A) 구간 분할 위의 최댓값 비교 (기하/해석적 조건), (B) 곱의 미분 역조립을 이용한 적분 조건 (대수적 조건)
- **얽힘 구조**: A가 파라미터 p를 결정하되 q를 미결로 남기고, B가 논리적 분기(q = 0 **또는** p = −1)를 생성하여 q와 p를 동시에 건드림. A+B 연립 시 B의 한 분기가 소멸하면서 q가 유일하게 결정됨.

[STEP 1] 객체 선정 + 제약 후보 탐색

**객체**: f(x) = x³ + px + q (최고차항 계수 1, x² 계수 = 0인 삼차함수)

**파라미터**: p, q (2개). 변곡점은 원점 (0, q). p < 0이면 극값 존재.

| 제약 후보 | 건드리는 파라미터 | 단독 결정 / 미결 | 영감 source |
|---|---|---|---|
| **C₁**: f의 [-1, 0] 위 최댓값 = [0, 1] 위 최댓값 | p (극점 위치), 간접적으로 q | p 결정 / **q 미결** | **P020** (구간 평행이동 극값집합 불변) |
| **C₂**: ∫₋₁¹ f(x)·f′(x) dx = 0 | p, q 모두 (끝점 함수값) | "q = 0 또는 p = −1" 분기 / 분기마다 하나 미결 | **P030** (곱미분 역조립) |
| C₃: f(x) = 0이 세 실근을 갖고 근의 절댓값 합 = 상수 | p, q | 근-계수 관계로 제약 | P003 |

**선택**: C₁(= 제약 A)과 C₂(= 제약 B)를 선택한다.

[STEP 2] 제약 쌍 선택 + 얽힘 검증

### 제약 A: f의 [-1, 0] 위 최댓값 = [0, 1] 위 최댓값

p < 0으로 설정하고 s = √(−p/3) > 0으로 놓으면 극점은 x = ±s.

- **[-1, 0] 위 후보**: f(−1) = −1−p+q, f(−s) = 2s³+q (극대), f(0) = q
- **[0, 1] 위 후보**: f(0) = q, f(s) = −2s³+q (극소), f(1) = 1+p+q

s < 1/√3 인 경우 (후에 유일 해가 이 영역에 속함):
- max[-1,0] = 2s³ + q (극댓값이 최대)
- max[0,1] = (1−3s²) + q (우끝점이 최대)

등치 조건: 2s³ = 1−3s² → **2s³ + 3s² − 1 = 0** → **(s+1)²(2s−1) = 0** → **s = 1/2**

따라서 **p = −3s² = −3/4**. q는 등치식에서 상쇄되어 **미결**.

### 제약 B: ∫₋₁¹ f(x)·f′(x) dx = 0

핵심 인식 (P030 역조립): f(x)·f′(x) = \frac{1}{2}\frac{d}{dx}[f(x)²]

따라서:

$$\int_{-1}^{1} f(x) f'(x)\,dx = \frac{f(1)^2 - f(-1)^2}{2} = 0$$

이는 **f(1)² = f(−1)²**, 즉 |f(1)| = |f(−1)|을 의미.

- f(1) = 1 + p + q
- f(−1) = −1 − p + q

두 경우가 발생:
- **분기 ①**: 1+p+q = −1−p+q → **p = −1**
- **분기 ②**: 1+p+q = 1+p−q → **q = 0**

B 단독으로는 p = −1 **또는** q = 0이며, 각 분기 내에서 나머지 파라미터가 미결.

#### 2-1. 자유도 분석

| 상황 | 결정 | 미결 | 자유도 |
|---|---|---|---|
| A 단독 | p = −3/4 | q | **1** |
| B 단독 | (분기 ①) p = −1 또는 (분기 ②) q = 0 | 각 분기마다 1개 | **1** |
| **A + B 동시** | A에서 p = −3/4. B의 분기 ① (p=−1)은 p=−3/4과 모순 → **소멸**. 분기 ②만 생존 → q = 0. | 없음 | **0** |

#### 2-2. 얽힘 판정

- [✓] A의 미결 파라미터(q)를 B가 건드린다: B의 분기 ②가 q = 0을 강제
- [✓] B의 미결 파라미터(분기 ① 에서의 자유 q, 분기 ② 에서의 자유 p)를 A가 건드린다: A가 p = −3/4를 강제하여 분기 ①(p=−1)을 소멸시키고, 분기 ②의 자유 p도 확정
- [✓] A+B 자유도(0) < A 단독(1), B 단독(1) 어느 쪽보다 **2 이상** 감소 (각각 1씩, 총 2 감소)

**→ 얽힘 성립.**

#### 2-3. 강제 결과 서술

A+B를 동시에 만족시키면:

> **f(x) = x³ − (3/4)x, 즉 f는 기함수(odd function)가 되어야 한다.**

구체적으로 드러나는 숨은 사실들:

1. **분기 소멸**: B의 적분 조건은 "q = 0 또는 p = −1"이라는 논리적 분기를 만드는데, A의 구간 최댓값 조건이 p = −3/4를 강제하여 p = −1 분기를 죽이고, 유일하게 q = 0만 생존시킨다.

2. **기함수의 출현**: q = 0이므로 f(x) = x³ − (3/4)x = x(x² − 3/4)는 기함수. 이는 두 조건 어디에서도 "기함수여야 한다"고 명시하지 않았지만, 구간 대칭 최댓값 + 적분 소멸이라는 두 조건의 교차에서 자연스럽게 강제된다.

3. **끝점값 = 극값**: f(1) = 1/4 = f(−1/2)(극댓값), f(−1) = −1/4 = f(1/2)(극솟값). 즉, 양 끝점에서의 함수값이 각각 극댓값·극솟값과 정확히 일치한다 — 이것이 구간 최댓값 대칭의 숨은 기하학적 이유이다.

[STEP 3] 수학적 상황 서술

**수학적 객체**: 최고차항의 계수가 1이고 이차항이 없는 삼차함수 f(x) = x³ + px + q (변곡점이 y축 위에 위치).

**제약 A** (구간 최댓값 대칭):
f(x)가 구간 [-1, 0] 위에서 갖는 최댓값과 구간 [0, 1] 위에서 갖는 최댓값이 서로 같다.
→ 이 조건은 극점의 위치 s = √(−p/3)에 대한 방정식 (s+1)²(2s−1) = 0을 만들어 **p = −3/4**를 결정하지만, 상수항 q는 양변에서 상쇄되어 미결로 남는다.

**제약 B** (곱미분 적분 소멸):
∫₋₁¹ f(x)·f′(x) dx = 0.
→ 곱의 미분법 역이용으로 [f(x)²/2]₋₁¹ = 0, 즉 f(1)² = f(−1)²로 환원된다. 이것은 **"q = 0" 또는 "p = −1"**이라는 논리적 분기를 생성한다. 각 분기 내에서 나머지 파라미터 하나가 자유.

**공유 파라미터**: p와 q 모두가 두 제약에 의해 공유된다. A는 p를 고정하고 q에 무관하며, B는 p 또는 q 중 하나를 고정하는 분기를 만든다.

**강제 결과**: A가 강제한 p = −3/4는 B의 분기 "p = −1"과 양립 불가능하므로, 이 분기가 소멸한다. 유일하게 생존하는 분기 "q = 0"에 의해 f(x) = x³ − (3/4)x가 완전히 결정된다. 이는 기함수이며, 끝점값과 극값이 절대값에서 일치하는 비자명한 성질을 갖는다.

**영감 source_pid**: **P020** (구간 평행이동 극값집합 불변성), **P030** (곱미분 역조립에 의한 함수계 축약)

[STEP 4] 수험생의 발견 순간

1. **처음 시도**: 제약 A를 먼저 전개한다. f′(x) = 3x² + p = 0의 근 x = ±√(−p/3)를 구하고, 구간 [-1, 0]과 [0, 1] 각각에서 극점·끝점의 함수값을 나열하여 최댓값을 비교한다. 경우 분류 후 등치 방정식 2s³ + 3s² − 1 = 0을 세우고, 인수분해 (s+1)²(2s−1) = 0으로 **s = 1/2, p = −3/4**를 얻는다.

2. **막히는 순간**: p는 결정됐지만 q가 어디에도 등장하지 않아 f가 유일하게 결정되지 않는다. "q는 아무 값이나 가능한 것인가?"라는 의문이 생긴다. 제약 A만으로는 더 이상 진행이 불가능하다.

3. **제약 B를 끌어옴**: ∫₋₁¹ f·f′ dx = 0을 전개한다. 처음에는 x³·(3x²+p) + (px+q)·(3x²+p) 등을 직접 적분하려 하여 복잡해진다. 그러다 **f·f′ = (f²)′/2**임을 인식하면 적분이 끝점 평가 f(1)² = f(−1)²로 극적으로 축약된다. 이를 전개하면 (1+p+q)² = (1+p−q)²이므로 **q = 0 또는 p = −1**이라는 분기가 나타난다.

4. **발견**: "p = −1은 제약 A에서 p = −3/4로 이미 확정된 것과 모순이니 **불가능**하다! 그러면 남은 건 q = 0뿐이다." 여기서 학생은 깨닫는다 — **두 조건 어디에도 '기함수'라는 요구가 없었는데, 구간 최댓값의 좌우 대칭과 적분 소멸이 합쳐지니 f(x) = x³ − (3/4)x라는 기함수가 강제된다.** 더 나아가, 이 함수에서 f(1) = 1/4 = f(−1/2)(극댓값)이고 f(−1) = −1/4 = f(1/2)(극솟값)이므로, 끝점값과 극값이 절대값에서 정확히 일치한다는 숨은 기하학적 사실을 발견한다.
