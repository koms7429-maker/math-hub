## 디테일 빌더

### 완성된 문제
> 최고차항의 계수가 $1$인 삼차함수 $f(x) = x^3 + ax + b$ 에 대하여
> 방정식 $f(x) = 0$이 서로 다른 세 실근 $\alpha,\;\beta,\;\gamma\;(\alpha < \beta < \gamma)$를 갖는다.
> 다음 조건을 만족시킬 때, $f(2)$의 값을 구하시오.
>
> **(가)** 방정식 $f\!\bigl(x - f(x)\bigr) = 0$의 서로 다른 실근의 개수는 $7$이다.
>
> **(나)** $|\alpha| + |\beta| + |\gamma| = \sqrt{2}$
>
> **\[답: $7$\]**
