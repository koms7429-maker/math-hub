## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: 극대·극소를 모두 갖는 삼차함수 $f(x)$. 극대점 $c_1$, 극소점 $c_2$ ($c_1 < c_2$). 변곡점 $p = \frac{c_1+c_2}{2}$. 고정 길이 $L$의 닫힌 구간 $[t,\, t+L]$.

**사실 A가 부과하는 제약** (P020 — 구간 이동 극값집합 불변):
서로 다른 두 위치 $t_1 \neq t_2$에서 구간 $[t_i,\, t_i+L]$ 위의 $f$의 치역이 동일하다. 이를 위해 구간의 끝점이 극값 수준집합 위에 놓여야 하는 전이 조건이 발생한다. 구체적으로, 어떤 끝점에서 $f(\text{끝점}) = f(c_1)$ 또는 $f(\text{끝점}) = f(c_2)$를 만족해야 한다.

**사실 B가 부과하는 제약** (P013 — 임계값의 이중근 구조):
방정식 $f(x) = f(c_1)$에서 $x = c_1$은 이중근이고, 세 번째 근 $s_1$은 비에타 공식에 의해 $s_1 = 3p - 2c_1$로 유일하게 결정된다. 마찬가지로 $f(x) = f(c_2)$의 세 번째 근은 $s_2 = 3p - 2c_2$.

**숨은 사실 (A+B 동시 성립 시 강제)**:
그림자 근까지의 거리를 계산하면:

$$s_1 - c_1 = 3p - 3c_1 = \frac{3}{2}(c_2 - c_1), \qquad c_2 - s_2 = 3c_2 - 3p = \frac{3}{2}(c_2 - c_1)$$

즉, **양쪽 그림자 근의 거리가 모두 $\frac{3}{2}d$** ($d = c_2 - c_1$)로 동일하다. 이 비율 $\frac{3}{2}$는 삼차함수의 계수·근 배치에 전혀 의존하지 않는 보편 상수이다. 따라서 구간 이동 문제에서 전이가 일어나는 끝점 위치, 허용 구간 길이, 이동 가능 범위 등이 모두 임계점 간격 $d$만으로 표현되며, 함수의 다른 세부 사항은 소거된다.

---

### 수험생의 발견 순간
1. **처음 시도하는 것**: 구간 $[t,\, t+L]$에서 삼차함수의 최댓값·최솟값이 변하지 않는 조건을 세우기 위해, 극값이 구간 안에 있는지 확인하고, 끝점 함숫값과 극값을 비교하는 부등식을 정리한다.

2. **막히는 순간**: 전이 조건 $f(\text{끝점}) = f(\text{극점})$을 풀어야 하는데, 삼차방정식이 나와서 근을 직접 구하기 어렵다. 매개변수가 많아 일반적인 풀이가 불가능해 보인다.

3. **발견**: $f(x) = f(c_1)$을 전개하면 $x = c_1$이 극점이므로 **자동으로 이중근**이 된다는 것을 인식한다. $(x - c_1)^2$으로 인수분해하면 나머지 단근 $s_1$이 비에타 공식으로 즉시 결정되고, 계산하면 $s_1 - c_1 = \frac{3}{2}(c_2 - c_1)$. 극소점에 대해서도 동일한 비율이 나온다. **"어? 삼차함수가 뭐든 간에 그림자 근의 거리가 항상 임계점 간격의 정확히 3/2배네?"**

4. **STEP 2-2의 숨은 사실과의 대응**: 이 발견은 "비에타에 의한 그림자 근 위치 → 보편 비율 $3/2$"라는 숨은 사실과 정확히 대응한다. 이후 구간 길이 $L$과 임계점 간격 $d$의 관계, 이동 가능 범위 등이 모두 이 비율 하나로 귀결되어 문제가 급격히 단순화된다.

# [STEP 1] 구조 분석

## 1-1. 조건이 만드는 방정식 체계

**무대 정리:**

$$f(x) = x^3 - 12x, \quad f'(x) = 3(x-2)(x+2)$$

| 점 | $x$ | $f(x)$ |
|---|---|---|
| 극대 $\alpha$ | $-2$ | $16$ |
| 극소 $\beta$ | $2$ | $-16$ |
| 변곡점 | $0$ | $0$ |

극대–극소 차: $f(\alpha) - f(\beta) = 32$.

**수준집합 (shadow root) 분석:**

$$f(x) = 16 \;\Longrightarrow\; x^3 - 12x - 16 = (x+2)^2(x-4) = 0 \quad\Rightarrow\quad x = -2\text{(이중근)},\; x = 4$$

$$f(x) = -16 \;\Longrightarrow\; x^3 - 12x + 16 = (x-2)^2(x+4) = 0 \quad\Rightarrow\quad x = 2\text{(이중근)},\; x = -4$$

따라서 $f(x) \in [-16,\,16]$인 영역은 $x \in [-4,\,4]$이다 ($f$의 증감을 감안하면, $-4$와 $4$ 바깥에서 $|f| > 16$).

**$h(t) = M(t) - m(t)$의 분석:**

$h(t) \geq 32$가 **모든** $t$에서 성립하려면, 어떤 위치의 길이-$L$ 구간이든 $f$의 진동 폭이 32 이상이어야 한다.

**핵심 방정식 체계:**

(A) 구간 $[t, t+L]$이 두 극점을 모두 포함하는 경우($t \leq -2$, $t+L \geq 2$):
$$h(t) = 32 \iff f(x) \in [-16, 16] \;\forall x \in [t,\,t+L] \iff t \geq -4 \;\text{and}\; t+L \leq 4$$

(B) 구간이 극점 하나만 포함하거나 단조 영역에 있는 경우: $L$이 충분히 커야 $h(t) \geq 32$.

- 극대만 포함($t < -2$, $t+L < 2$): $h(t) = 16 - \min(f(t), f(t+L))$이고, $f(t+L) > -16$ (for $t+L < 2$), $f(t) > -16$ (for $t > -4$)이므로 $h(t) < 32$.
  → 이런 $t$가 존재하지 않으려면 $2 - L \leq -4$, 즉 $L \geq 6$.

- 극소만 포함(대칭): 역시 $L \geq 6$ 필요.

- 순단조 영역($t+L \leq -2$ 또는 $t \geq 2$): $\min_t h(t) = L^3 + 6L^2$이고, $L^3 + 6L^2 - 32 = (L-2)(L+4)^2 \geq 0$ iff $L \geq 2$. (binding보다 약함)

**결론:** 조건 (가)를 만족하는 $L$의 범위는 $L \geq 6$.

등호 집합 $\{t : h(t) = 32\}$는:
$$t \geq -4,\quad t + L \leq 4,\quad t \leq -2,\quad t + L \geq 2$$

$L \geq 6$이면 $2 - L \leq -4$이므로 세 번째·네 번째 조건은 첫 번째·두 번째에 의해 자동 충족. 따라서:

$$\{t : h(t) = 32\} = [-4,\; 4 - L]$$

이것이 길이 $a > 0$인 닫힌 구간이 되려면:

$$a = (4 - L) - (-4) = 8 - L > 0 \quad\Longrightarrow\quad L < 8$$
