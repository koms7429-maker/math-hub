### [STEP 1] 독립 풀이

#### 1-1. 조건별 독립 전개 + 자유도 분석

**일반 설정:** $f(x)=x^3+ax^2+bx+c$ (최고차항의 계수 1).

---

**조건 (가) 단독:**

$$f''(x)=6x+2a,\quad f''(0)=2a=0 \;\Longrightarrow\; a=0$$

$$\therefore\; f(x)=x^3+bx+c,\quad f'(x)=3x^2+b$$

- 결정되는 파라미터: $a=0$
- 미결 파라미터: $b,\,c$
- **자유도: 2**

---

**조건 (나) 단독 (with (가) 적용 후):**

$f'(x)=3x^2+b=0$의 실근 조건 $b<0$, $s=\sqrt{-b/3}>0$에서 극점 $x=\pm s$.

| 구간 | 후보 | 함수값 |
|------|------|--------|
| $[-1,0]$ | $x=-s$ (극대) | $f(-s)=2s^3+c$ |
| | $x=-1$ | $f(-1)=-1+3s^2+c$ |
| | $x=0$ | $f(0)=c$ |
| $[0,1]$ | $x=s$ (극소) | $f(s)=-2s^3+c$ |
| | $x=0$ | $f(0)=c$ |
| | $x=1$ | $f(1)=1-3s^2+c$ |

$[-1,0]$에서 최댓값: $f(-s)$와 $f(-1)$ 비교.

$$f(-s)-f(-1)=2s^3-(-1+3s^2)=2s^3-3s^2+1=(s-1)^2(2s+1)>0\;\;(0<s<1)$$

따라서 $\max_{[-1,0]}f = f(-s)=2s^3+c$.

$[0,1]$에서 최댓값: $s<1/\sqrt3$이면 $\max_{[0,1]}f=f(1)=1-3s^2+c$.

등치 방정식:

$$2s^3+c=1-3s^2+c \;\Longrightarrow\; 2s^3+3s^2-1=0$$

$$\boxed{(2s-1)(s+1)^2=0} \;\Longrightarrow\; s=\tfrac12\;\Longrightarrow\; b=-3s^2=-\tfrac34$$

검증: $s=\tfrac12<1/\sqrt3\approx0.577$ ✓

- 결정되는 파라미터: $b=-3/4$
- 미결 파라미터: $c$
- **자유도: 1** (가+나 결합 후)

---

**조건 (다) 단독 (with (가) 적용 후):**

$$\int_{-1}^{1}f(x)f'(x)\,dx = \left[\frac{f(x)^2}{2}\right]_{-1}^{1}=\frac{f(1)^2-f(-1)^2}{2}=0$$

$$\Longrightarrow\; f(1)^2=f(-1)^2$$

$f(1)=1+b+c$, $f(-1)=-1-b+c$이므로

$$(1+b+c)^2=(-1-b+c)^2$$

전개하면 $(1+b+c)^2-(c-1-b)^2=0$, 합차공식 적용:

$$[(1+b+c)+(c-1-b)]\cdot[(1+b+c)-(c-1-b)]=0$$

$$[2c]\cdot[2+2b]=0$$

$$\boxed{c=0 \;\;\text{또는}\;\; b=-1}$$

- 미결 파라미터: 두 분기 각각에서 하나 남음
- **자유도: 1** (가+다 결합 후)

---

#### 1-2. 얽힘 검증

| 검증 항목 | 결과 | 근거 |
|----------|------|------|
| (나)의 미결 파라미터 $c$가 (다)의 등식에 등장하는가 | **O** | (다)의 분기 조건 $c=0$ 또는 $b=-1$에서 $c$가 직접 등장 |
| (다)의 미결 파라미터 $b$가 (나)의 등식에 등장하는가 | **O** | (나)의 등식 $2s^3+3s^2-1=0$은 $b=-3s^2$을 통해 $b$를 결정; (다)의 분기 $b=-1$은 (나)가 고정한 $b$와 직접 충돌 |
| (나)+(다)의 자유도가 각 단독 자유도보다 2 이상 줄어드는가 | **O** | (가) 단독: 2, (가+나): 1, (가+다): 1, **(가+나+다): 0**. 전체 시스템 자유도 $2\to0$, 순감소 2 |
| 연립 과정에서 단독 전개에 없던 새로운 등식이 나타나는가 | **O** | (나)에서 $b=-3/4$가 (다)의 $b=-1$ 분기를 **소멸**시키며, 이 소멸은 어느 조건 단독에서도 나타나지 않는 새로운 강제 결과 $c=0$을 산출 |

**얽힘 등급: 강한 얽힘 (4/4 O)**

---

#### 1-3. 조건 연립 + 풀이

**연립 순서:** (가) → (나) → (다)

1. **(가)** 적용: $f(x)=x^3+bx+c$, 자유도 2.

2. **(나)** 적용: 구간 최댓값 분석 → $(2s-1)(s+1)^2=0$ → $s=1/2$ → $b=-3/4$. 자유도 1 ($c$ 미결).

   **→ 전환 지점 ①:** $b$는 결정되었으나 $c$가 (나)의 등치 방정식 양변에서 상쇄되어 미결. "더 이상 (나)만으로 진행 불가 → (다)를 끌어와야 한다."

3. **(다)** 적용: $\int_{-1}^1 ff'\,dx=0$ → $ff'=(f^2)'/2$ 인식 → $f(1)^2=f(-1)^2$.

   **→ 전환 지점 ②:** 직접 전개($x^3\cdot(3x^2+b)$ 등) 대신 **곱의 미분법 역이용** $ff'=\frac{d}{dx}\frac{f^2}{2}$을 발견하여 적분을 끝점 평가로 축약.

   $f(1)=1/4+c$, $f(-1)=-1/4+c$.
   
   $(1/4+c)^2=(-1/4+c)^2$ 전개:
   
   $$\tfrac{1}{16}+\tfrac{c}{2}+c^2 = \tfrac{1}{16}-\tfrac{c}{2}+c^2 \;\Longrightarrow\; c=0$$

   (또는 합차공식으로: $2c\cdot(2b+2)=2c\cdot(-3/2+2)=2c\cdot(1/2)=c=0$.)

   **→ 전환 지점 ③ (핵심):** 일반적으로 (다)는 "$c=0$ 또는 $b=-1$" 분기를 생성하지만, (나)에서 확정된 $b=-3/4\neq-1$이 $b=-1$ 분기를 소멸시켜 $c=0$이 **강제**됨.

4. **최종:** $f(x)=x^3-\frac34x$.

$$2f(2)=2\!\left(8-\frac32\right)=2\cdot\frac{13}{2}=\boxed{13}$$

---

#### 1-4. 유일성 확인

$a=0$, $b=-3/4$, $c=0$이 모두 유일하게 결정됨. 해 $f(x)=x^3-\frac34x$는 유일. → **O**

---

#### 1-5. 조건 과잉/부족 체크

- **(가) 제거:** $f(x)=x^3+ax^2+bx+c$로 3 파라미터. (나)+(다)만으로는 2개의 구속만 생기므로 자유도 1 잔존. 답 비유일. → **(가) 필요** ✓

- **(나) 제거:** (가)+(다)만으로 $b=-1$ 또는 $c=0$의 두 분기, 각 분기에서 자유도 1. 답 비유일. → **(나) 필요** ✓

- **(다) 제거:** (가)+(나)에서 $f(x)=x^3-\frac34x+c$, $c$ 자유. $2f(2)=13+2c$로 답 비유일. → **(다) 필요** ✓

**모든 조건 필수, 과잉 없음.**

### [STEP 2] 설계 의도 대조

설계 의도의 핵심 메커니즘:
1. (가) → 이차항 소멸로 $f=x^3+px+q$ 형태 강제 ✓ — STEP 1에서 동일하게 작동
2. (나) → $p=-3/4$ 결정, $q$ 무관 ✓ — STEP 1-3 전환 지점 ①에서 정확히 확인
3. (다) → $(f^2)'/2$ 역이용 → $f(1)^2=f(-1)^2$ → "$q=0$ ∨ $p=-1$" 분기 ✓ — 전환 지점 ②에서 확인
4. **분기 소멸:** $p=-3/4\neq-1$로 $p=-1$ 분기 탈락 → $q=0$ 강제 ✓ — **전환 지점 ③에서 정확히 나타남**

> **설계 의도의 "강제 결과/숨은 사실"이 STEP 1-3에서 실제로 나타났는가?** → **O**
> 
> 나타난 위치: 전환 지점 ③ — (나)의 $b=-3/4$가 (다)의 $b=-1$ 분기를 소멸시키는 순간.

추가로, $f(x)=x^3-\frac34x$가 기함수이며 $f(1)=\frac14=f(-\frac12)$(극댓값)인 대칭적 성질도 설계 의도와 일치.
