# 문제 초안 작성

## [0] 이번 시도의 차별성 선언

- **조건 형태**: 대수적 등식/부등식 → **기하적 접선 조건** (이유: "중근을 갖는다"를 "$y=f(2)$가 곡선에 접한다"로 바꾸면, 제약 B가 대수가 아닌 기하 언어로 포장되어 핵심 구조(Vieta 분석)가 표면에서 사라짐)
- **숨은 사실의 표현**: 직접적 판별식 강제 → **간접적 이중 접선 구조** (이유: 조건 (나)의 접선과 조건 (다)가 강제하는 접선이 서로 다른 수평선이므로, 두 접선이 연결된다는 사실이 조건 배치만으로는 보이지 않음)

---

## [무대 설정]

삼차함수 $f(x) = x^3 + ax + b$ ($a, b$는 실수, $b > 0$). 파라미터는 $a, b$.

---

## [조건 배치 계획]

| 수학적 상황 요소 | 번역될 조건 | 설계 의도 |
|---|---|---|
| $a < 0$ (극대·극소 존재) | **(가)** 극댓값과 극솟값을 갖는다 | 무해한 전제 조건. $a<0$만 강제 |
| 제약 B: 극대 수평선이 곡선과 $x=k$에서 재교차 → $a=-3k^2/4$ | **(나)** 직선 $y=f(2)$가 곡선에 $x\neq 2$인 점에서 접한다 | $k=2$를 고정. "접한다"는 이중근 존재의 기하적 위장. $a=-3$만 결정되고 $b$는 미결 |
| 제약 A: $\|f(x)\|=\|f(0)\|$의 근이 5개 → $b^2 = -a^3/27$ | **(다)** $\|f(x)\|=f(0)$의 서로 다른 실근의 개수가 5 | $b>0$이므로 $\|f(0)\|=f(0)$으로 자연스럽게 처리. $a$를 알아야 $b$가 나오므로 (나)와 필연적 연립 |
| 숨은 사실: $-f(0)=$ 극솟값 ↔ $y=-b$가 극소점에서 접선 | **어떤 조건에도 명시되지 않음** | (나)로 $a$를 구한 뒤 (다)의 판별식을 풀면 $b=1$이 나오고, 그 이유가 $-b=$극솟값임을 사후적으로 깨달음 |

**독립성 검증**: (나)만으로는 $a$만 결정 → $b$ 미결. (다)만으로는 $a,b$ 관계식 하나 → 둘 다 미결. 두 조건을 연립해야만 $(a,b)=(-3,1)$이 확정된다.

## [문제 초안]

> 실수 $a$, $b$에 대하여 함수 $f(x) = x^3 + ax + b$ $(b > 0)$이
> 다음 조건을 만족시킨다.
>
> **(가)** 함수 $f(x)$는 극댓값과 극솟값을 갖는다.
>
> **(나)** 직선 $y = f(2)$와 곡선 $y = f(x)$는 $x \neq 2$인 점에서 접한다.
>
> **(다)** 방정식 $|f(x)| = f(0)$의 서로 다른 실근의 개수는 $5$이다.
>
> $f(2)$의 값을 구하시오.

---

## [조건-상황 대응표]

| 조건 | 수학적 상황의 요소 | 수험생에게 보이는 역할 |
|---|---|---|
| $b>0$ | $f(0)>0$ 보장, $\|f(0)\|=f(0)$으로 (다) 단순화 | 부호 정리용 부가 조건 |
| **(가)** | $a<0$ 강제 (극값 존재) | 함수 형태 제한 |
| **(나)** | 제약 B → $a=-3k^2/4$, $k=2$ 고정 → $a=-3$ | 기하 조건(접선)으로 위장된 $a$ 결정 장치 |
| **(다)** | 제약 A → $b^2=-a^3/27$, $a=-3$ 대입 → $b=1$ | 근의 개수 조건으로 위장된 $b$ 결정 장치 |
| (숨은 사실) | $-f(0)=$ 극솟값 $\Leftrightarrow$ $y=-b$가 극소점 접선 | **어디에도 명시되지 않음** — (나)+(다) 연립 과정에서 자연 발견 |
