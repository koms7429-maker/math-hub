> **(가)** 방정식 $f(x)=0$은 서로 다른 세 실근을 가지며, $f(\alpha)=f(-\alpha)=0$인 양수 $\alpha$가 존재한다.

> **(나)** 방정식 $f(f(x))=0$은 $x=0$을 근으로 가지며, 서로 다른 실근의 개수는 $8$이다.

$f(0)$의 값을 구하시오.

---

## [조건-상황 대응표]

| 조건 | 수학적 상황 요소 |
|---|---|
| **(가)** 전반부 "서로 다른 세 실근" | 삼차함수의 세 실근 존재 및 극대·극소 보유 |
| **(가)** 후반부 "$f(\alpha)=f(-\alpha)=0$" | **제약 A**: $f(x)=(x^2-d^2)(x-r)$ 꼴 인수분해 강제 (접선-재교점 영점 조건의 결과물) |
| **(나)** "$x=0$이 근" | **제약 B 전반**: $f(f(0))=0$ → $f(0)=rd^2$이 $f$의 근 → $r$과 $d$의 비선형 관계 |
| **(나)** "실근의 개수 $8$" | **제약 B 후반**: 근값 중 하나가 극값과 일치하는 경계조건 → 남은 자유 파라미터 소진 |
| **발문** $f(0)$ | **강제 결과**: 확정된 $r$, $d$로부터 $f(0) = rd^2$의 수치 결정 |
