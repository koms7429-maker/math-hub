## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
- **수학적 객체**: 최고차항의 계수가 1인 삼차함수 $f(x) = x^3 + ax^2 + bx + c$
- **제약 A**: $f(0) = 0$이고 $f'(0) = 9$
- **제약 B**: 양의 실수 $t$가 존재하여 $f(t) = 0$이고 $f'(t) = 0$
- **공유 파라미터**: 계수 $a$는 A에서 미결이고 B의 $t, r$ 매개화에도 등장. $b$는 A가 고정하지만 B에서 $t^2+2tr$로 표현되어 $t$를 구속. $c$는 A가 0으로 고정하지만 B에서 $-t^2 r$로 표현되어 $r$을 구속.
- **강제 결과**: $f(x) = x(x-3)^2$. 원점에서의 기울기 9가 이중근의 위치 $t=3$을 $b = t^2$을 통해 결정하고, 원점 통과 조건이 단근을 원점에 고정.
- **영감 source_pid**: **P022** (비음수 분해 영점 고정 — $t^2 r = 0$에서 $t^2 \ge 0$, $t>0$이므로 $r=0$ 강제), **P028** (곱미분 역조립의 미분적합성 — 이중근 조건 $f(t)=f'(t)=0$이 함수값·도함수 동시 일치를 요구)

---

### 수험생의 발견 순간
1. **처음 시도**: A를 적용 → $c=0$, $b=9$ → $f(x) = x^3 + ax^2 + 9x$. "$a$만 결정하면 되는데…"
2. **막히는 순간**: B에서 이중근 $t>0$이 존재해야 하므로 $f(t)=0$과 $f'(t)=0$을 세우지만, 미지수가 $a$와 $t$ 두 개이고 방정식이 두 개 — 풀 수 있을 것 같지만 $a$와 $t$가 얽혀 있어 직접 풀기가 막막함.
3. **연립 과정**: $f(t)=0$에서 $t^2 + at + 9 = 0$ …(ⓘ), $f'(t)=0$에서 $3t^2 + 2at + 9 = 0$ …(ⓙ). (ⓙ)−(ⓘ)를 계산하면 $2t^2 + at = 0$ → $a = -2t$. 이를 (ⓘ)에 대입: $t^2 - 2t^2 + 9 = 0$ → $t^2 = 9$.
4. **발견**: "$t = 3$이네! $f'(0) = 9$에서 나온 $b = 9$가 곧 $t^2$이었구나. 원점의 기울기가 이중근까지의 거리를 결정하다니!" → $a = -6$, $f(x) = x(x-3)^2$.
