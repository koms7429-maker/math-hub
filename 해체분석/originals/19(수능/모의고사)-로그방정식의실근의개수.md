# 19. (수능/모의고사) - 로그 방정식의 실근의 개수

## Problem

상수 $k\ (k > 4\sqrt{5})$와 2 이상의 자연수 $n$에 대하여 $x$에 대한 방정식

$$\frac{\log_n (x^2-20)^n}{n} = \frac{\log_n (2x+k)^{n+1}}{n+1} + 1$$

의 실근의 개수가 3이 되도록 하는 모든 $n$의 값이 $n_1,\; n_2\ (n_2 > n_1 \geq 2)$일 때, $k + n_2 - n_1$의 값은? [4점]

## Choices
① 10 &emsp; ② 11 &emsp; ③ 12 &emsp; ④ 13 &emsp; ⑤ 14

**Answer**: ①
**Difficulty**: 상
**Score**: 4점
**Type**: multiple choice

---

## Solution

문제의 방정식을 정리하면

**① $n$이 홀수일 때,** $\log_n(x^2-20) = \log_n|2x+k| + 1$

$$\Rightarrow x^2 - 20 = 2n\left|x+\frac{k}{2}\right|,\quad x^2-20>0,\quad x \neq -\frac{k}{2}$$

**② $n$이 짝수일 때,** $\log_n|x^2-20| = \log_n(2x+k) + 1$

$$\Rightarrow |x^2-20| = 2n\left(x+\frac{k}{2}\right),\quad x^2 \neq 20,\quad x > -\frac{k}{2}$$

이때, $k > 4\sqrt{5} \Rightarrow -\dfrac{k}{2} < -2\sqrt{5}$

임을 함께 고려하면 방정식의 실근의 개수가 3이 되는 경우를 다음 그림과 같이 나타낼 수 있다.

[FIGURE: ①의 경우 — $y = x^2 - 20$과 $y = 2n\left|x + \frac{k}{2}\right|$의 그래프가 $x < -2\sqrt{5}$에서 접하는 그림. ②의 경우 — $y = |x^2 - 20|$과 $y = 2n\left(x + \frac{k}{2}\right)$의 그래프가 $-2\sqrt{5} < x < 2\sqrt{5}$에서 접하는 그림.]

**①의 경우:**

$y = x^2 - 20$, $y = -2n\!\left(x + \dfrac{k}{2}\right)$이 $x < -2\sqrt{5}$에서 접함

$$\Rightarrow x^2 + 2nx + kn - 20 = 0 \text{이 } x < -2\sqrt{5} \text{인 중근을 가짐}$$

$$\Rightarrow \begin{cases} D/4 = n^2 - kn + 20 = 0 \cdots \text{ⓐ} \\ \text{(대칭축)} = -n < -2\sqrt{5} \Rightarrow n > 2\sqrt{5}\text{인 홀수} \\ \Rightarrow n = 5,\; 7,\; \cdots \quad \cdots \text{ⓑ} \end{cases}$$

**②의 경우:**

$y = -(x^2-20)$, $y = 2n\!\left(x + \dfrac{k}{2}\right)$이 $-2\sqrt{5} < x < 2\sqrt{5}$에서 접함

$$\Rightarrow x^2 + 2nx + kn - 20 = 0 \text{이 } -2\sqrt{5} < x < 2\sqrt{5}\text{인 중근을 가짐}$$

$$\Rightarrow \begin{cases} \text{중근을 가질 조건은 ⓐ과 동일} \\ -2\sqrt{5} < -n < 2\sqrt{5}\ (\because \text{ⓑ의 과정 참고}) \\ \Rightarrow n \text{은 } -2\sqrt{5} < n < 2\sqrt{5}\text{인 짝수} \\ \Rightarrow n = 2,\; 4 \quad \cdots \text{ⓒ} \end{cases}$$

따라서 문제의 조건과 ⓐ에 의해

$$\begin{cases} n\text{에 대한 이차방정식 } n^2 - kn + 20 = 0 \text{이} \\ \text{서로 다른 두 실근이 } n_1,\; n_2\ (n_2 > n_1 \geq 2) \end{cases}$$

임을 알 수 있고, 이때

$$n_1 \times n_2 = 20 \quad (\because \text{근과 계수의 관계})$$

이므로 ⓑ, ⓒ에서

$$n_1 = 4,\quad n_2 = 5$$

임을 알 수 있고,

$$\text{ⓐ} \Rightarrow k = 9$$

를 얻는다.

$$\therefore\; k + n_2 - n_1 = 9 + 5 - 4 = \boxed{10}$$
