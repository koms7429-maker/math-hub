## 프라블럼 메이커

### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x) = x^3 + ax + b$ 에 대하여
> 방정식 $f(x) = 0$ 이 서로 다른 세 실근 $\alpha, \, \beta, \, \gamma$ $(\alpha < \beta < \gamma)$ 를 갖는다.
> 다음 조건을 만족시킬 때, $f(T)$ 의 값을 구하시오.
>
> **(가)** 방정식 $f(x - f(x)) = 0$ 의 서로 다른 실근의 개수는 $N$ 이다.
>
> **(나)** $|\alpha| + |\beta| + |\gamma| = S$

*(여기서 $N$, $S$, $T$는 수치 결정 단계에서 확정할 상수)*

##

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 숨은 사실 |
|---|---|---|
| **문두**: $f(x)=x^3+ax+b$ | 이차항 계수 0인 삼차함수 | α+β+γ = 0 (비에타) → 위치값 합 3b 고정, 절댓값합 축약 |
| **문두**: 서로 다른 세 실근 | a < 0 (극대·극소 존재), 세 근 실수 | 판별식이 양 → a, b 범위 제한 |
| **(가)**: $f(x-f(x))=0$의 근의 개수 $= N$ | 제약 A: 세 평행선 $y=x-r_i$와 곡선 교점 수의 합 | 기울기 1 평행선 분해; 임계 간격 $D$와 위치값 비교; 접선 분기 발생 |
| **(나)**: $\lvert\alpha\rvert+\lvert\beta\rvert+\lvert\gamma\rvert = S$ | 제약 B: 절댓값합 등식 | 영합근 조건과 결합 시 $S=2\times$(양근의 합)으로 환원 → 부호 분배 확정 → 접선 분기 선별 |
