#### 2-2. 얽힘 판정

- [✓] A의 미결 파라미터 $(a, b)$ 중 $b$를 B가 방정식으로 고정함 (판별식 $\Delta_2=0$ 조건이 $b$를 $a$의 함수로 결정)
- [✓] B의 미결 파라미터 $(a,b)$에 대해, A가 $b\ge a^2/4$이라는 부등식을 부과하여 B의 해집합을 절단함 (B 단독이면 $a$가 자유이지만, A를 더하면 $a\le -3/2$로 범위 강제)
- [✓] A 단독 자유도 ≈ 2, B 단독 자유도 ≈ 1, A+B 동시 자유도 ≤ 1 (차이 ≥ 2 이상 감소)

#### 2-3. 강제 결과 서술

A+B를 동시에 만족시키면:

> **$x\ge 0$ 분기에서 이차식이 양의 이중근을 가져야 하고 (근 1개 추가), $x\le 0$ 분기에서 음의 근이 정확히 2개여야 한다.** A의 부등식 $b\ge a^2/4$가 이중근 조건 $b=(a^2-2a-3)/4$와 결합하면 등호 $b=a^2/4$는 $a=-3/2$에서만 양립한다.

즉, **$a=-3/2$이고 $b=9/16$이 유일해가 되거나**, 혹은 부등식과 근 개수 조건이 결합하여 $a$의 특정 범위에서 $b$가 유일하게 결정되며, 그 범위의 끝점에서 비자명한 등식이 성립한다.

수험생이 발견할 핵심: **"부호 부등식이 근의 개수 조건과 만나면, 이차식의 판별식이 0이 되는 접선 조건과 이차식 계수의 부호 부등식이 정확히 한 점에서 교차하여, $a$와 $b$가 동시에 확정된다."**

[STEP 3] 수학적 상황 서술

- **수학적 객체**: 최고차항의 계수가 1이고 원점을 지나는 삼차함수 $f(x) = x^3 + ax^2 + bx$

- **제약 A** (부호인수 부등식): 모든 실수 $x$에 대하여 $x\cdot f(x) \ge 0$이 성립한다.
  → $f(x)$가 양의 $x$에서 양, 음의 $x$에서 음이어야 하므로, 이차인수 $x^2+ax+b$가 전역 비음수 → $a^2-4b\le 0$.

- **제약 B** (절댓값 수준집합의 교차-접촉 분류): 방정식 $|f(x)| = |f(1)|$의 서로 다른 실근의 개수가 정확히 4개이다.
  → $f$의 부호 구조에 의해 $|f(x)|$이 $x\ge 0$과 $x\le 0$에서 다른 형태를 가지며, 각 분기의 근의 개수가 $(a,b)$에 의존.

- **공유 파라미터**: $b$. 제약 A는 $b$의 하한($b\ge a^2/4$)을, 제약 B는 $b$의 정확한 값(근 개수 조건에서 오는 판별식 등호)을 요구한다. 동시에 $a$에 대해서도, A가 설정한 범위 안에서 B의 판별식 조건이 해를 가지려면 $a$의 범위가 한정된다.

- **강제 결과**: A의 부등식 경계 $b = a^2/4$와 B의 판별식 등호 조건이 교차하는 점에서 $(a,b)$가 유일하게 결정된다. 이는 **$x^2+ax+b$가 완전제곱식이 되는 순간**(즉 $f(x)=x(x+a/2)^2$ 꼴)과 **수준집합이 정확히 접선 조건을 만족하는 순간**이 일치함을 의미한다.
