## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체:** x² 항이 없는 삼차함수 f(x) = x³ + px (p는 실수 매개변수)

**사실 A가 부과하는 제약 (P005):**
곡선 위 두 점 (a, f(a)), (a+h, f(a+h))가 축정렬 정사각형의 대각 꼭짓점이 되려면 f(a+h)−f(a) = ±h. 이를 h로 나누면 (a, h)에 대한 이차방정식이 생기고, 양의 h에 대한 실근 존재 여부가 정사각형의 개수를 결정한다.

**사실 B가 부과하는 제약 (P010):**
변곡점(원점)에서의 접선 기울기 f′(0) = p가 곡선의 "굽힘 정도"를 결정하며, 이것이 정사각형이 곡선에 내접할 수 있는지를 지배한다.

**숨은 사실 (A + B 동시 성립):**
중심좌표로 치환하면 모든 내접 정사각형의 파라미터가 **d² + 3m² = 1 − p** 라는 타원 방정식 위에 놓인다. 따라서:
- p < 1이면 타원이 존재하고, 연속적으로 많은 정사각형이 내접 가능
- p = 1이면 타원이 한 점으로 퇴화 → 정확히 하나의 정사각형
- p > 1이면 해 없음 → 내접 정사각형 불가능

변곡점 접선 기울기가 정확히 1인 것이 임계값이라는 사실은 두 조건을 결합해야 비로소 드러난다.

---

### 수험생의 발견 순간
1. **처음 시도:** 정사각형 조건 f(a+h)−f(a) = h를 전개하고 h로 나눠 h² + 3ah + (3a² + p − 1) = 0을 얻는다. 판별식으로 h의 실근 존재 조건을 구하려 한다.

2. **막히는 순간:** 판별식이 a에 대한 이차식이 되어 "a의 범위에 따라 h가 존재하는 영역"을 분류해야 하는데, 두 변수가 얽혀 있어 정리가 어렵다.

3. **발견:** m = a + h/2, d = h/2로 치환하면 교차항이 모두 소거되고 **d² + 3m² = 1 − p**라는 놀랍도록 단순한 타원이 나타난다. 정사각형의 존재성이 이 타원의 존재(1 − p > 0, 즉 p < 1)로 완전히 판정된다.

4. **숨은 사실과의 대응:** 이 타원의 존재 조건 p < 1은 곧 f′(0) < 1, 즉 변곡점에서의 접선 기울기가 1 미만이라는 조건이다. 차분방정식의 대수적 구조(P005)와 변곡점 기울기의 기하적 의미(P010)가 타원이라는 하나의 객체로 합류하는 것—이것이 수험생이 발견하는 순간이다.

## 3. 디테일 빌더 (Detail Builder)

> 곡선 $C$ 위의 서로 다른 두 점을 대각선의 양 끝점으로 하고,
> 네 변이 모두 좌표축에 평행한 정사각형을
> 곡선 $C$의 **내접 정사각형**이라 하자.

다음 조건을 만족시키는 삼차함수 $f(x)$에 대하여

**(가)** 방정식 $f'(x)=1$은 서로 다른 두 실근 $\alpha$, $\beta$를 갖고, $\alpha\beta=-3$이다.

**(나)** 곡선 $C$의 내접 정사각형의 넓이의 최댓값은 $48$이다.

**(다)** 곡선 $C$의 변곡점의 $x$좌표는 양수이다.

방정식 $f(x)=x$의 모든 실근의 합을 구하시오. $\boxed{3}$

### 원본 문제 B
## Problem

**10.**

$f(0)=0$이고, $f'(0) \neq 0$이고 최고차항의 계수가 1인 삼차함수 $f(x)$와 양의 상수 $k$가 다음 조건을 만족시킨다.

> (가) 직선 $y=f'(0)x$가 두 곡선
> $\qquad y=f(x-k)$, $y=f(x)+k$
> $\qquad$와 만나는 서로 다른 점의 개수는 각각 2이다.
> (나) $f(\alpha)=0$인 어떤 실수 $\alpha$에 대하여 $x$에 대한 방정식
> $\qquad f(x)=m(x-\alpha)$의 서로 다른 실근의 개수가 2 이상이
> $\qquad$되도록 하는 실수 $m$의 최솟값은 $f'(0)$이다.

$f(7k)$의 값을 구하시오.

### 풀이 시도 (STEP 1)
#### 1-1. 조건별 독립 전개

**설정:** f(x) = x³ + ax² + bx (최고차항 계수 1, f(0)=0)

**조건 (가) 단독:**
f'(x) = 3x² + 2ax + b이므로 f'(x) = 1은 3x² + 2ax + (b-1) = 0.
Vieta 정리:
- α + β = -2a/3
- αβ = (b-1)/3 = -3 → **b = -8**

판별식 4a² + 108 > 0은 항상 성립 → a에 대한 추가 제약 없음.
- **결정:** b = -8
- **미결:** a

**조건 (나) 단독:**
내접 정사각형 조건 f(t+h) - f(t) = h (h>0)를 전개하면:
$$h[3t^2 + 3th + h^2 + 2at + ah + b] = h$$
h로 나누고 m = t + h/2, d = h/2 치환:
$$d^2 + 3\!\left(m + \frac{a}{3}\right)^2 = 1 - b + \frac{a^2}{3}$$
이것은 타원 방정식이고, 넓이 h² = 4d²의 최댓값은 m = -a/3일 때:
$$\text{최대 넓이} = 4\!\left(1 - b + \frac{a^2}{3}\right) = 48$$
따라서 **a²/3 - b = 11**, 즉 **a² - 3b = 33**.
- **결정:** a와 b 사이의 관계식 1개
- **미결:** a, b 각각

**조건 (다) 단독:**
f''(x) = 6x + 2a = 0 → 변곡점 x = -a/3.
x > 0이면 **a < 0**.
- **결정:** a의 부호

**요약:** (가)만으로 b 확정되나 a 미결. (나)만으로는 a,b 모두 미결. (다)만으로는 부호만 확정.

---

#### 1-2. 조건 연립

**연립 순서:** (가) → (나) → (다)

1. **(가)에서** b = -8 확정.
2. **(나)와 연립:** a² - 3(-8) = 33 → a² + 24 = 33 → **a² = 9** → a = ±3.
3. **(다)에서** a < 0 → **a = -3**.

따라서 **f(x) = x³ - 3x² - 8x**.

**검증:**
- f'(x) = 3x² - 6x - 8 = 1 → x² - 2x - 3 = 0 → (x-3)(x+1) = 0, αβ = -3 ✓
- 타원: 3(m-1)² + d² = 12, 최대 h² = 48 ✓
- 변곡점 x = 1 > 0 ✓

**발문 계산:**
f(x) = x → x³ - 3x² - 8x = x → x³ - 3x² - 9x = 0 → x(x² - 3x - 9) = 0

Vieta 정리로 모든 실근의 합 = 0 + 3 = **3**

**비자명한 전환:**
1. 내접 정사각형 조건을 전개 후 m, d 치환으로 타원 방정식 도출 (핵심 발견)
2. 타원에서 최대 넓이를 기하적으로 읽어내어 파라미터 방정식 수립
3. f(x) - x의 근의 합을 Vieta로 직접 계산 (개별 근을 구하지 않아도 됨)

---

#### 1-3. 유일성 확인

- (가): b = -8 (유일)
- (가)+(나): a² = 9 (두 해 a = ±3)
- (다): a = -3 선별 → **유일**

**유일성: O** — (다)가 소거 장치로 기능

---

#### 1-4. 조건 과잉/부족 체크

**조건 (가) 제거:** a² - 3b = 33, a < 0만 남음 → 무한히 많은 (a,b) 쌍 → **답 달라짐**

**조건 (나) 제거:** b = -8, a < 0만 남음 → a 미결정 → **답 달라짐**

**조건 (다) 제거:** b = -8, a = ±3 → a = 3이면 f(x) = x³ + 3x² - 8x, f(x) = x의 근의 합 = -3. a = -3이면 합 = 3. **답 달라짐** (3 또는 -3)

**결론:** 세 조건 모두 필수, 과잉/부족 없음.
