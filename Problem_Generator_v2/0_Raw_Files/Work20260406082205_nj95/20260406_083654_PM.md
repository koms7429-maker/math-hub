| 본문 ($f(0)=0$, 최고차 계수 $1$) | 삼차함수 $f(x) = x^3+ax^2+bx$ 설정 | $f$의 형태 확정 |
| **(가)** 서로 다른 세 실근 | 근의 실재성·상이성 보장 | 판별식 부등식 (배경 제약) |
| **(나)** $x$축 교점 두 곳에서 접선 기울기 동일 | **제약 B** — $f'(r_i) = f'(r_j)$ | $r_i + r_j = -\tfrac{2a}{3}$; $a=0$ 또는 한 근 $= -\tfrac{2a}{3}$ |
| **(다)** $f'(0)$ 기울기 접선이 영 아닌 근 통과 | **제약 A** — 평행접선 접촉 | $27r^3+27ar^2-4a^3=0$ → 두 케이스; $a=0$ 배제 |
| **(라)** $f'(0) = N$ | 수치 앵커 (공유 파라미터 $b$ 고정) | $b=N$ → $a^2 = \tfrac{9N}{2}$ → $a$ 확정 |
