## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: 이차함수 $\varphi(x)$와 일차함수 $\psi(x)$가 정확히 한 점 $x = t_0$에서 접하는 구조(값과 도함수 모두 일치). 전역 미분가능 함수 $h(x)$는 모든 점에서 $h(x) = \varphi(x)$ 또는 $h(x) = \psi(x)$를 만족한다.

**사실 A가 부과하는 제약 (P021 — 끼임)**:
두 경계함수 $g(x) \le h(x) \le G(x)$가 주어지고, 특정 점 $x = a$에서 $g(a) = G(a)$이다. 이로 인해 $h(a)$와 $h'(a)$가 완전히 고정된다. 이 고정된 도함수 값은 $\varphi$의 꼭짓점(극값)에서의 도함수(= 0)와 일치하지 않는 특정 비영 값이다.

**사실 B가 부과하는 제약 (P027 — 분기 전환 극값 소거)**:
$h$의 전역 미분가능성에 의해 분기 전환은 $\varphi$와 $\psi$가 접하는 유일한 점 $x = t_0$에서만 가능하다. $\varphi$는 $t_0$의 한쪽에서 극값(꼭짓점)을 가지지만, $t_0$에서 $\psi$로 전환하면 그 극값은 $h$의 그래프에 나타나지 않는다.

**숨은 사실 (A+B 동시 성립 시 강제)**:
끼임 조건이 고정한 $h'(a) \neq 0$이 $\varphi$의 극값 존재와 양립하려면, $h$는 $\varphi$의 꼭짓점에 도달하기 전에 반드시 $\psi$로 전환해야 한다. 전환점은 접점 $t_0$으로 유일하게 결정된다. 이 전환은 $\varphi$의 극값을 정확히 잘라내어 $h$를 예상보다 단조롭게 만들며, 동시에 $t_0$의 위치가 경계함수 $g, G$의 매개변수에 의해 완전히 결정된다.

---

### 수험생의 발견 순간
1. **처음 시도**: 수험생은 $h(x)$가 전역 미분가능하고 매 점에서 $\varphi$ 또는 $\psi$의 값을 취한다는 조건에서, "접점 $t_0$에서만 전환 가능"이라는 것을 확인한다. 그리고 끼임 조건 $g(a) = G(a)$에서 $h(a)$와 $h'(a)$를 계산한다.

2. **막히는 순간**: $h$가 $\varphi$를 따른다고 가정하면, $\varphi$의 꼭짓점 근방에서 $h'$이 0이 되는데, 이것이 경계 $G$를 뚫고 나가거나 $g$ 아래로 내려간다. "분기 전환 없이는 끼임 부등식을 만족시킬 수 없다"는 벽에 부딪힌다.

3. **발견**: "$h$가 접점 $t_0$에서 $\psi$로 전환하면 $\varphi$의 극값이 사라진다! 그러면 $h$는 전환 이후 일차함수(단조)가 되어 끼임 부등식을 자연스럽게 만족한다. 전환이 선택이 아니라 **필수**였던 것이다."

4. **숨은 사실과의 대응**: 이 발견은 STEP 2-2의 숨은 사실—"끼임 조건이 분기 전환을 강제하고, 강제된 전환이 극값을 소거하며, 전환점이 유일하게 결정된다"—에 정확히 대응한다. 부등식이라는 연속적 조건이 분기 전환이라는 이산적 위상 구조를 완전히 지배한다는 것이 핵심적 놀라움이다.

## 2. 프라블럼 메이커 (Problem Maker)

### 문제 초안
> 최고차항의 계수가 $1$인 이차함수 $g(x)$와 일차함수 $l(x)$에 대하여
> 실수 전체의 집합에서 미분가능한 함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** 모든 실수 $x$에 대하여 $f(x) = g(x)$ 또는 $f(x) = l(x)$이다.
>
> **(나)** 모든 실수 $x$에 대하여 $\left|\, f(x) - (px + q) \,\right| \leq (x - a)^{2}$이다.
>
> **(다)** 모든 실수 $x$에 대하여 $g(x) \geq l(x)$이다.
>
> **(라)** $f(c) = g(c) \neq l(c)$
>
> $f(d)$의 값을 구하시오.

*(여기서 $p,\, q,\, a,\, c,\, d$는 수치 결정 단계에서 확정할 구체적 상수)*

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 역할 |
|:---:|---|---|
| (가) | $h(x) = \varphi(x)$ 또는 $\psi(x)$, 전역 미분가능 | 분기 구조 + 접점 전환 제약을 암묵적으로 부과 |
| (나) | 끼임 조건 $g \le h \le G$, 특정 점에서 $g(a)=G(a)$ | $f(a)$와 $f'(a)$를 고정하는 전역 부등식. 끼임의 도함수 결정력이 숨겨짐 |
| (다) | $\varphi - \psi \ge 0$ (접점 유일성) | "부등식"으로 포장된 접선 조건. 전환점이 유일하게 결정됨을 숨김 |
| (라) | $f$가 $\varphi$를 실제로 따르는 점 존재 | $f \equiv \psi$ 배제 → 전환이 반드시 발생함을 강제 |
| (발문) | 전환 후 $f = l$인 영역에서의 값 계산 | 분기 전환·극값 소거를 올바르게 파악해야 답에 도달 |
