# [STEP 1] 구조 분석

## 1-1. 조건이 만드는 방정식 체계

**(가)** 로부터: $f(x)=(x^2-d^2)(x-r)$, $d>0$, $r\neq\pm d$, 세 근 $d,-d,r$ 서로 다름.

자유 파라미터: $(d,\,r)$ — 2개.

**추가 조건 (포장으로 배치할 것)** $f(-1)=\tfrac{9}{2}$:

$$f(-1)=(1-d^2)(-1-r)=\tfrac{9}{2} \quad\cdots\;(\star)$$

→ $r$을 $d$로 표현 가능. 자유도 $2\to1$.

**(다)** $f(f(x))=0$이 $x=0$을 근으로 가짐:

$$f\!\bigl(f(0)\bigr)=0,\quad f(0)=r\,d^2$$

$r\,d^2$이 $f$의 근이므로 $r\,d^2\in\{d,\,-d,\,r\}$.

| 분기 | 조건 | $r$과 $d$의 관계 |
|---|---|---|
| **Case A** | $rd^2=d$ | $r=1/d$ |
| **Case B** | $rd^2=-d$ | $r=-1/d$ |
| **Case C** | $rd^2=r\;(r\neq0)$ | $d=1$ |

## 1-2. 분기 분석

**Case A** ($r=1/d$)를 $(\star)$에 대입:

$$(1-d^2)\!\left(-1-\frac{1}{d}\right)=\frac{9}{2}$$

$$\frac{(d^2-1)(d+1)}{d}=\frac{9}{2} \;\Longrightarrow\; 2(d-1)(d+1)^2=9d$$

$$2d^3+2d^2-11d-2=0$$

인수분해: $d=2$ 시도 → $16+8-22-2=0$ ✓

$$(d-2)(2d^2+6d+1)=0$$

$2d^2+6d+1=0$의 근 $=\dfrac{-3\pm\sqrt{7}}{2}<0$. 따라서 $d>0$인 해는 **$d=2$ 유일**.

→ $r=\tfrac{1}{2}$, 근: $2,\,-2,\,\tfrac{1}{2}$ (서로 다른 세 근 ✓)

**Case B** ($r=-1/d$)를 $(\star)$에 대입:

$$\frac{(d-1)^2(d+1)}{d}=\frac{9}{2} \;\Longrightarrow\; 2d^3-2d^2-11d+2=0$$

유리근 검사($\pm1,\pm2,\pm\frac{1}{2}$): 모두 불성립. 양의 실근 $\approx0.18,\,2.7$ — 무리수이므로 $f(0)$이 정수 불가. **기각.**

**Case C** ($d=1$): $(\star)$에서 $(1-1)(-1-r)=0\neq\frac{9}{2}$. **불가.**

#### 판정 1-2
유효 분기: **Case A, $d=2,\;r=\tfrac{1}{2}$** 단 하나. ✅ CONTINUE.

## 1-3. 자유도 계산

| 단계 | 파라미터 | 방정식 | 잔여 자유도 |
|---|---|---|---|
| (가) 후 | $d,\,r$ | 0 | 2 |
| $(\star)$ 후 | $d,\,r$ | 1 | 1 |
| (다) 후 | $d,\,r$ | 2 | **0** |

#### 판정 1-3
자유도 0, 해 유일. ✅ CONTINUE.

## 1-4. 탐색 전략

유일해 $(d,r)=(2,\frac{1}{2})$ 확정. 수치 검증 후 포장 설계로 이행.

# [STEP 2] 수치 탐색 + 포장 설계

## 2-1. 채택 후보 수치 검증

$$f(x)=(x^2-4)\!\left(x-\tfrac{1}{2}\right)=x^3-\tfrac{1}{2}x^2-4x+2$$

| 검증 항목 | 계산 | 결과 |
|---|---|---|
| 세 근 | $-2,\;\frac{1}{2},\;2$ | 서로 다른 세 실근 ✓ |
| 대칭근 | $f(2)=f(-2)=0$, $\alpha=2>0$ | (가) ✓ |
| $f(-1)$ | $(1-4)(-\frac{3}{2})=\frac{9}{2}$ | (나) ✓ |
| $f(0)$ | $(-4)(-\frac{1}{2})=2$ | |
| $f(f(0))$ | $f(2)=0$ | (다) ✓ |
| 극점 | $f'(x)=(3x-4)(x+1)$, 극대 $x=-1$, 극소 $x=\frac{4}{3}$ | |
| 극댓값 | $f(-1)=\frac{9}{2}$ | |
| 극솟값 | $f(\frac{4}{3})=-\frac{50}{27}\approx-1.85$ | |

**$f(f(x))=0$의 근 개수 (참고 검증):**

| $f(x)=c$ | $c$ 값 | $m<c<M$? | 근 개수 |
|---|---|---|---|
| $f(x)=2$ | $c=2$ | $-\frac{50}{27}<2<\frac{9}{2}$ ✓ | 3 ($x=0$ 포함) |
| $f(x)=-2$ | $c=-2$ | $-2<m$ (∵ $-2<-\frac{50}{27}$) | 1 |
| $f(x)=\frac{1}{2}$ | $c=\frac{1}{2}$ | $-\frac{50}{27}<\frac{1}{2}<\frac{9}{2}$ ✓ | 3 |

총 $3+1+3=7$개. ✓

**손계산 확인:** 입시 현장에서 다항식 인수분해, 이차부등식 수준. ✅

## 2-2. 유일성 확인

Case A만 정수해를 산출하며 $(d,r)=(2,\frac{1}{2})$가 유일. $f(0)=2$. ✅
