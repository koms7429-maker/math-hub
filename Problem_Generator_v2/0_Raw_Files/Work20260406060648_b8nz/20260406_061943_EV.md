$u=x-m$ 치환 시 $x-f(x)=m-u^3+(1+d^2)u$이므로:

| 분기 | 방정식 |
|------|--------|
| 가운데 ($=m$) | $u^3-(1+d^2)u=0$ → 근: $u=0,\;\pm\sqrt{1+d^2}$ → 항상 **3개** |
| 왼쪽 ($=m-d$) | $h_1(u)=u^3-(1+d^2)u-d=0$ |
| 오른쪽 ($=m+d$) | $h_3(u)=u^3-(1+d^2)u+d=0$ |

$h_1(-u)=-h_3(u)$ 대칭이므로 $h_1$과 $h_3$의 실근 개수 동일 $=k$.

세 분기 사이 근 중복 없음 확인: 가운데 근을 $h_1,h_3$에 대입 시 $\pm d\neq0$; $h_1$과 $h_3$의 공통근 시 $2d=0$ 모순.

∴ 총 근 개수 $= 3+2k$.

$3+2k=7$ → $k=2$ (각 측면 분기: 이중근 포함 2개)

**[전환 지점 2]** $k=2$를 만족하려면 $h_1$이 정확히 이중근을 가져야 함 → 극댓값 $=0$ 조건.

$h_1$의 임계점: $s=\sqrt{(1+d^2)/3}$

$$h_1(-s)=2s^3-d=0 \quad\Longrightarrow\quad 4s^6=d^2$$
$$4\cdot\frac{(1+d^2)^3}{27}=d^2$$
$$4(1+d^2)^3=27d^2$$

$t=d^2$로 치환:
$$4(1+t)^3=27t$$
$$4t^3+12t^2-15t+4=0$$
$$= (2t-1)(2t^2+7t-4)=0$$

$2t^2+7t-4=0$ → $t=\frac{-7\pm9}{4}$ → $t=\frac{1}{2}$ 또는 $t=-4$

$t=d^2>0$이므로 **유일한 해 $d^2=\frac{1}{2}$**.

$$\boxed{f'(\alpha)=2d^2=2\times\frac{1}{2}=1}$$

---

#### 1-4. 유일성 확인

$d^2=\frac{1}{2}$가 유일한 양수 해. $m$은 자유이나 $f'(\alpha)=2d^2=1$은 $m$ 무관. → **O (유일)**

#### 1-5. 조건 과잉/부족 체크

- **(가) 제거:** $\alpha$ 자체가 정의되지 않고, $f$의 근 구조가 미결 → 답 결정 불가 → **(가) 필수**
- **(나) 제거:** $d$가 자유이므로 $f'(\alpha)=2d^2$는 임의의 양수 → 답 미결 → **(나) 필수**
