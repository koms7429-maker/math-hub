## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: 차수 4 이하의 다항함수 p(x)와, Q(x) = (x−α)²(x−β)² (α ≠ β) 형태의 비음수 사차함수.

**사실 A가 부과하는 제약 (끼임에 의한 도함수 일치)**: 부등식 −Q(x) ≤ p(x) ≤ Q(x)가 모든 실수 x에서 성립한다. x = α, β에서 Q = 0이므로 p(α) = p(β) = 0. 또한 α, β가 Q의 이중영점이라 Q는 이 점들에서 아래로 볼록하게 0에 접촉한다. −Q와 Q 모두 이 점에서 0이므로 p는 양쪽에서 끼여 p'(α) = p'(β) = 0도 강제된다.

**사실 B가 부과하는 제약 (영점 차수의 자유도 소진)**: p(x)가 x = α에서 이중근, x = β에서 이중근을 가지므로, p(x) = (x−α)²(x−β)² · r(x). p가 사차 이하이면 r(x)는 상수 c. 따라서 p(x) = c(x−α)²(x−β)².

**숨은 사실**: **전역 부등식 |p(x)| ≤ Q(x)가 사차 다항함수 p에 대해 성립하면, p는 반드시 Q의 상수배이다.** 무한히 많은 점에서의 부등식 조건이, Q의 영점이라는 유한 개의 점에서 도함수까지 고정시키는 방식으로 자유도를 모두 소진시켜, p(x) = cQ(x) (−1 ≤ c ≤ 1)라는 1-파라미터 가족으로 축소된다.

---

### 수험생의 발견 순간
1. **처음 시도**: p(x) = ax⁴ + bx³ + cx² + dx + e로 놓고, |p(x)| ≤ Q(x)를 여러 점에 대입하여 a, b, c, d, e의 연립부등식을 세우려 한다.

2. **막히는 순간**: 부등식이 모든 x에서 성립해야 하므로 대입할 점이 무한히 많고, 5개 미지수에 대한 부등식을 체계적으로 풀 수 없다.

3. **발견**: x = α에서 Q(α) = 0이므로 −0 ≤ p(α) ≤ 0, 즉 p(α) = 0. 마찬가지로 p(β) = 0. 그런데 Q가 이 점에서 아래로 볼록하게 0에 접촉하므로, p도 이 점 근방에서 위아래로 눌린다. "어? p'(α)도 0이어야 하네?" — 끼임 정리를 도함수에 적용하면 p'(α) = p'(β) = 0. 이중근이 두 곳이면 사차식의 자유도 5개 중 4개가 소진되어, p(x) = c(x−α)²(x−β)²로 결정된다.

4. **숨은 사실과의 대응**: STEP 2-2의 "영점 중복도 총합 ≥ 차수이면 스칼라 배수로 결정"이 정확히 이 순간에 실현된다. 학생은 "부등식을 풀었을 뿐인데, 다항함수의 형태가 하나로 고정되었다"는 사실에 놀라게 된다.

# STEP 1. 구조 분석

## 1-1. 조건이 만드는 방정식 체계

**문자**: f, g는 사차 이하 다항함수. a는 양의 정수(내가 결정).

| 조건 | 수학적 내용 |
|---|---|
| (가) | $f(x)+g(x) = x^4 - 2ax^3 + a^2x^2$ |
| (나) | $f(x)g(x)\ge 0\;\;\forall x\in\mathbb{R}$ |
| (다) | $f(p) = q$ (구체값 미정) |

**(가)+(나) 연립 도출:**

$g = Q - f$를 (나)에 대입:

$$f(x)\bigl(Q(x)-f(x)\bigr)\ge 0\quad\forall x$$

**부호 분석:** $Q(x)=x^2(x-a)^2\ge 0$이므로,

- $f(x)>0 \Rightarrow Q(x)-f(x)\ge 0 \Rightarrow f(x)\le Q(x)$
- $f(x)<0$인 점 $x_0$ 존재 가정 → $Q(x_0)-f(x_0)>0$ → $f(x_0)(Q(x_0)-f(x_0))<0$, 모순.

∴ $0\le f(x)\le Q(x)$ (전역 끼임)

**영점 분석:** $Q(0)=Q(a)=0$이므로 $f(0)=f(a)=0$.

$f\ge 0$이고 $f(0)=0$ → $x=0$은 극솟값점 → $f'(0)=0$ (이중근).
마찬가지로 $f'(a)=0$ (이중근).

$f(x)=x^2\cdot h(x)$ ($h$는 이차 이하)로 쓰면 $h(a)=0$, $h'(a)=0$ → $h(x)=c(x-a)^2$.

$$\boxed{f(x)=c\,x^2(x-a)^2,\quad 0\le c\le 1}$$

**(다):** $f(p)=c\,p^2(p-a)^2=q$ → $c=\dfrac{q}{p^2(p-a)^2}$ (유일 결정, $p\ne 0,\,a$일 때)

## 1-2. 분기 분석

개수 조건 없음. 끼임 논증은 등식/부등식 혼합이지만 결과적으로 $f=cQ$라는 1-파라미터 가족으로 수렴.
