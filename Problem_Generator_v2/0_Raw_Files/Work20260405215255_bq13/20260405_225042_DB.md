## [STEP 1] 구조 분석

### 1-1. 조건이 만드는 방정식 체계

**기본 설정:** $f(x) = x^3 - 3px + q$, $p > 0$

$$f'(x) = 3(x^2 - p),\quad x = \pm\sqrt{p}\text{에서 극값}$$

$$f(-\sqrt{p}) = 2p\sqrt{p} + q \;\;(\text{극대}),\quad f(\sqrt{p}) = -2p\sqrt{p} + q \;\;(\text{극소})$$

**$g(x) = |f(x)|\cdot f'(x)$의 미분불가능점 분석:**

- $|f(x)|$가 미분불가능한 점: $f$의 **단근**(부호 변환점)
- 단근 $x_0$에서 $f'(x_0)\neq 0$이면: $g'(x_0^-) = -(f'(x_0))^2 \neq (f'(x_0))^2 = g'(x_0^+)$ → **미분불가능**
- 이중근 $x_0$에서: $f(x) \sim c(x-x_0)^2$이므로 $|f(x)|$가 매끄럽고, $f'(x_0)=0$이므로 $g \sim C(x-x_0)^3$ → **미분가능**

따라서 **$g$의 미분불가능점 수 = $f$의 단근 수**.

### 1-2. 분기 분석

**조건 (가): $g$의 미분불가능점이 1개 $\Leftrightarrow$ $f$의 단근이 1개**

| 분류 | $q$의 범위 | $f$의 근 구조 | 단근 수 | (가) 충족 |
|---|---|---|---|---|
| Case A | $q = 2p\sqrt{p}$ | 이중근($\sqrt{p}$) + 단근($-2\sqrt{p}$) | 1 | ✓ |
| Case A′ | $q = -2p\sqrt{p}$ | 이중근($-\sqrt{p}$) + 단근($2\sqrt{p}$) | 1 | ✓ |
| Case B | $\lvert q\rvert > 2p\sqrt{p}$ | 단근 1개 + 복소근 2개 | 1 | ✓ |
| Case C | $\lvert q\rvert < 2p\sqrt{p}$ | 단근 3개 | 3 | ✗ |

**조건 (나): $y=|f(x)|$와 $y=|f'(x)|$의 교점이 5개**

교점 방정식: $(f(x))^2 = (f'(x))^2$, 즉

$$h_1(x)\cdot h_2(x) = 0,\quad h_1 = f-f',\;\; h_2 = f+f'$$

두 삼차식 $h_1, h_2$의 **공통근** 조건: $h_1(x)=h_2(x)=0 \Rightarrow f(x)=0 \wedge f'(x)=0$ $\Rightarrow$ **$x$가 $f$의 중근**

| 분류 | 공통근 | 총 교점 수 가능값 | (나)"5개" 충족 가능? |
|---|---|---|---|
| Case A/A′ | 이중근 1개 공유 | $3+3-1=5$ (대수적으로 확인: 나머지 이차식들의 판별식 항상 양) | **항상 ✓** |
| Case B | 없음 | $\{1,3\}+\{1,3\} \in \{2,4,6\}$ (짝수) | **절대 ✗** |
| Case C | 없음 | 짝수만 가능 | **절대 ✗** |

> **검증 (Case A, $a=\sqrt{p}$):**
> $h_1 = (x-a)[x^2+(a-3)x-(2a^2+3a)]$, $h_2 = (x-a)[x^2+(a+3)x+(3a-2a^2)]$
> 각 이차식의 판별식: $D_1 = 9(a^2+\frac{2}{3}a+1) > 0$, $D_2 = 9(a^2-\frac{2}{3}a+1) > 0$ (항상 양).
> 두 이차식의 공통근 존재 조건: $x=-a$ 대입 시 $Q_1(-a)=-2a^2\neq 0$. 없음.
> ∴ 모든 $a>0$에 대해 정확히 5개 교점. ✓

### 판정 1-2

## [STEP 2] 수치 탐색 + 포장 설계

### 후보별 판정

**후보 ① ($p=1$):** $f(0)=\pm 2$, $(f(0))^2=4$. 답이 너무 작아 수능 주관식(자연수 답)으로는 너무 쉬운 인상. 기각.

**후보 ② ($p=4$):** $f(0)=\pm 16$, $(f(0))^2=256$. 
- Case A: $f(x)=(x-2)^2(x+4)=x^3-12x+16$
- Case A′: $f(x)=(x+2)^2(x-4)=x^3-12x-16$
- 교점 검증 ($a=2$):
  - $h_1=(x-2)(x^2-x-14)$: 근 $2,\;\frac{1\pm\sqrt{57}}{2}$ → 3개 ✓
  - $h_2=(x-2)(x^2+5x-2)$: 근 $2,\;\frac{-5\pm\sqrt{33}}{2}$ → 3개 ✓
  - 공통근 $x=2$. 총 5개 ✓
- 답 $256$: 수능 주관식 범위(0~999) 내. **채택.**

**후보 ③:** $q=\pm 6\sqrt{3}$ 비정수 → $f(0)$ 비정수이나 $(f(0))^2=108$은 정수. 가능하나 풀이 과정에서 $\sqrt{3}$ 다루기 번거로움. 후보 ②보다 열위.

**후보 ④:** 답 $2916>999$. 수능 주관식 범위 초과. 기각.
