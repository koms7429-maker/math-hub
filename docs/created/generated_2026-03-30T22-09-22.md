### 완성된 문제

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 서로 다른 세 실근 $\alpha,\;\beta,\;\gamma$ $\;(\alpha<\beta<\gamma)$를 가진다. 다음 조건을 만족시킨다.
>
> **(가)** 곡선 $y=f(x)$ 위의 점 $(\alpha,\,0)$에서의 접선과 점 $(\gamma,\,0)$에서의 접선이 서로 평행하다.
>
> **(나)** $f'(\alpha)+f'(\beta)+f'(\gamma)=9$
>
> $f'(\alpha)\times f'(\gamma)$의 값을 구하시오. $\quad\boldsymbol{[4\text{점}]}$

> **정답: $36$**

## 출처 아이디어
**수학적 객체:** 세 서로 다른 실근 $r_1 < r_2 < r_3$을 가진 모닉 삼차함수 $f(x) = (x-r_1)(x-r_2)(x-r_3)$

**사실 A(P018)가 부과하는 제약:**
삼차곡선 위의 두 점 $(r_1, 0)$과 $(r_3, 0)$에서의 접선이 평행하다. 즉 $f'(r_1) = f'(r_3)$이다. 이 조건은 도함수 $f'(x)$(이차식)의 대칭축에 대해 $r_1$과 $r_3$이 대칭임을 의미하고, 삼차함수의 변곡점이 $r_2$와 일치하며 $r_2 = (r_1 + r_3)/2$가 강제된다. 결과적으로 세 근이 **등간격**으로 배치된다.

**사실 B(P011)가 부과하는 제약:**
모닉 삼차의 항등식 $f'(r_1) + f'(r_2) + f'(r_3) = (r_1+r_2+r_3)^2 - 3(r_1r_2 + r_2r_3 + r_3r_1)$이 성립한다. 이 우변은 근과 계수의 관계에서 직접 계산 가능한 양이다.

**숨은 사실 (A+B 동시 성립 시 강제되는 것):**
평행접선 조건(A)에 의해 세 근이 등간격($d = r_3 - r_1$)이 되면, 각 근에서의 도함수값이

$$f'(r_1) = f'(r_3) = \frac{d^2}{2}, \qquad f'(r_2) = -\frac{d^2}{4}$$

로 **단일 매개변수 $d^2$의 고정 유리배수**가 된다. 이때 항등식(B)은

$$\frac{3}{4}d^2 = e_1^2 - 3e_2$$

로 붕괴하여, 계수 조합 $e_1^2 - 3e_2$의 값이 곧 근의 폭 $d$를 유일하게 결정하고, 나아가 **세 근 각각에서의 접선 기울기를 부호까지 모두 확정**한다. 특히 중간근에서의 접선 기울기의 절댓값은 양 끝근 접선 기울기의 **정확히 절반**: $|f'(r_2)| = \frac{1}{2}f'(r_1)$이다.

## 수험생의 발견 순간
**1. 수험생이 처음 시도하는 것:**
"두 근에서의 접선이 평행"이라는 조건을 $f'(r_1) = f'(r_3)$으로 번역하고, $f'(x) = 3x^2 + \cdots$라는 이차식에서 대칭축 조건을 이용하여 $r_1 + r_3 = 2r_2$, 즉 세 근이 등간격임을 밝힌다. 여기까지는 자연스럽다.

**2. 막히는 순간:**
세 근이 등간격이라는 사실만으로는 $f'(r_2)$의 구체적 값이나 세 접선 기울기 사이의 관계를 알 수 없다. 문제가 요구하는 양(예: 접선들이 만드는 도형의 넓이, 접선 기울기의 합 등)을 계산하려면 개별 도함수값이 필요한데, 등간격 조건만으로는 $d$가 미정이어서 진행이 막힌다.

**3. 발견:**
$f(x) = (x-r_1)(x-r_2)(x-r_3)$에서 각 근의 도함수를 직접 전개하면:
$$f'(r_i) = \prod_{j \neq i}(r_i - r_j)$$
등간격 조건 $r_2 - r_1 = r_3 - r_2 = d/2$를 대입하는 순간, 세 값이 모두 $d^2$의 **상수배**로 정리된다:

> $f'(r_1) = f'(r_3) = d^2/2$, $\quad f'(r_2) = -d^2/4$

그리고 항등식 $\Sigma f'(r_i) = e_1^2 - 3e_2$에 이 값들을 넣으면 좌변이 $3d^2/4$로 깔끔하게 정리되어, **계수로부터 $d^2$이 즉시 결정**된다. $d^2$이 결정되면 세 도함수값이 전부 확정되어 요구되는 양이 한꺼번에 계산된다.

**4. 이 발견이 STEP 2-2의 숨은 사실과 어떻게 대응하는가:**
STEP 2-2의 숨은 사실은 "평행접선 조건 하에서 세 도함수값이 $d^2$의 고정 유리배수가 되고, 대칭함수 항등식이 $\frac{3}{4}d^2$으로 붕괴한다"는 것이었다. 수험생은 등간격 조건을 도함수 곱 표현에 대입하는 과정에서 이 붕괴를 직접 체험하며, "계수 하나($e_1^2 - 3e_2$)로 접선 기울기 세 개가 전부 결정된다"는 예상 밖의 결론에 도달한다. 두 조건(기하적 평행 + 대수적 항등식)을 각각 따로 보면 절대 예측할 수 없는, 결합에서만 드러나는 완전결정성이 핵심적 놀라움이다.

## 문제 구조
### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 서로 다른 세 실근 $\alpha,\;\beta,\;\gamma$ $(\alpha<\beta<\gamma)$를 가진다.
>
> **(가)** 곡선 $y=f(x)$ 위의 점 $(\alpha,\,0)$에서의 접선과 점 $(\gamma,\,0)$에서의 접선이 서로 평행하다.
>
> **(나)** $f'(\alpha)+f'(\beta)+f'(\gamma)=k$
>
> $f'(\alpha)\times f'(\gamma)$의 값을 구하시오.
>
> *(수치 조정 예시: $k=9$이면 답 $36$)*

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 역할 |
|---|---|---|
| **(가)** 두 접선 평행 | 사실 A: $f'(\alpha)=f'(\gamma)$ → 세 근 등간격 강제 | 구조적 제약 (등간격을 강제하되, $d$를 확정하지 않음) |
| **(나)** 도함수값의 합 $=k$ | 사실 B: 항등식 $\sum f'(r_i)=e_1^2-3e_2$의 수치 실현 | 수치적 앵커 (등간격과 결합 시 $d^2=4k/3$ 확정) |
| **발문** $f'(\alpha)\times f'(\gamma)$ | 숨은 사실의 활용을 강제 | 곱을 구하려면 $f'(\alpha)$의 개별 값이 필요 → 숨은 비율 $f'(\beta)=-\frac{1}{2}f'(\alpha)$ 발견 필수 |
