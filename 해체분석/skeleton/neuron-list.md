| 연산29 | 다항식을 $a(x-p)^n+q$ 형태로 변환한다 (완전제곱식, 꼭짓점 형태) |
| 연산30 | 두 점이 주어지면 지나는 직선의 방정식을 구할 수 있다 |
| 연산31 | $n\log_a b = \log_a b^n$ (로그 거듭제곱 법칙) |

## 전략 (strategy)

| ID | Statement |
|---|---|
| 전략1 | $a^{f(x)}$를 하나의 변수로 치환하면 지수방정식을 다항방정식으로 변환할 수 있다 |
| 전략2 | 두 조건식을 연립하면 미지수를 결정할 수 있다 |
| 전략3 | 구간별 정의 함수에서 $f(x_1)=f(x_2)$인 서로 다른 쌍이 없으려면, 각 구간에서 단사이고 구간별 치역이 겹치지 않아야 한다 |
| 전략4 | 연속함수의 치역 $[a,b]$에 정수가 포함되지 않으려면 구간 전체가 두 연속 정수 사이에 놓여야 한다 |
| 전략5 | 방정식 $f(x)=c$의 서로 다른 실근의 개수는 $y=f(x)$의 그래프와 직선 $y=c$의 교점 수와 같다 |
| 전략6 | $|a| = a$ ($a \geq 0$), $-a$ ($a < 0$). 절댓값 내부의 부호에 따라 경우를 분리한다 |
| 전략7 | $f(x_1)=g(x_2)$인 실수쌍 $x_1, x_2$가 존재 $\iff \exists c$: $f(x_1)=c$이고 $g(x_2)=c$ (매개변수 분리) |
| 전략8 | 명제 $P \to Q$가 성립하면 대우 $\lnot Q \to \lnot P$도 성립한다. '존재한다$\to$A'의 대우는 '$\lnot$A$\to$존재하지 않는다' |
| 전략9 | 조건을 만족시키는 자명한 해(특수한 경우)를 먼저 확인한다 |
| 전략10 | 한 변수를 고정하고 나머지 변수의 가능한 값을 세어 합산한다 (변수 고정 카운팅) |
| 전략11 | 두 등식의 양변끼리 더하거나, 빼거나, 곱하거나, 나누어 새로운 등식을 만든다 |
| 전략12 | 거듭제곱근 문제에서 $n$의 홀짝에 따라 실수 근의 존재 조건이 달라지므로 경우를 분리한다 |
| 전략13 | 함수의 그래프를 $x$축(또는 $y$축)에 대해 대칭시켜 새로운 관계를 얻는다 |
| 전략14 | 두 함수(또는 도형) 사이에 대칭 구조가 있는지 추측한다 |
| 전략15 | 두 함수(또는 도형) 사이에 평행이동 구조가 있는지 추측한다 |
| 전략16 | 두 함수(또는 도형) 사이에 회전 구조가 있는지 추측한다 |
| 전략17 | 미지수의 가능한 값이 유한개이면 모두 대입하여 확인한다 (전수조사) |
| 전략18 | 사각형의 넓이를 구하기 위해 대각선으로 두 삼각형으로 분할한다 |
| 전략19 | 합성함수 $g \circ f$의 최댓값(또는 최솟값)을 구하려면, $f$의 치역을 먼저 구한 뒤 그 치역에서 $g$의 최댓값(또는 최솟값)을 구한다 |
| 전략20 | 가능한 경우를 나열하고, 모순이 되는 경우를 제거한다 (소거법) |
| 전략21 | 조건을 가정하고 결과를 확인한다 |
| 전략22 | 기준이 되는 경우를 발견하면, 그 경우를 중심으로 케이스를 분류한다 |
| 전략23 | 모순이 발생하면 해당 가능성을 배제한다 |
| 전략24 | 점근선이 있을 때는 점근선 주변부터 관찰한다 |
| 전략25 | 후보가 있으면 대입하여 조건을 만족하는지 확인한다 |
| 전략26 | 함수가 단조인 구간에서 부등식의 해의 범위를 알려면, 경계만 조사하면 된다 |
| 전략27 | 유한 집합의 원소 개수 조건이 주어지면, 무한 집합을 만드는 시나리오를 먼저 배제한다 |
| 전략28 | 부등식 $f(x) > g(x)$가 주어지면, $h(x) = f(x) - g(x)$로 이항하여 $h(x) > 0$의 문제로 변환한다 |
| 전략29 | 이차식 또는 삼차식이 나오면 인수분해를 시도한다 |
| 전략30 | 로그 부등식 $\log u \times \log v \ge 0$에서, 정의 조건 ($u>0, v>0, u\neq 1, v\neq 1$)과 부호 조건 ($(u-1)(v-1) \ge 0$)을 분리하여 얻는다 |
| 전략31 | 조건 $|g(x)| \ge c$의 해 집합이 주어지면, $g(x) \ge c$ 또는 $g(x) \le -c$로 분해하여 해석한다 |
| 전략32 | 유한 구간의 합이 목표 구간을 덮는지 확인하려면, 부분합(누적합)을 계산하여 목표값 도달 여부를 탐색한다 |
| 전략33 | 약수와 배수가 나오면 소인수 분해를 시도한다 |
| 전략34 | $A$, $B$의 공약수는 $\gcd(A, B)$를 나눈다 |
| 전략35 | 유한 집합에서 합이 목표값인 부분집합을 탐색한다 |
| 전략36 | 공배수 조건이 주어지면 lcm을 계산하고, lcm의 배수를 탐색한다 |
| 전략37 | 거듭제곱·거듭제곱근에서 정수 조건이 주어지면, 이를 지수의 약수·배수 조건으로 번역한다 |
