→ 절댓값에 의한 접힘(cusp)이 교차점의 성격(관통 vs 접촉)을 결정하며, 교점 총수가 f의 근 구조와 극점 배치에 민감하게 의존한다.

**공유 파라미터**: p (극점 간격 및 극값 크기), q (수직 위치)
- A는 q를 p의 함수로 고정하지만 p는 미결
- B는 p와 q 모두에 의존하지만 단독으로 결정 불가
- A+B 동시: q가 p로 치환된 후 B가 p에 대한 단일 조건을 부여하여 완전 결정

**강제 결과**: f는 이중근을 극소점에서 가지며, 이 이중근에서 |f|와 |f′|가 동시에 0이 되는 "이중 소멸"이 발생. 이것이 교점 하나를 흡수하여, 나머지 영역에서 정확히 5교점이 되려면 p가 유일한 값으로 결정된다. 결과적으로 이중근 위치, 단근 위치, 극값이 모두 하나의 값으로 확정된다.

[STEP 4] 수험생의 발견 순간

1. **처음 시도**: 제약 A를 분석. F(x) = |f(x)|·f′(x)의 영점은 f = 0인 점과 f′ = 0인 점의 합집합. |f|가 항상 ≥ 0이므로 f의 영점에서 F는 부호가 바뀌지 않음(팬텀근)을 인식. 영점 개수를 세면: f의 실근 개수 + f′의 실근 2개. "4개"가 되려면 하나가 겹쳐야 함을 발견 → q = 2p√p 또는 q = −2p√p.

2. **막히는 순간**: q를 p로 표현했지만 p가 자유. "p가 어떤 양수든 A를 만족하는데, 답을 구할 수가 없다."

3. **연립 과정**: 제약 B를 가져옴. q = 2p√p를 대입하면 f(x) = (x − √p)²(x + 2√p). |f(x)| = |f′(x)|를 풀면 f = ±f′ 각각의 삼차방정식이 되지만, **이중근 √p에서 |f|가 0에 접촉하고 동시에 |f′(√p)| = |3(p − p)| = 0**이므로 이 점이 양쪽 곡선의 공통 영점이 됨. 이 접촉이 교점 하나를 "소비"하는지 "생성"하는지가 p에 의존함을 발견.

4. **발견**: "이중근 점에서의 이중 소멸 때문에 |f|와 |f′|가 접촉만 하고 관통하지 않아 교점이 1개 줄어든다. 나머지 영역에서 교점 5개를 맞추려면 f = f′와 f = −f′ 각각의 근 배치가 특정 조건을 만족해야 하고, 이것이 p의 값을 유일하게 강제한다!" — 팬텀근이 교차-접촉 분류를 변형시키는 것이 핵심 메커니즘임을 깨닫는 순간.
