# S1-NEW. (Prob_New) — 정밀 로직 스켈레톤

## 문제

$-2 \leq a \leq 4$인 정수 $a$와 $2 \leq n \leq 5$인 자연수 $n$에 대하여, 방정식 $(x^2-a)(x^n-a^2)=0$의 서로 다른 실근의 개수가 2가 되도록 하는 모든 순서쌍 $(a,n)$의 개수는?

**답**: 11

---

## 단서

| ID | 내용 |
|---|---|
| 단서A | 방정식 $(x^2-a)(x^n-a^2)=0$ |
| 단서B | 서로 다른 실근의 개수가 2이다 |
| 단서C | $-2 \leq a \leq 4$인 정수, $2 \leq n \leq 5$인 자연수 |

---

## 1단계: 본해설 완전 전개

1. $(x^2-a)(x^n-a^2)=0$이므로 영인자법칙에 의해 $x^2=a$ 또는 $x^n=a^2$이다.
2. 두 방정식의 실근 집합을 각각 $S_1$, $S_2$라 하면, 서로 다른 실근의 개수는 합집합 $|S_1 \cup S_2|$이다. 이것이 2가 되어야 한다.
3. $a$의 부호에 따라 $S_1$이 달라진다.
   - **$a < 0$일 때**: $x^2 = a$의 실근이 없다. $S_1 = \emptyset$이다.
   - **$a = 0$일 때**: $x^2 = 0$에서 $x=0$, $x^n = 0$에서 $x=0$이다. $S_1 \cup S_2 = \{0\}$이다. 실근 1개로 조건 불만족이다.
   - **$a > 0$일 때**: $x^2 = a$의 실근은 $\pm\sqrt{a}$로 2개이다. $S_1 = \{-\sqrt{a},\, \sqrt{a}\}$이다.
4. **$a < 0$인 경우**: $S_1 = \emptyset$이므로 $|S_1 \cup S_2| = |S_2|$이다. $|S_2| = 2$이어야 한다.
   - $a^2 > 0$이므로 $x^n = a^2$을 분석한다.
   - $n$이 홀수이면: 실근은 $(a^2)^{1/n}$ 하나이다. $|S_2|=1$로 불만족이다.
   - $n$이 짝수이면: 실근은 $\pm|a|^{2/n}$으로 2개이다. $|S_2|=2$로 만족한다.
   - $a \in \{-2, -1\}$이고 $n \in \{2, 4\}$이므로 $2 \times 2 = 4$가지이다.
5. **$a > 0$인 경우**: $|S_1| = 2$이다. $n$의 홀짝에 따라 $S_2$를 구한다.
   - **$n$ 홀수일 때**: $x^n = a^2$의 실근은 $a^{2/n} > 0$ 하나이다. $S_2 = \{a^{2/n}\}$이다. $a^{2/n} > 0$이고 $-\sqrt{a} < 0$이므로 가능한 중복은 $a^{2/n} = \sqrt{a}$뿐이다. $|S_1 \cup S_2| = 2$이려면 $a^{2/n} = a^{1/2}$이어야 한다.
     - $a \geq 2$이면: $a \neq 1$이므로 $a^{2/n} = a^{1/2}$에서 $2/n = 1/2$, 즉 $n = 4$이어야 한다. $n$이 홀수이므로 불가하다.
     - $a = 1$이면: $1^{2/n} = 1 = \sqrt{1}$이므로 항상 성립한다. $|S_1 \cup S_2| = |\{-1, 1\}| = 2$이다. 만족한다.
     - $a = 1$이고 $n \in \{3, 5\}$이므로 $1 \times 2 = 2$가지이다.
   - **$n$ 짝수일 때**: $x^n = a^2$의 실근은 $\pm a^{2/n}$으로 2개이다. $S_2 = \{-a^{2/n},\, a^{2/n}\}$이다. $|S_1| = |S_2| = 2$이고 $|S_1 \cup S_2| = 2$이려면 $S_1 = S_2$이어야 한다. 즉 $a^{1/2} = a^{2/n}$이다.
     - $a \neq 1$이면 $1/2 = 2/n$, 즉 $n = 4$이어야 한다.
     - $a = 1$이면 항상 성립한다.
     - $n = 4$이면 $a \in \{1, 2, 3, 4\}$이고, $n = 2$이면 $a = 1$뿐이다. $4 + 1 = 5$가지이다.
6. 총: $4 + 2 + 5 = 11$.
