## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: 최고차항의 계수가 1이고 이차항이 없는 삼차함수 f(x) = x³ + px + q (변곡점이 y축 위에 위치).

**제약 A** (구간 최댓값 대칭):
f(x)가 구간 [-1, 0] 위에서 갖는 최댓값과 구간 [0, 1] 위에서 갖는 최댓값이 서로 같다.
→ 이 조건은 극점의 위치 s = √(−p/3)에 대한 방정식 (s+1)²(2s−1) = 0을 만들어 **p = −3/4**를 결정하지만, 상수항 q는 양변에서 상쇄되어 미결로 남는다.

**제약 B** (곱미분 적분 소멸):
∫₋₁¹ f(x)·f′(x) dx = 0.
→ 곱의 미분법 역이용으로 [f(x)²/2]₋₁¹ = 0, 즉 f(1)² = f(−1)²로 환원된다. 이것은 **"q = 0" 또는 "p = −1"**이라는 논리적 분기를 생성한다. 각 분기 내에서 나머지 파라미터 하나가 자유.

**공유 파라미터**: p와 q 모두가 두 제약에 의해 공유된다. A는 p를 고정하고 q에 무관하며, B는 p 또는 q 중 하나를 고정하는 분기를 만든다.

**강제 결과**: A가 강제한 p = −3/4는 B의 분기 "p = −1"과 양립 불가능하므로, 이 분기가 소멸한다. 유일하게 생존하는 분기 "q = 0"에 의해 f(x) = x³ − (3/4)x가 완전히 결정된다. 이는 기함수이며, 끝점값과 극값이 절대값에서 일치하는 비자명한 성질을 갖는다.

**영감 source_pid**: **P020** (구간 평행이동 극값집합 불변성), **P030** (곱미분 역조립에 의한 함수계 축약)

---

### 수험생의 발견 순간
1. **처음 시도**: 제약 A를 먼저 전개한다. f′(x) = 3x² + p = 0의 근 x = ±√(−p/3)를 구하고, 구간 [-1, 0]과 [0, 1] 각각에서 극점·끝점의 함수값을 나열하여 최댓값을 비교한다. 경우 분류 후 등치 방정식 2s³ + 3s² − 1 = 0을 세우고, 인수분해 (s+1)²(2s−1) = 0으로 **s = 1/2, p = −3/4**를 얻는다.

2. **막히는 순간**: p는 결정됐지만 q가 어디에도 등장하지 않아 f가 유일하게 결정되지 않는다. "q는 아무 값이나 가능한 것인가?"라는 의문이 생긴다. 제약 A만으로는 더 이상 진행이 불가능하다.

3. **제약 B를 끌어옴**: ∫₋₁¹ f·f′ dx = 0을 전개한다. 처음에는 x³·(3x²+p) + (px+q)·(3x²+p) 등을 직접 적분하려 하여 복잡해진다. 그러다 **f·f′ = (f²)′/2**임을 인식하면 적분이 끝점 평가 f(1)² = f(−1)²로 극적으로 축약된다. 이를 전개하면 (1+p+q)² = (1+p−q)²이므로 **q = 0 또는 p = −1**이라는 분기가 나타난다.

4. **발견**: "p = −1은 제약 A에서 p = −3/4로 이미 확정된 것과 모순이니 **불가능**하다! 그러면 남은 건 q = 0뿐이다." 여기서 학생은 깨닫는다 — **두 조건 어디에도 '기함수'라는 요구가 없었는데, 구간 최댓값의 좌우 대칭과 적분 소멸이 합쳐지니 f(x) = x³ − (3/4)x라는 기함수가 강제된다.** 더 나아가, 이 함수에서 f(1) = 1/4 = f(−1/2)(극댓값)이고 f(−1) = −1/4 = f(1/2)(극솟값)이므로, 끝점값과 극값이 절대값에서 정확히 일치한다는 숨은 기하학적 사실을 발견한다.
