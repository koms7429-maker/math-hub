## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체:** 극대와 극소를 모두 갖는 삼차함수 f(x), 닫힌 구간 [a, b] 위에서 고려.

**사실 A가 부과하는 제약 (P014 — 치역 보존 정의역 확장):**
삼차함수의 치역이 구간 [a, b]에서 [m, M]이라 하자. 이 치역 [m, M]을 그대로 유지하면서 구간의 한쪽 끝점(예: a)을 더 왼쪽으로 이동시킬 수 있는 최대 한계를 구해야 한다. 치역이 바뀌지 않으려면, 이동된 끝점에서의 함숫값이 [m, M]을 벗어나면 안 된다. 한계는 f(a*) = m 또는 f(a*) = M이 되는 a*이다.

**사실 B가 부과하는 제약 (P019 — 접선 차감 이중근 승격):**
극소점 α에서 f(x) − f(α) = 0은 x = α를 이중근으로 가지므로 (x − α)²·(x − β) = 0으로 인수분해된다. 최고차 계수 비교와 Vieta 공식에 의해 β는 α와 변곡점의 위치만으로 유일하게 결정된다: **β는 변곡점을 기준으로 α의 정확히 2배 거리 반대편**에 있다. (극대점에 대해서도 동일한 구조 성립.)

**숨은 사실 (A+B 동시 성립 시 강제되는 것):**
치역 보존의 한계 방정식 f(a*) = f(극소) (또는 f(b*) = f(극대))를 풀면, 이것은 바로 사실 B의 이중근 인수분해가 만드는 단근 β이다. 따라서:

> **정의역 확장의 최대 한계점 = 접선 차감의 단근 = 변곡점에서 극점까지 거리의 2배 반대편**

이 세 가지가 동일한 점임이 대수적으로 강제된다. 결과적으로, "얼마나 더 확장할 수 있는가?"의 답은 삼차함수의 계수에 무관한 **보편적 기하 비율**(변곡점-극점 거리의 2배)로 귀결된다.

---

### 수험생의 발견 순간
1. **수험생이 처음 시도하는 것:**
   치역 [m, M]을 보존하면서 구간을 확장할 수 있는 한계를 구하기 위해, f(a*) = f(극소) 방정식을 세운다. 삼차방정식을 직접 풀려 하거나, 근의 공식을 떠올린다.

2. **막히는 순간:**
   삼차방정식 f(x) = f(극소)는 매개변수가 포함되어 있어 일반적인 풀이가 복잡하다. "극소점이 이미 이 방정식의 근이라는 건 알겠는데, 나머지 근은 어떻게 구하지?"에서 멈춘다.

3. **발견:**
   극소점 α에서 f'(α) = 0이므로, f(x) − f(α)는 x = α에서 **단순근이 아닌 이중근**을 가진다! (x − α)²으로 나누면 일차식이 남아, 나머지 근 β가 **나눗셈 한 번**으로 결정된다. 계산하면 β = (변곡점) × 3 − 2α, 즉 변곡점에서 극점까지의 거리의 정확히 2배 반대편이다. "아, 이게 바로 확장 한계점이었구나!"

4. **숨은 사실과의 대응:**
   수험생이 발견한 것은 STEP 2-2의 숨은 사실 그 자체다: 치역 보존이라는 **해석적 조건**의 답이, 접선 차감이라는 **대수적 기법**의 부산물(이중근 인수분해의 단근)로 즉시 나온다. 정의역 확장 한계가 변곡점-극점 거리의 2배라는 보편 비율로 귀결된다는 사실은, 두 조건을 따로 봤을 때는 전혀 예상할 수 없었던 연결이다.

## 2. 프라블럼 메이커 (Problem Maker)

### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$와 상수 $c$에 대하여
> $$h(x) = |f(x) - c|$$
> 로 정의한다. 다음 조건을 만족시킨다.
>
> **(가)** $f(x)$는 $x = 0$에서 극대이고, $x = a\;(a > 0)$에서 극소이다.
>
> **(나)** 방정식 $f(x) = c$의 서로 다른 실근의 개수가 $2$이다.
>
> **(다)** $f(p) = q$
>
> $h(x)$가 미분가능하지 않은 $x$의 값을 $\alpha$라 할 때, $f'(\alpha)$의 값을 구하시오.

*(주: $c, p, q$는 구체적 정수. detail\_builder가 $(a, d, c, p, q)$를 결정하되 자유도는 실질 2 $(a, d)$이며 $c=d$ 또는 $c=d-a^3/2$ 중 하나가 강제됨. 발문 $f'(\alpha)$는 이중근 인수분해의 단근에서의 도함수 값으로, 정수 답이 되도록 설계.)*

---

##

### 조건-상황 대응표
| 조건 | 담당하는 수학적 상황 요소 |
|---|---|
| 전제 (최고차항 계수 $1$, $h(x)=\|f(x)-c\|$) | 극대·극소를 갖는 삼차함수 존재; 절댓값 포장으로 이중근 구조를 미분가능성 문제로 변환 |
| **(가)** $x=0$ 극대, $x=a$ 극소 | 극대점 고정으로 자유도 축소; $f'(x)=3x(x-a)$, 변곡점 $a/2$ 확정 |
| **(나)** $f(x)=c$의 서로 다른 실근이 $2$개 | **사실 B 유도** — 실근이 정확히 2개이려면 이중근이 필수 → $c$가 극값이어야 함 → $(x-\text{극점})^2(x-\beta)$ 인수분해로 단근 $\beta$ 발견 |
| **(다)** $f(p) = q$ | 파라미터 $(a, d)$를 유일하게 확정; 사실 A의 한계점 좌표가 수치로 결정됨 |
| **숨은 사실** (명시 없음) | $f(x)-c = (x-\text{극점})^2(x-\beta)$에서 단근 $\beta$가 $h(x)$의 유일한 비미분가능점이며, 이는 변곡점 기준 극점의 2배 거리 반대편 = 치역 보존 한계점 |
