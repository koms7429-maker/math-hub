# STEP 1. 구조 분석

## 1-1. 조건이 만드는 방정식 체계

$f(x)=x^3+ax^2+bx$ (전제문 + (가))

**(나)의 번역:** $f(x)=-x$, 즉

$$x^3+ax^2+(b+1)x=0 \;\;\Longrightarrow\;\; x\bigl(x^2+ax+(b+1)\bigr)=0$$

$x=0$은 항상 해이므로, "서로 다른 두 점"이 되려면 이차식 $h(x)=x^2+ax+(b+1)$이 **정확히 하나의 추가 교점**을 줘야 합니다.

**(다)의 번역:** $f(x)=x(x^2+ax+b)=0$의 영이 아닌 두 근 $\alpha,\beta$에 대해

$$f'(\alpha)\cdot f'(\beta)=-N$$

$\alpha+\beta=-a,\;\alpha\beta=b$이고, $\alpha^2=-a\alpha-b$를 이용하면

$$f'(\alpha)=3\alpha^2+2a\alpha+b = -a\alpha-2b$$

마찬가지로 $f'(\beta)=-a\beta-2b$. 따라서

$$f'(\alpha)\cdot f'(\beta)=(a\alpha+2b)(a\beta+2b)=a^2 b-2a^2 b+4b^2 = -b(a^2-4b) \quad\cdots\text{★}$$

## 1-2. 분기 분석

"서로 다른 두 점에서 만남"의 분기:

| 분기 | 조건 | 교점 개수 | 판정 |
|---|---|---|---|
| ① $h$의 판별식 $=0$, 중근 $\neq 0$ | $a^2-4(b+1)=0$, $a\neq 0$ | $\{0,\;-a/2\}$ → **2개** | ✔ 채택 |
| ② $h$의 한 근이 $0$ | $b+1=0$, $a\neq 0$ | $\{0,\;-a\}$ → **2개** | ✔ 잠재 분기 |
| ③ $h$의 두 근이 모두 $\neq 0$이고 판별식 $>0$ | $a^2-4(b+1)>0$, $b+1\neq 0$ | 3개 | ✘ |
| ④ $h$의 판별식 $<0$ | | 1개 | ✘ |

**분기 ②의 소거 가능성:** $b=-1$이면 (★)에서

$$f'(\alpha)\cdot f'(\beta) = -(-1)(a^2+4) = a^2+4 > 0$$

(다)를 $f'(\alpha)\cdot f'(\beta)=-N$($N>0$)으로 주면 $a^2+4=-N<0$이 되어 **불가능**. 따라서 $N>0$으로 설정하면 분기 ②는 자동 소거됩니다.

### 판정 1-2
분기 ①만 생존. **CONTINUE.**

## 1-3. 자유도 계산

| 단계 | 제약 | 남은 자유도 |
|---|---|---|
| 초기 | $a,\;b$ | 2 |
| (나) 분기 ① | $a^2-4b=4$ | 1 |
| (다) | $-b(a^2-4b)=-N$ → $-b\cdot 4=-N$ → $b=N/4$ | 0 |

$b$ 확정 후 $a^2=4b+4=N+4$이므로 $a=\pm\sqrt{N+4}$. $a$의 부호는 미정이나, 발문 $f'(0)=b$는 $a$에 무관하게 **유일**.

### 판정 1-3
자유도 0, 발문값 유일. **CONTINUE.**

## 1-4. 탐색 전략 수립

$N$은 양의 정수, $b=N/4$가 양의 정수, $a^2=N+4$가 완전제곱수인 값을 탐색합니다.
$a^2-4b>0$ (두 근 $\alpha,\beta$가 실수이고 서로 다름)은 $a^2-4b=4>0$으로 자동 충족.
$b\neq 0$은 $N\neq 0$으로 자동 충족.

**CONTINUE.**

## 후보별 검증

### 후보 ❶: $N=12$, $b=3$, $a=\pm 4$

$f(x)=x^3+4x^2+3x=x(x+1)(x+3)$ (또는 $x(x-1)(x-3)$)

- $f'(x)=3x^2+8x+3$
- $f'(-1)=3-8+3=-2$, $f'(-3)=27-24+3=6$, 곱$=-12$ ✓
- (나): $x^3+4x^2+4x=x(x+2)^2=0$ → 교점 $\{0,\,-2\}$ 2개 ✓
- $f'(0)=3$

손계산 무난, **채택 후보**.

### 후보 ❷: $N=32$, $b=8$, $a=\pm 6$

$f(x)=x^3+6x^2+8x=x(x+2)(x+4)$ (또는 $x(x-2)(x-4)$)

- $f'(x)=3x^2+12x+8$
- $f'(-2)=12-24+8=-4$, $f'(-4)=48-48+8=8$, 곱$=-32$ ✓
- (나): $x^3+6x^2+9x=x(x+3)^2=0$ → 교점 $\{0,\,-3\}$ 2개 ✓
- $f'(0)=8$

손계산 쾌적, 수치 규모 적절. **채택 후보**.

## 포장 설계 — 후보 ❷ 채택

| 설계 항목 | 결정 | 근거 |

| $\alpha,\beta$ 표기 | 조건 내 정의 | 수능 관례에 맞게 조건 안에서 기호 도입 |
