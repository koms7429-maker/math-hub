### 1-3. 조건 연립 + 풀이

**연립 순서**: (다) → (가) → (나)

1. (다)에서 $p = 4$ 즉시 결정.
2. (가)에서 $q = \pm 16$ 또는 $|q| > 16$.
3. (나)에서 $q \in \{16, -16, 2-10\sqrt{5}, -2+10\sqrt{5}\}$.
4. 교집합: 4개 값 모두 유효.

**전환 지점**:

- **전환 1**: (나)에서 $h_1 h_2 = 0$의 교점 수를 세기 위해 공통근 분석 필요 → $x_0 = \pm 2$에서 공통근 조건 $q = \pm 16$ 발견.
- **전환 2**: 공통근이 없는 경우($|q| > 16$)에서 5교점을 만들려면 이중근 조건 분석 → $h_1$ 또는 $h_2$의 극값 = 0 조건에서 무리수 $q$ 값 도출.

**최종 답**: $(f(0))^2 = q^2$

- $q = \pm 16$: $(f(0))^2 = 256$ ✓
- $q = 2 - 10\sqrt{5}$: $(f(0))^2 = (2-10\sqrt{5})^2 = 504 - 40\sqrt{5}$ (무리수) ✗ 정수 아님
- $q = -2 + 10\sqrt{5}$: $(f(0))^2 = (-2+10\sqrt{5})^2 = 504 - 40\sqrt{5}$ (무리수) ✗ 정수 아님

수능 답 형식(0~999 정수) 관점에서 무리수 답은 부적합하나, **수학적으로는 유효한 해**임. 문제 자체에 무리수 해를 배제하는 장치 없음.

## [STEP 2] 설계 의도 대조

설계 의도에 따르면:

**대조 결과**:
- "이중근+단근" 구조는 풀이에서 나타남 (O)
