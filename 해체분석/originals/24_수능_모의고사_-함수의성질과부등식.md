# 24. (수능/모의고사) - 함수의 성질과 부등식

## Problem

2 이상의 자연수 $n$에 대하여 $(n-p)(n-p-8)$의 $n$제곱근 중 실수인 것의 개수를 $f(n)$이라 하자. 함수 $f(n)$이 다음 조건을 만족시키도록 하는 20 이하의 자연수 $p$의 개수를 구하시오. [4점]

$$\text{2 이상의 모든 자연수 } m \text{에 대하여 } \sum_{n=2}^{m} \{f(n)-1\} \geq 1 \text{이다.}$$

**Answer**: 11
**Difficulty**: unknown
**Score**: 4점
**Type**: short answer

---

## Solution

$(n-p)(n-p-8)$의 부호를 고려하면

$$f(n)-1 = \begin{cases} 1 & (n\text{은 짝수},\ n < p \text{ 또는 } n > p+8) \\ -1 & (n\text{은 짝수}, \ p < n < p+8) \\ 0 & (\text{나머지 경우}) \end{cases}$$

이므로 문제의 박스 조건의 부등식

$$\sum_{n=2}^{m} \{f(n)-1\} \geq 1 \quad \cdots\ (\bigstar)$$

에서

$(\bigstar)$이 $m=2$일 때 성립함 $\Rightarrow f(2)-1=1 \Rightarrow p > 2$ ($\because$ $p$는 자연수)

임을 알 수 있고, $S_m = \displaystyle\sum_{n=2}^{m} \{f(n)-1\}$이라 하면

- $n \leq p$일 때 $f(n)-1 \geq 0 \Rightarrow S_2 = 1 \leq S_2 \leq \cdots \leq S_p$
$\Rightarrow 2 \leq m \leq p$일 때 $(\bigstar)$이 성립함

이고,

$$\begin{cases} p < n \leq p+8 \text{일 때 } f(n)-1 \leq 0 \Rightarrow S_p \geq \cdots \geq S_{p+8} \\ n > p+8 \text{일 때 } f(n)-1 \geq 0 \Rightarrow S_{p+8} \leq S_{p+9} \leq \cdots \end{cases}$$

$\Rightarrow S_{p+8} \geq 1$이면 $m \geq p$일 때 $(\bigstar)$이 성립함

을 알 수 있다.

이때,

$$S_{p+8} = \sum_{n=2}^{p+8} \{f(n)-1\}$$
$$= (2 \leq n < p \text{인 짝수 } n\text{의 개수}) - (p < n < p+8 \text{인 짝수 } n\text{의 개수})$$

이므로

① $p$가 홀수인 경우

$$S_{p+8} = \frac{p-1}{2} - 4 = \frac{p-9}{2} \Rightarrow S_{p+8} \geq 1 \text{이려면 } p \geq 11$$

② $p$가 짝수인 경우

$$S_{p+8} = \frac{p-2}{2} - 3 = \frac{p-8}{2} \Rightarrow S_{p+8} \geq 1 \text{이려면 } p \geq 10$$

이어야 한다.

따라서 ①, ②에 의해

$$(20 \text{ 이하의 자연수 } p\text{의 개수}) = 11$$

이다.
