### 완성된 문제

> $f(0)=0$인 최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여,
> 곡선 $y=f(x)$ 위의 서로 다른 두 점 $\mathrm{A}(\alpha,\,f(\alpha))$, $\mathrm{B}(\beta,\,f(\beta))$ $(\alpha<\beta)$에서의 각 접선의 기울기가 서로 같다.
> 직선 $\mathrm{AB}$와 곡선 $y=f(x)$의 교점 중 $\mathrm{A}$, $\mathrm{B}$가 아닌 점을 $\mathrm{C}(t,\,f(t))$라 하자.
> 다음 조건을 만족시킨다.
>
> **(가)** 직선 $\mathrm{AB}$의 기울기는 $0$이다.
>
> **(나)** $\beta-\alpha=2$
>
> **(다)** $f(t)=6$
>
> $t$의 값을 구하시오. **[정답: 2]**

## 문제 구조
### 문제 초안
> $f(0)=0$인 최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여,
> 곡선 $y=f(x)$ 위의 서로 다른 두 점 $\mathrm{A}(\alpha,\,f(\alpha))$, $\mathrm{B}(\beta,\,f(\beta))$ $(\alpha<\beta)$에서의 접선의 기울기가 서로 같다.
> 직선 $\mathrm{AB}$와 곡선 $y=f(x)$의 교점 중 $\mathrm{A}$, $\mathrm{B}$가 아닌 점을 $\mathrm{C}(t,\,f(t))$라 하자.
> 다음 조건을 만족시킨다.
>
> **(가)** 직선 $\mathrm{AB}$의 기울기는 $m$이다.
>
> **(나)** $\beta-\alpha=k$
>
> **(다)** $f(t)=v$
>
> $t$의 값을 구하시오.

*(여기서 $m$, $k$, $v$는 detail\_builder가 결정할 구체적 수치 자리표)*

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 숨은 사실과의 관계 |
|---|---|---|
| 본문 전제 "$f(0)=0$, 최고차 계수 $1$" | $f(x)=x^3+px^2+qx$ 설정 | 자유도를 $p,q$ 두 개로 축소 |
| 본문 설정 "접선의 기울기가 서로 같다" | 사실 A: $f'(\alpha)=f'(\beta)$ → $\alpha+\beta=-\tfrac{2p}{3}$ | 숨은 사실의 절반 (Vieta와 결합 시 $t$ 확정) |
| **(가)** 직선 AB의 기울기 $=m$ | 할선 기울기 → $q = m+\tfrac{p^2}{3}-\tfrac{k^2}{4}$ | (나)와 결합해야 $q(p)$ 도출; 단독으로는 $p$ 미결 |
| **(나)** $\beta-\alpha=k$ | 두 점의 간격 → $\alpha,\beta$를 $p$의 식으로 표현 | (가)와 결합해야 $q(p)$ 도출; 단독으로는 $p,q$ 미결 |
| **(다)** $f(t)=v$ | 제3 교점의 함숫값 → 숨은 사실 $t=-\tfrac{p}{3}$ 사용 시 $p$의 삼차방정식 | 숨은 사실 없이는 $t$ 자체를 모르므로 사용 불가 |
