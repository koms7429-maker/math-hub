## 완성된 문제

# [0] 이번 시도의 차별성 선언

첫 번째 시도이므로 이전 기록 없음.

- **수치 설정**: $m=2,\; d=\sqrt{3}$ (극댓값 2, 극솟값 −2인 대칭적 삼차)
- **조건 추가/변경**: 없음 (초안 구조 그대로)
- **발문 형태**: $f(5)$ (정수 답)
- **유일성 처리**: 접선 공명(tangency)이 $N=6$을 강제 → 추가 부등식 불필요

# STEP 1. 파라미터 결정 가능성 검토

## (가)의 해석

최고차항의 계수가 1인 삼차함수를 일반형으로 놓자:
$$f(x) = x^3 + ax^2 + bx + c$$

변곡점 $x_0 = -\tfrac{a}{3}$에서 $f$는 점대칭이다. $t = x - x_0$로 치환하면

$$f(x) = (x - m)^3 - d^2(x - m) + f(m)$$

여기서 $m = x_0$, $d^2 > 0$. 극값은

$$\text{극댓값} = f(m) + \frac{2d^3}{3\sqrt{3}},\quad \text{극솟값} = f(m) - \frac{2d^3}{3\sqrt{3}}$$

$M = \frac{2d^3}{3\sqrt{3}}$로 놓으면, 조건 (가) $|\text{극댓값}| = |\text{극솟값}|$는

$$|f(m) + M| = |f(m) - M|$$

$M > 0$이므로 유일한 해는 $f(m) = 0$. 따라서

$$\boxed{f(x) = (x-m)^3 - d^2(x-m)}$$

**자유 파라미터**: $m$과 $d\;(d>0)$ — 2개.

## (나)의 해석

$f(t)=0$의 근: $m-d,\; m,\; m+d$ (등차수열!)

$f(f(x))=0$ ⟺ $f(x) \in \{m-d,\; m,\; m+d\}$

각 수준방정식 $f(x)=k$의 서로 다른 실근 개수는:

| $k$ 범위 | 실근 수 |
|---|---|
| $-M < k < M$ | **3** |
| $k = \pm M$ | **2** (접선) |
| $|k| > M$ | **1** |

따라서 $N=6$이 되려면 근 개수 조합이 $(1,\;2,\;3)$이어야 한다. 즉 **정확히 하나의 근 값이 $\pm M$에 접선**, 하나는 범위 밖, 하나는 범위 안이어야 한다.

$m>0$ 가정 시 $m+d > m > m-d$이므로 자연스러운 배치:

- $m+d > M$ → 1개의 근 ✓
- $m = M$ → 2개의 근 (접선!) ✓
- $|m-d| < M$ → 3개의 근 ✓

접선 조건 $m = M$:
$$m = \frac{2d^3}{3\sqrt{3}} \quad \cdots (\star)$$

이것은 **1개의 등식 관계**이므로 2-파라미터 계열이 **1-파라미터**로 줄어든다.

## (다) $f(0) = -2$의 역할

$f(0) = (0-m)^3 - d^2(0-m) = -m^3 + d^2 m = m(d^2 - m^2)$

$(\star)$와 연립하면 $m$과 $d$가 유일하게 결정된다 → **미결 파라미터 0개**. ✅

# STEP 2. 수치 결정

## $(\star)$에서 $m = \frac{2d^3}{3\sqrt{3}}$을 $f(0) = -2$에 대입

$$m(d^2 - m^2) = -2$$

$m = \frac{2d^3}{3\sqrt{3}}$이므로 $m^2 = \frac{4d^6}{27}$. 따라서

$$\frac{2d^3}{3\sqrt{3}} \cdot \left(d^2 - \frac{4d^6}{27}\right) = -2$$

$$\frac{2d^5}{3\sqrt{3}} \cdot \frac{27 - 4d^4}{27} = -2$$

$$\frac{2d^5(27 - 4d^4)}{81\sqrt{3}} = -2$$

$$d^5(27 - 4d^4) = -81\sqrt{3}$$

**$d = \sqrt{3}$ 시도:**
$$(\sqrt{3})^5 (27 - 4 \cdot 9) = 9\sqrt{3} \cdot (27 - 36) = 9\sqrt{3} \cdot (-9) = -81\sqrt{3} \quad \checkmark$$

따라서 $d = \sqrt{3}$, $m = \frac{2 \cdot 3\sqrt{3}}{3\sqrt{3}} = 2$.

$$f(x) = (x-2)^3 - 3(x-2) = x^3 - 6x^2 + 9x - 2$$

### 검증

| 항목 | 계산 |
|---|---|
| 근 | $f(x) = (x-2)(x^2-4x+1)=0$ → $x = 2-\sqrt{3},\; 2,\; 2+\sqrt{3}$ ✓ |
| $f'(x) = 3(x-1)(x-3)$ | 극대 $x=1$, 극소 $x=3$ |
| 극댓값 | $f(1) = 1-6+9-2 = 2$ |
| 극솟값 | $f(3) = 27-54+27-2 = -2$ |
| $\|2\| = \|-2\|$ | ✓ **(가) 만족** |
| $f(0) = 0-0+0-2 = -2$ | ✓ **(다) 만족** |

$f(f(x))=0$의 근 개수 (나):

| 수준방정식 | 값 | $\pm M = \pm 2$과 비교 | 실근 수 |
|---|---|---|---|
| $f(x) = 2+\sqrt{3} \approx 3.73$ | $> 2$ | 범위 밖 | **1** |
| $f(x) = 2$ | $= M$ | 접선 | **2** |
| $f(x) = 2-\sqrt{3} \approx 0.27$ | $\in (-2, 2)$ | 범위 안 | **3** |

세 방정식의 우변이 모두 다르므로 근의 중복 없음.
$$N = 1 + 2 + 3 = 6 \quad \checkmark \quad \textbf{(나) 만족}$$

### 발문 계산

$$f(5) = 125 - 150 + 45 - 2 = \boxed{18}$$

# STEP 3. 유일성 확인

$N = 6$을 만드는 접선 배치는 다음 네 가지 시나리오:

| 시나리오 | 접선 조건 | $f(0) = -2$ 대입 결과 |
|---|---|---|
| ① $m = M$ ($m > 0$) | $m = \frac{2d^3}{3\sqrt{3}}$ | $d = \sqrt{3},\; m = 2$ ✅ |
| ② $m = -M$ ($m < 0$) | $m = -\frac{2d^3}{3\sqrt{3}}$ | $d^5(27 - 4d^4) = 81\sqrt{3}$; 함수 $g(d)$의 최댓값 $\approx 62.5 < 81\sqrt{3} \approx 140.3$ → **해 없음** ✗ |
| ③ $m - d = -M$ | $m = d - \frac{2d^3}{3\sqrt{3}}$ | $d = \sqrt{3}$일 때 $f(0) \approx -0.78 \neq -2$; 전체 수치 탐색에서도 $-2$ 도달 불가 ✗ |
| ④ $m + d = -M$ | 모든 근 음수 | $f(0) \approx -40.78$; 규모 불일치 ✗ |

**결론: 해는 $f(x) = x^3 - 6x^2 + 9x - 2$로 유일.** 답 $f(5) = 18$. ✅
