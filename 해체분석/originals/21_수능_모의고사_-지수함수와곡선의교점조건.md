# 21. (수능/모의고사) - 지수함수와 곡선의 교점 조건

## Problem

두 곡선 $y = 2^{x-1} - 4$, $y = -\dfrac{1}{7} \times 2^{n-x} + a$가 서로 다른 두 점 A, B에서 만날 때, 다음 조건이 성립하도록 하는 모든 자연수 $n$의 값의 합은? [4점]

$$\text{선분 AB가 } x\text{축과 만나지 않으면서 } y\text{축과 만나도록 하는 정수 } a\text{의 개수가 3이다.}$$

## Choices
① 1 &emsp; ② 3 &emsp; ③ 5 &emsp; ④ 7 &emsp; ⑤ 9

**Answer**: ③ 5
**Difficulty**: 상
**Score**: 4점
**Type**: multiple choice

---

## Solution

곡선 $y = 2^{x-1} - 4$는 두 점 $\left(0,\, -\dfrac{7}{2}\right)$, $(3,\, 0)$을 지나고 증가함을 알 수 있으므로

$f(x) = -\dfrac{1}{7} \times 2^{n-x} + a$라 하고 다음 그림을 참고하면

선분 AB가 $x$축과 만나지 않으면서 $y$축과 만남 $\Rightarrow$ $f(0) \ge -\dfrac{7}{2}$, $f(3) < 0$ $\cdots$ ⓐ

[FIGURE: 곡선 $y = 2^{x-1} - 4$와 $y = f(x)$가 두 점 A, B에서 만나는 그래프. A는 $y$축 왼쪽 아래, B는 $y$축 오른쪽에 위치하며, 선분 AB가 $x$축과 만나지 않고 $y$축과 만나는 상황을 보여줌]

따라서

$$\text{ⓐ} \Rightarrow a - \frac{1}{7} \times 2^n \ge -\frac{7}{2}, \quad a - \frac{1}{7} \times 2^{n-3} < 0$$

$$\Rightarrow \frac{1}{7} \times 2^n - \frac{7}{2} \le a < \frac{1}{7} \times 2^{n-3} \quad \cdots \text{ⓑ}$$

이고, 이때

- ⓑ을 만족시키는 $a$가 존재함 $\Rightarrow \dfrac{1}{7} \times 2^n - \dfrac{7}{2} < \dfrac{1}{7} \times 2^{n-3}$
  $\Rightarrow n \le 4$ ($\because$ $n$은 자연수)

- $n \le 4$일 때, $0 < \dfrac{1}{7} \times 2^{n-3} < 1$

이므로 다음 그림을 참고하면

ⓑ을 만족시키는 정수 $a$의 개수가 3 ($\because$ 문제의 박스 조건)

$\Rightarrow a = 0,\, -1,\, -2$만 ⓑ을 만족시킴

ⓑ의 첫 번째 부등식이 $a = -2$일 때 성립하고, $a = -3$일 때 성립하지 않아야 함

$$\Rightarrow -3 < \frac{1}{7} \times 2^n - \frac{7}{2} \le -2$$

$$\Rightarrow n = 2,\, 3 \quad (\because\, n\text{은 자연수})$$

임을 알 수 있다.

[FIGURE: $y = \dfrac{1}{7} \times 2^{x-3}$과 $y = \dfrac{1}{7} \times 2^x - \dfrac{7}{2}$의 그래프에서 $n$의 범위를 확인하는 그림]

$$\therefore\, (\text{모든 자연수 } n\text{의 값의 합}) = 2 + 3 = 5$$
