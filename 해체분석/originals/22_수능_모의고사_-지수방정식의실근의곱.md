# 22. (수능/모의고사) - 지수 방정식의 실근의 곱

## Problem

다음 조건을 만족시키는 $2$ 이상의 모든 자연수 $n$의 값의 합은? [4점]

$x$에 대한 방정식
$$(x^n - 2^{25})(x^{2n} - 2^{20}) = 0$$
의 서로 다른 모든 실근의 곱은 $-100$보다 큰 정수이다.

## Choices
① 165 &emsp; ② 168 &emsp; ③ 171 &emsp; ④ 174 &emsp; ⑤ 177

**Answer**: ①
**Difficulty**: unknown
**Score**: 4점
**Type**: multiple choice

---

## Solution

주어진 방정식을 인수분해하면

$$(x^n - 2^{25})(x^n - 2^{10})(x^n + 2^{10}) = 0$$

**( i ) $n$이 홀수일 때:**

$$x^n - 2^{25} = 0 \Rightarrow x = 2^{\frac{25}{n}}$$
$$x^n - 2^{10} = 0 \Rightarrow x = 2^{\frac{10}{n}}$$
$$x^n + 2^{10} = 0 \Rightarrow x = -2^{\frac{10}{n}}$$

모든 실근의 곱 $= 2^{\frac{25}{n}} \cdot 2^{\frac{10}{n}} \cdot \left(-2^{\frac{10}{n}}\right) = -2^{\frac{45}{n}}$

이 값이 정수이려면 $\frac{45}{n}$이 음이 아닌 정수, 즉 $n \mid 45$이어야 한다.

$-100$보다 큰 조건: $-2^{\frac{45}{n}} > -100$이므로 $2^{\frac{45}{n}} < 100$

$45$의 홀수 약수 중 $2$ 이상: $3,\; 5,\; 9,\; 15,\; 45$

- $n = 3$: $2^{15} = 32768 > 100$ ❌
- $n = 5$: $2^{9} = 512 > 100$ ❌
- $n = 9$: $2^{5} = 32 < 100$ ✅
- $n = 15$: $2^{3} = 8 < 100$ ✅
- $n = 45$: $2^{1} = 2 < 100$ ✅

$\Rightarrow n = 9,\; 15,\; 45$

**( ii ) $n$이 짝수일 때:**

$$x^n - 2^{25} = 0 \Rightarrow x = \pm\, 2^{\frac{25}{n}}$$
$$x^n - 2^{10} = 0 \Rightarrow x = \pm\, 2^{\frac{10}{n}}$$
$$x^n + 2^{10} = 0 \Rightarrow \text{실수 } x \text{는 없음}$$

모든 실근의 곱 $= \left(-2^{\frac{25}{n}}\right)\cdot 2^{\frac{25}{n}} \cdot \left(-2^{\frac{10}{n}}\right)\cdot 2^{\frac{10}{n}} = 2^{\frac{70}{n}}$

이 값이 정수이려면 $n \mid 70$이어야 한다. 곱이 양수이므로 $> -100$ 조건은 자동 충족.

$70$의 짝수 약수 중 $2$ 이상: $2,\; 10,\; 14,\; 70$

$\Rightarrow n = 2,\; 10,\; 14,\; 70$

**따라서 모든 자연수 $n$의 값의 합:**

$$(9 + 15 + 45) + (2 + 10 + 14 + 70) = 69 + 96 = \boxed{165}$$

23. (수능/모의고사) - 지수함수와 직선의 교점 거리

## Problem

직선 $y = k$가 두 곡선 $y = 2^{7-x}$, $y = 8^x$과 각각 두 점 P, Q에서 만날 때, 두 점 P, Q 사이의 거리가 $3$이 되도록 하는 모든 양수 $k$의 값의 곱은? [4점]

## Choices
① $2^{\frac{13}{2}}$ ② $2^{\frac{15}{2}}$ ③ $2^{\frac{17}{2}}$ ④ $2^{\frac{19}{2}}$ ⑤ $2^{\frac{21}{2}}$

**Answer**: ⑤
**Difficulty**: unknown
**Score**: 4
**Type**: multiple choice

---

## Solution

$y = 2^{7-x}$은 감소하고, $y = 8^x$은 증가하므로 P, Q로 가능한 점을

① (P의 $x$좌표) $<$ (Q의 $x$좌표)인 경우 $P_1$, $Q_1$

② (P의 $x$좌표) $>$ (Q의 $x$좌표)인 경우 $P_2$, $Q_2$

로 나타내면 다음과 같다.

[FIGURE: 두 곡선 $y=2^{7-x}$, $y=8^x$의 그래프에서 높은 위치의 수평선이 $P_1$, $Q_1$과 만나고 (거리 3), 낮은 위치의 수평선이 $Q_2$, $P_2$와 만나는 (거리 3) 모습]

이때,

$$(\text{P의 } y\text{좌표}) = (\text{Q의 } y\text{좌표}) = k \implies \begin{cases} (\text{P의 } x\text{좌표}) = 7 - \log_2 k \\ (\text{Q의 } x\text{좌표}) = \log_8 k \end{cases}$$

이므로 ①의 경우 $\overline{\text{PQ}} = 3$이려면

$$\log_8 k - (7 - \log_2 k) = 3 \implies \log_2 k = \frac{15}{2} \implies k = 2^{\frac{15}{2}}$$

이어야 하고, ②의 경우 $\overline{\text{PQ}} = 3$이려면

$$(7 - \log_2 k) - \log_8 k = 3 \implies \log_2 k = 3 \implies k = 2^3$$

이어야 한다.

$$\therefore \ (\text{모든 양수 } k\text{의 값의 곱}) = 2^{\frac{15}{2}} \times 2^3 = 2^{\frac{21}{2}}$$
