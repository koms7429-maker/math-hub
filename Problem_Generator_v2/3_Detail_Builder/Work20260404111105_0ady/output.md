# STEP 1. 구조 분석 (기호만 사용)

## 1-1. 조건이 만드는 방정식 체계

**(가)+(나)에서 함수 형태 확정**

최고차항의 계수가 $1$인 사차함수 $f(x) \ge 0$이 서로 다른 두 실근을 가지면, 모든 실근은 짝수 중복도를 가져야 하고, 차수 4를 두 근에 분배하면:

$$f(x) = (x-\alpha)^2(x-\beta)^2, \quad \alpha \neq \beta$$

$s = \alpha+\beta,\; p = \alpha\beta$로 놓으면 $f(x) = (x^2 - sx + p)^2$이다.

**(다)에서 연립방정식 생성**

$\{f(a)-b\}^2 + \{f(-a)-b\}^2 = 0$에서 비음수의 합이 $0$이므로:
$$f(a) = b \quad \text{그리고} \quad f(-a) = b$$

$$f(a) = (a^2 - sa + p)^2 = b \;\Longrightarrow\; a^2 - sa + p = \pm\sqrt{b}$$
$$f(-a) = (a^2 + sa + p)^2 = b \;\Longrightarrow\; a^2 + sa + p = \pm\sqrt{b}$$

이 두 식을 조합하면 **4가지 부호 경우**가 발생한다:

| # | $a^2-sa+p$ | $a^2+sa+p$ | 결과 |
|---|---|---|---|
| ① | $+\sqrt{b}$ | $+\sqrt{b}$ | $s = 0$, $p = \sqrt{b} - a^2$ |
| ② | $+\sqrt{b}$ | $-\sqrt{b}$ | $s = \sqrt{b}/a$, $p = -a^2$ |
| ③ | $-\sqrt{b}$ | $+\sqrt{b}$ | $s = -\sqrt{b}/a$, $p = -a^2$ |
| ④ | $-\sqrt{b}$ | $-\sqrt{b}$ | $s = 0$, $p = -\sqrt{b} - a^2$ |

## 1-2. 분기 분석

각 경우에 $\alpha \neq \beta$ (즉 $s^2 - 4p > 0$) 및 $\alpha, \beta \in \mathbb{R}$ (즉 $s^2 - 4p \ge 0$) 확인 필요:

- **①**: $s^2 - 4p = 4a^2 - 4\sqrt{b} > 0 \iff a^2 > \sqrt{b}$. 경계 $a^2 = \sqrt{b}$이면 $\alpha = \beta$로 탈락.
- **④**: $s^2 - 4p = 4a^2 + 4\sqrt{b} > 0$. 항상 성립.
- **②, ③**: $s^2 - 4p = b/a^2 + 4a^2 > 0$. 항상 성립.

**판정 1-2**: 분기 4개가 명확하고, $a^2 > \sqrt{b}$일 때 최대 4해, $a^2 \le \sqrt{b}$일 때 최대 3해. 고교 범위 내에서 완결. → **CONTINUE**

## 1-3. 자유도 계산

- 원래 자유도: $(s, p)$ 2개
- 방정식 2개 ($f(a) = b$, $f(-a) = b$)가 절대값을 포함하여 이산 분기를 생성
- 자유도 $2 - 2 = 0$ (이산해)

**판정 1-3**: 자유도 $0$으로 해가 유한 개. → **CONTINUE**

## 1-4. 탐색 전략

$a, b$를 정하여 (i) $\sqrt{b}$가 정수, (ii) $s$가 정수, (iii) 해가 정확히 4개, (iv) $\sum f(c)$가 깔끔한 정수가 되도록 탐색.

$s$가 정수이려면 $\sqrt{b}/a \in \mathbb{Z}$. → **CONTINUE**

## 후보 탐색

### 후보 1: $a = 3,\; b = 9$

$\sqrt{b} = 3$. $a^2 = 9 > 3 = \sqrt{b}$ → 4해 기대.

| 분기 | $s$ | $p$ | $s^2-4p$ | 유효 |
|---|---|---|---|---|
| ① | $0$ | $-6$ | $24 > 0$ | ✓ |
| ② | $1$ | $-9$ | $37 > 0$ | ✓ |
| ③ | $-1$ | $-9$ | $37 > 0$ | ✓ |
| ④ | $0$ | $-12$ | $48 > 0$ | ✓ |

**4개의 함수:**

| 함수 | 수식 | 실근 |
|---|---|---|

| $f_3$ | $(x^2+x-9)^2$ | $\frac{-1\pm\sqrt{37}}{2}$ |
| $f_4$ | $(x^2-12)^2$ | $\pm 2\sqrt{3}$ |

**조건 검증:**

| 함수 | $f(3)$ | $f(-3)$ | (다) 확인 |
|---|---|---|---|
| $f_1$ | $(9-6)^2=9$ | $(9-6)^2=9$ | $(0)^2+(0)^2=0$ ✓ |
| $f_2$ | $(9-3-9)^2=9$ | $(9+3-9)^2=9$ | ✓ |
| $f_3$ | $(9+3-9)^2=9$ | $(9-3-9)^2=9$ | ✓ |
| $f_4$ | $(9-12)^2=9$ | $(9-12)^2=9$ | ✓ |

**정수성** ✓, **손계산** ✓, **유일성** (4해 모두 확정, 발문이 합이므로 답 유일) ✓.

## 완성된 문제

> 최고차항의 계수가 $1$인 사차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** 모든 실수 $x$에 대하여 $f(x) \ge 0$이다.
>
> **(나)** 방정식 $f(x) = 0$은 서로 다른 두 실근을 갖는다.
>
> **(다)** $\{f(3)-9\}^{2}+\{f(-3)-9\}^{2}=0$
>
> 조건을 만족시키는 모든 함수 $f$에 대하여 $f(2)$의 값의 합을 구하시오.
>
> **\[126\]**
