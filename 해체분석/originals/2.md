# 2. 지수·로그 — 집합과 이차함수

## 문제

이차함수 $f(x)$에 대하여 두 집합

$$A = \{x \mid x \text{는 실수}\},$$

$$B = \{x \mid \log|4^x - 3| \times \log f(x) \ge 0\}$$

이 있다. $n(A - B) = 2$일 때, $f(3)$의 값을 구하시오. [4점]

---

## 정답

$$\boxed{81}$$

---

## 해설

### 집합 B의 원소 조건

집합 $B$의 원소인 $x$는

- $4^x - 3 \ne 0$이고, $f(x) > 0$ (∵ 진수조건)
- $(|4^x - 3| - 1) \times \{f(x) - 1\} \ge 0$

을 **모두** 만족시켜야 한다.

### $A - B$의 원소 분석

$$4^x - 3 = 0 \iff x = \log_4 3$$

은 집합 $A - B$의 원소이다.

이때

- $f(x) < 0$인 $x$는 없거나 무수히 많음
- $(|4^x - 3| - 1) \times \{f(x) - 1\} < 0$인 $x$는 없거나 무수히 많음

을 참고하면

$$n(A - B) = 2$$

를 만족시키기 위해서는

> ① $f(x) = 0$인 $x$의 개수는 1이고, 이때의 $x \ne \log_4 3$
>
> ② 모든 $x$에 대하여 $(|4^x - 3| - 1) \times \{f(x) - 1\} \ge 0$

임을 알 수 있다.

### $f(x)$의 결정

한편,

$$|4^x - 3| \ge 1 \iff x \le \frac{1}{2}, \quad x \ge 1$$

이므로

$$②\implies \begin{cases} x \le \dfrac{1}{2},\; x \ge 1 \text{에서 } f(x) \ge 1 \\[6pt] \dfrac{1}{2} < x < 1 \text{에서 } f(x) < 1 \end{cases}$$

$$\implies f(x) = k\!\left(x - \frac{1}{2}\right)(x - 1) + 1 \quad (\text{단, } k > 0)$$

로 놓으면

$$① \implies f\!\left(\frac{3}{4}\right) = 0 \quad (\because \text{이차함수의 대칭축})$$

$$\implies k = 16$$

을 얻는다.

### 최종 답

$$\therefore\; f(x) = 16\!\left(x - \frac{1}{2}\right)(x - 1) + 1 \implies f(3) = \boxed{81}$$
