# STEP 1: 구조 분석 (기호만 사용)

## 1-1. 조건이 만드는 방정식 체계

$f(x) = x^3 + ax^2 + bx + c$로 놓자. 극대점 $x = \alpha$, 극소점 $x = \beta$ ($\alpha < \beta$, $d = \beta - \alpha > 0$).

$$f'(x) = 3(x-\alpha)(x-\beta) \implies a = -\tfrac{3(\alpha+\beta)}{2},\quad b = 3\alpha\beta$$

**직선 $\ell$과 곡선의 교점 방정식:**

$$f(x) - f(\alpha) - (x-\alpha) = (x-\alpha)\,Q(x) = 0$$

여기서 $Q(x)$는 이차식이고, $Q(x) = 0$의 두 근이 B, C의 $x$좌표 $h_1, h_2$이다.

**핵심 관계 (기호 수준 도출):**

| 관계식 | 내용 |
|--------|------|
| $(h_1 - \alpha)(h_2 - \alpha) = -1$ | 항상 성립 (숨은 구조) |
| $(h_1 - \alpha) + (h_2 - \alpha) = \frac{3d}{2}$ | $h_1 + h_2 = 2\alpha + \frac{3d}{2}$ |
| $h_1 h_2 = \alpha^2 + 3\alpha\frac{d}{2} + \frac{3d^2}{4} - 1$ (간략히 $= \alpha^2 + \frac{3d}{2}\alpha + \frac{3d^2}{4} - 1$) | Vieta 상수항 |
| $\overline{BC}^2 = 2(h_1 - h_2)^2 = \frac{9d^2}{2} + 8$ | **$d$만의 함수** |
| 극값 차 $= f(\alpha) - f(\beta) = \frac{d^3}{2}$ | **$d$만의 함수** |

> **관찰:** $\overline{BC}^2$와 극값 차 모두 $d$만으로 결정된다. 따라서 PM 초안의 (나)와 (다)는 같은 자유도($d$)만 구속하여 하나가 잉여이다.

**$f(0)$을 구하려면 $\alpha$와 $c$도 필요하다.**

$$f(\beta) = (\alpha+d)^2(\alpha + d - 3(\alpha+1) + 3\alpha)\Big|_{\text{정리}} + c \;=\; \text{(}\alpha, d, c\text{의 함수)}$$

$$f(0) = c$$

## 1-2. 분기 분석

직선 $\ell$이 A가 아닌 **서로 다른 두 점**에서 만나려면 $Q(x)=0$의 판별식 $> 0$이어야 한다:

$$\Delta = \frac{9d^2}{4} + 4 > 0 \quad (\text{항상 성립})$$

또한 $Q(\alpha) \neq 0$이어야 하는데, $Q(\alpha) = (h_1-\alpha)(h_2-\alpha)|_{x=\alpha} = -1 \neq 0$. ✅

분기 없이 $d > 0$이면 항상 B, C 존재.

## 1-3. 자유도 계산

| 항목 | 개수 |
|------|------|
| 파라미터 | $\alpha, d, c$ — **3개** |
| 방정식 필요 | $f(0)$을 유일하게 결정하려면 **3개** |

PM 초안의 (나), (다)는 $d$만 결정 → 실질 방정식 **1개**. 부족분 **2개**.

## 1-4. 탐색 전략 수립

**전략: 조건 체계를 수정하여 3개의 독립 방정식을 확보한다.**

PM 초안을 최소한으로 변형:
- **(나)** $\overline{BC}^2 = k$ → $d$ 결정 (방정식 1)
- **(다)** "두 점 B, C의 $x$좌표의 곱 $= n$" → $h_1 h_2 = n$ → $\alpha$ 결정 (방정식 2)
- **(가)** "극솟값 $= v$" → $c$ 결정 (방정식 3)

이렇게 하면 PM이 의도한 핵심 구조(극대점·기울기 1 직선·교점 B, C)는 유지된다.

---

**$h_1 h_2$를 $\alpha, d$로 표현:**

$$h_1 + h_2 = 2\alpha + \frac{3d}{2}, \qquad (h_1-\alpha)(h_2-\alpha) = -1$$

$$h_1 h_2 = (h_1-\alpha)(h_2-\alpha) + \alpha(h_1+h_2) - \alpha^2 = -1 + \alpha\!\left(2\alpha + \frac{3d}{2}\right) - \alpha^2 = \alpha^2 + \frac{3d}{2}\alpha - 1$$

**$f(\beta) = v$로부터 $c$ 결정:**

$$f(\alpha) = \alpha^3 + 6\alpha^2 + c \quad\text{(}d=4\text{일 때 확인 예정)}$$

$$f(\beta) = f(\alpha) - \frac{d^3}{2}$$

**$\alpha$의 유일성 조건:** $h_1 h_2 = n$에서

$$\alpha^2 + \frac{3d}{2}\alpha - 1 = n \quad \Longrightarrow \quad \alpha^2 + \frac{3d}{2}\alpha - (1+n) = 0$$

$\alpha$가 유일하려면 **판별식 $= 0$**: $\frac{9d^2}{4} + 4(1+n) = 0$, 즉 $n = -1 - \frac{9d^2}{16}$.

**$d$ 후보** ($\frac{9d^2}{2}+8$과 $\frac{d^3}{2}$이 정수):

| $d$ | $\overline{BC}^2 = \frac{9d^2}{2}+8$ | 극값 차 $= \frac{d^3}{2}$ | $n = -1-\frac{9d^2}{16}$ |
|-----|------|------|------|
| 2 | 26 | 4 | $-1-\frac{9}{4} = -\frac{13}{4}$ ← 비정수 |
| 4 | 80 | 32 | $-1-9 = -10$ ✅ 정수 |
| 6 | 170 | 108 | $-1-\frac{81}{4}$ ← 비정수 |
| 8 | 296 | 256 | $-1-36 = -37$ ✅ 정수 |

**$d = 4$ 선택** (수치가 가장 깔끔).

$d=4$일 때:
- $\overline{BC}^2 = 80$
- $n = h_1 h_2 = -10$
- $\alpha^2 + 6\alpha + 9 = 0 \Longrightarrow (\alpha+3)^2 = 0 \Longrightarrow \alpha = -3$ (유일)
- $\beta = 1$

**$c$ 결정을 위한 (가) 극솟값 $v$ 후보:** $v$는 임의이므로 $v = 0$ 선택 (깔끔).

$$f(\beta) = f(1) = 0 \Longrightarrow c \text{ 결정}$$

직선: $y = f(-3) + 1\cdot(x-(-3)) = 32 + (x+3) = x + 35$

$$x^3 + 3x^2 - 9x + 5 = x + 35$$

$$x^3 + 3x^2 - 10x - 30 = 0$$

$$x^2(x+3) - 10(x+3) = (x+3)(x^2-10) = 0$$

## 완성된 문제

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여 $f'(x)=0$의 서로 다른 두 실근을 $\alpha,\;\beta$ $(\alpha<\beta)$라 하자.
>
> **(가)** $f(\beta) = 0$
>
> 곡선 $y=f(x)$ 위의 점 $\mathrm{A}\!\left(\alpha,\; f(\alpha)\right)$를 지나고 기울기가 $1$인 직선 $\ell$이 곡선 $y=f(x)$와 만나는 $\mathrm{A}$가 아닌 서로 다른 두 점을 $\mathrm{B}$, $\mathrm{C}$라 하자.
>
> **(나)** $\overline{\mathrm{BC}}^{\,2} = 80$
>
> **(다)** 두 점 $\mathrm{B}$, $\mathrm{C}$의 $x$좌표의 곱은 $-10$이다.
>
> $f(0)$의 값을 구하시오. $\qquad$ **[정답: $5$]**

## 출처 아이디어
**수학적 객체:** 극대와 극소를 모두 갖는 삼차함수 $f(x)$ (매개변수를 포함)

**사실 A가 부과하는 제약 (P005 — 차분방정식 환원):**
삼차곡선 위의 한 점 $(a, f(a))$에서 출발하여, 곡선 위의 다른 점 $(a+h, f(a+h))$와 함께 축정렬 정사각형의 인접 꼭짓점을 이루는 조건은 $f(a+h) - f(a) = h$로 번역된다. $h \neq 0$으로 나누면 $h$에 대한 이차방정식이 되어, 기준점 $a$에서 가능한 변위는 최대 2개이며, 그 곱과 합은 비에타 관계식으로 $a$와 함수의 매개변수에 의해 표현된다.

**사실 B가 부과하는 제약 (P013 — 임계값의 이중근 구조):**
기준점 $a$가 삼차함수의 극점(극대점 또는 극소점)인 경우, $f'(a) = 0$이 성립한다. 이 조건은 차분방정식에서 $h$의 이차방정식의 상수항을 결정하는 핵심 계수($f'(a)$가 포함된 항)를 소실시킨다.

**숨은 사실 (A + B 동시 성립 시 강제되는 것):**
극점 조건 $f'(a) = 0$이 이차방정식의 상수항에서 자유 매개변수를 정확히 상쇄하여, **두 변위의 곱이 삼차함수의 형태와 무관한 보편 상수 $-1$**이 된다. 이는 곱이 음수이므로 두 변위가 반드시 부호가 반대—극점의 좌우로 하나씩 정사각형이 형성됨—을 의미하며, 더 나아가 한 변위를 $h_1$으로 놓으면 다른 변위는 $-1/h_1$로 완전히 결정된다.

## 수험생의 발견 순간
1. **수험생이 처음 시도하는 것:**
   삼차함수의 극점에서 축정렬 정사각형 조건을 세우고, $f(a+h) - f(a) = h$를 전개한 뒤 $h$로 나누어 이차방정식을 얻는다. 이 이차방정식의 판별식이나 근의 개수를 분석하려 한다.

2. **막히는 순간:**
   이차방정식의 계수에 $a$와 매개변수 $p$가 모두 얽혀 있어, 두 변위 $h_1, h_2$의 구체적 값을 구할 수 없다. 근의 곱 $h_1 h_2 = 3a^2 - p - 1$이 두 미지수에 의존하는 것처럼 보여 진행이 멈춘다.

3. **발견:**
   $a$가 극점이라는 조건 $f'(a) = 3a^2 - p = 0$, 즉 $3a^2 = p$를 대입하는 순간, $h_1 h_2 = p - p - 1 = -1$이 되어 **매개변수가 완전히 사라진다.** 삼차함수가 어떤 모양이든 극점에서의 정사각형 변위곱은 항상 $-1$이다.

4. **숨은 사실과의 대응:**
   이 발견은 STEP 2-2의 숨은 사실—"극점 조건이 차분방정식의 매개변수 의존성을 정확히 소거하여 변위곱이 보편 상수가 된다"—과 정확히 대응한다. 수험생은 $h_1 h_2 = -1$이라는 단순한 관계를 통해 한 변위로부터 다른 변위를 즉시 결정할 수 있게 되며, 이를 이용해 정사각형의 개수나 넓이의 합 등 후속 조건을 해결할 수 있다.

## 문제 구조
### 문제 초안
> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** $f(x)$는 극댓값과 극솟값을 모두 갖는다.
>
> 곡선 $y=f(x)$ 위의 극대점을 $\mathrm{A}$라 하고, 점 $\mathrm{A}$를 지나며 기울기가 $1$인 직선 $\ell$이 곡선 $y=f(x)$와 만나는 $\mathrm{A}$가 아닌 서로 다른 두 점을 $\mathrm{B}$, $\mathrm{C}$라 하자.
>
> **(나)** $\overline{\mathrm{BC}}^{\,2} = k$ $\quad$ *(수치 $k$는 미결)*
>
> **(다)** $f(x)$의 극댓값과 극솟값의 차는 $m$이다. $\quad$ *(수치 $m$은 미결)*
>
> $f(0)$의 값을 구하시오. $\quad$ *(또는 $\overline{\mathrm{AB}}\times\overline{\mathrm{AC}}$ 등 적절한 발문)*
