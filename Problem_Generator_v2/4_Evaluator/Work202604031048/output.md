## 이벨류에이터

### 원본 문제 A
## Problem

**1.** 2022학년도 6월 평가원

삼차함수 $f(x)$가 다음 조건을 만족시킨다.

> (가) 방정식 $f(x) = 0$의 서로 다른 실근의 개수는 2이다.
> (나) 방정식 $f(x - f(x)) = 0$의 서로 다른 실근의 개수는 3이다.

$f(1) = 4$, $f'(1) = 1$, $f'(0) > 1$일 때, $f(0) = \frac{q}{p}$이다. $p + q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

### 원본 문제 B
## Problem

**11.** 두 상수 $a$, $b$ ($b < 0$)에 대하여 함수

$$f(x) = \begin{cases} x^3 + 4x^2 + b & (x < 0) \\ x^3 - ax^2 + ax + b & (x \geq 0) \end{cases}$$

이 다음 조건을 만족시킬 때, $27(a+b)$의 값을 구하시오.

> (가) 방정식 $f(x) = 0$의 모든 실근을 나열하면
> $\alpha$, $\beta$, $\gamma$ ($\alpha < \beta < \gamma$)이고 $\alpha + \beta + \gamma = -4$이다.
> (나) $f'(\alpha) + f'(\beta) + f'(\gamma) = 14$
