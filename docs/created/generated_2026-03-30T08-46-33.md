# [2] 각 아이디어가 만드는 계수 관계식 (기호 수준)

**공통 무대**: 모닉 삼차 다항식 $f(x) = (x-\alpha)(x-\beta)(x-\gamma)$, 근 $\alpha, \beta, \gamma$ (서로 다른 실수), 변곡점 $s = \frac{\alpha+\beta+\gamma}{3}$.

### 아이디어 P004가 만드는 관계식 $R_1$

> 두 서로 다른 근 $\alpha, \beta$에서 도함수 값이 같다: $f'(\alpha) = f'(\beta)$.

$f'(\alpha) - f'(\beta) = (\alpha - \beta)\bigl[(\alpha+\beta) - 2\gamma\bigr] = 0$이고 $\alpha \neq \beta$이므로,

$$R_1:\quad \alpha + \beta = 2\gamma$$

이는 $\gamma$가 $\alpha$와 $\beta$의 **중점**임을 의미한다. $s := \gamma = \frac{\alpha+\beta}{2}$로 놓고 $d := \frac{\alpha-\beta}{2}$ ($d \neq 0$)로 매개변수화하면 $\alpha = s+d,\; \beta = s-d$이다. 이때 $\alpha+\beta+\gamma = 3s$이므로 변곡점은 정확히 $x = s$에 위치한다.

변환 $t = x - s$를 적용하면 $f$는 **짝수 이차항이 소멸한 감소형(depressed form)**으로 환원된다:

$$g(t) := f(t+s) = t^3 - d^2\,t$$

자유 파라미터: $s$ (평행이동), $d$ (근의 반간격). $R_1$만으로는 $s$와 $d$ 모두 미결정.

---

### 아이디어 P001이 만드는 관계식 $R_2$

> 자기합성 방정식 $f(x - f(x)) = 0$의 **실근의 총 개수**가 특정 값 $N$으로 고정된다.

$f(x-f(x))=0$은 $x-f(x) \in \{\alpha,\,\beta,\,\gamma\}$로 분해되어, **기울기 1인 세 평행선** $y = x - r_i$와 $y = f(x)$의 교점 문제의 합집합이 된다:

$$\text{(i)}\; f(x) = x - \alpha,\quad \text{(ii)}\; f(x) = x - \beta,\quad \text{(iii)}\; f(x) = x - \gamma$$

이는 모두 $h(x) := f(x) - x$의 수준집합(level set)이므로, $h(x) = -\alpha,\; -\beta,\; -\gamma$의 실근 개수의 합이 $N$이다.

$$R_2:\quad \sum_{i} \#\{x \in \mathbb{R} : h(x) = -r_i\} = N$$

각 수준집합의 근 개수는 $-r_i$가 $h$의 **극대값·극소값**(임계값)과 어떤 위치 관계에 있느냐로 결정된다. $N$을 적절히 고정하면, 최소 하나의 분기에서 $-r_i$가 임계값과 **정확히 일치**(접선/이중근 조건)해야 하는 제약이 발생한다.

자유 파라미터: $f$의 계수 전체 ($\alpha, \beta, \gamma$ 또는 등가적으로 $s, d$, 그리고 기울기 조건으로부터 추가 구속).

# [3] 파라미터 공유 확인

$R_1$과 $R_2$는 모두 **같은 파라미터 집합 $(s, d)$**를 공유한다.

- $R_1$은 $\gamma = s$, $\alpha = s+d$, $\beta = s-d$로 세 근의 구조를 $(s,d)$로 축약.
- $R_2$는 $h(x) = f(x)-x$의 임계값 대(對) 근의 값 관계를 통해 $(s,d)$에 방정식 부과.

따라서 **연동 가능**.

# [4] 수학적 상황 서술

## 핵심 연동 메커니즘

$R_1$을 $R_2$에 대입하면, 변환 $t = x-s$ 하에서 $h$의 감소형이 된다:

$$\tilde{h}(t) := g(t) - t = t^3 - (d^2+1)\,t$$

세 분기 방정식은:

| 분기 | 원래 식 | 감소형 ($\tilde{h}(t) = -r_i + s$를 정리) | **상수항** |
|---|---|---|---|
| (i) $h(x) = -\alpha$ | $f(x)-x = -(s+d)$ | $t^3 - (d^2+1)t = -d$ | $+d$ |
| (ii) $h(x) = -\beta$ | $f(x)-x = -(s-d)$ | $t^3 - (d^2+1)t = +d$ | $-d$ |
| (iii) $h(x) = -\gamma$ | $f(x)-x = -s$ | $t^3 - (d^2+1)t = 0$ | $0$ |

### 예기치 못한 소거 (구조적 발견)

> **평행이동 파라미터 $s$가 세 분기 방정식에서 완전히 소멸한다.**

이는 $R_1$($\gamma = (\alpha+\beta)/2$)이 강제한 대칭 때문이다: 세 근이 변곡점 기준으로 $\{s+d, s-d, s\}$ 구조를 가지면, $h(x)=-r_i$의 우변에서 $s$가 $-r_i + s$로 조합되어, 결과적으로 $\{-d, +d, 0\}$만 남는다.

### 근 개수 분석이 $d$ 하나의 방정식으로 환원

$\tilde{h}(t) = t^3 - (d^2+1)t$의 임계값은:

$$\tilde{h}_{\text{max/min}} = \pm\,\frac{2}{3\sqrt{3}}\,(d^2+1)^{3/2}$$

- **분기 (iii)**: $\tilde{h}(t) = 0 \Rightarrow t(t^2 - (d^2+1)) = 0$. $d^2+1 > 0$이므로 **항상 3개의 실근**.

- **분기 (i)와 (ii)**: 상수항이 $\pm d$로 **크기 같고 부호 반대**. 따라서 두 분기의 실근 개수는 **항상 동시에 같은 값**을 취한다 — 두 분기가 동시에 3개 → 3개이거나, 동시에 1개 → 1개이거나, 동시에 2개(접선) → 2개.

근 개수의 임계 경계: $|d| = \frac{2}{3\sqrt{3}}(d^2+1)^{3/2}$, 즉

$$R_2^*:\quad 27\,d^2 = 4\,(d^2+1)^3$$

이것은 $u = d^2$에 대한 **삼차 방정식** $4u^3 + 12u^2 - 15u + 4 = 0$으로 환원된다.

### 쌍방향 구속 구조

| 조건 | 단독 효과 | 미결정 파라미터 |
|---|---|---|
| $R_1$ (P004) 단독 | $\gamma = (α+β)/2$로 구조 축약, 감소형 도출 | $s$와 $d$ 모두 미결정 |
| $R_2$ (P001) 단독 | $h(x)$의 임계값-근값 관계식, 그러나 $s$가 남아 있어 4변수 문제 | $\alpha, \beta, \gamma$(또는 $s, d$ + 이차항 계수) 미결정 |
| $R_1 \wedge R_2$ 연립 | $s$가 소멸하여 $d$만의 방정식 $R_2^*$가 출현 → $d$의 이산적 후보 결정 | $s$만 미결정 (추가 조건 1개로 완결) |

**핵심**: $R_1$ 없이는 $R_2$에서 $s$가 소멸하지 않아 단일 변수 방정식으로 환원 불가능하고, $R_2$ 없이는 $R_1$이 $d$를 전혀 구속하지 못한다. **두 조건이 동시에 작동해야만** $d$가 삼차 방정식의 이산적 근으로 결정된다.

# [5] 수험생의 발견 지점

**발견 순간**: 수험생이 $f'(\alpha)=f'(\beta)$로부터 $\alpha+\beta=2\gamma$를 얻고 변환 $t=x-\gamma$를 적용한 뒤, 자기합성 분해 $f(x-f(x))=0$의 세 분기를 감소형으로 다시 쓰는 순간, **평행이동 파라미터 $s$($=\gamma$)가 세 분기 방정식의 우변에서 완전히 상쇄되어 사라진다**는 사실을 목격한다.

이 소거는 "두 근에서 도함수가 같다"는 조건이 "세 근이 변곡점 기준 등간격"이라는 배치를 강제한 결과이지만, 그 배치가 자기합성 분해에서 $s$를 소멸시킨다는 것은 사전에 예측하기 어렵다.

그 직후, 남은 분기 (i)과 (ii)의 상수항이 $+d$와 $-d$로 크기가 같고 부호만 반대라는 것을 발견하면서, **두 분기의 근 개수 전이(1개 ↔ 3개)가 반드시 동시에 발생**함을 깨닫고, 총 근의 개수가 $5 \to 7 \to 9$로 **2개씩이 아니라 4개씩 점프**하는 비자명한 이산 구조를 발견한다. 이 점프 구조가 $N$의 값에 따라 접선 임계 조건 $27d^2 = 4(d^2+1)^3$을 **유일하게** 강제하여, 겉으로는 무관해 보이던 "도함수 대칭"과 "자기합성 근 개수"가 하나의 대수방정식으로 수렴하는 순간이 풀이의 절정이 된다.

# [0] 이번 시도의 차별성 선언

- **조건 추가/변경**: 이전(양수 조건 포함 → 잉여) → 이번(양수 조건 완전 삭제, (가)+(나) 두 조건만으로 해 결정). 피드백 (1안) 적용 — $s$ 가 자유 파라미터로 남지만 답 $f'(\alpha)=2a^2$ 는 $s$ 에 무관하므로, 양수 조건 없이도 답이 유일하게 결정됨. 모든 전제가 "제거 시 답 변동"을 충족.
- **유일성 처리**: 추가 조건 불필요 — 파라미터 $a^2$ 를 결정하는 방정식 $(2u-1)^2(u+4)=0$ 이 양수 해를 단 하나($u=\tfrac{1}{2}$)만 가지므로 자연 유일성 확보.

# STEP 1. 파라미터 결정 가능성 검토

## 1-1. (가) 해석

$f(x)=(x-\alpha)(x-\beta)(x-\gamma)$ (모닉 삼차, 세 근 상이)로 놓으면

$$f'(\alpha)=(\alpha-\beta)(\alpha-\gamma),\qquad f'(\beta)=(\beta-\alpha)(\beta-\gamma).$$

$f'(\alpha)=f'(\beta)$에서

$$(\alpha-\beta)\bigl[(\alpha-\gamma)+(\beta-\gamma)\bigr]=0 \;\Longrightarrow\; \alpha+\beta=2\gamma.$$

변곡점 $s=\tfrac{\alpha+\beta+\gamma}{3}$ 를 쓰면 $\gamma=s$, $\alpha=s-a$, $\beta=s+a$ $(a\neq0)$.

치환 $t=x-s$ 로 **감소형**:

$$g(t)=t^3-a^2\,t, \qquad f(x)=g(x-s).$$

이 때 $f'(\alpha)=g'(-a)=3a^2-a^2=\boxed{2a^2}$. — 미지 파라미터 $a$ 하나.

## 1-2. (나) 해석

$f(x-f(x))=0 \;\Longleftrightarrow\; g\!\bigl(t-g(t)\bigr)=0$ ($t=x-s$, $s$ 소거됨!)

$g=0$의 세 근이 $0,\,\pm a$ 이므로 세 부분방정식으로 분해:

| 분기 | 방정식 ($t$ 기준) | 정리 |
|:---:|:---:|:---:|
| $t-g(t)=0$ | $t^3-(a^2+1)t=0$ | $t\bigl(t^2-(a^2+1)\bigr)=0$ → **항상 3근** |
| $t-g(t)=a$ | $t^3-(a^2+1)t-a=0$ | $\equiv h_{-}(t)$ |
| $t-g(t)=-a$ | $t^3-(a^2+1)t+a=0$ | $\equiv h_{+}(t)$ |

$h_{+}(t)=-h_{-}(-t)$ 이므로 두 방정식의 판별식이 동일:

$$\Delta = 4(a^2+1)^3 - 27a^2.$$

- $\Delta>0$: $h_{\pm}$ 각 3근 → 총 $3+3+3=9$ (겹침 없음 — 증명 아래)
- $\Delta=0$: $h_{\pm}$ 각 2근(이중근+단근) → 총 $2+2+3=\mathbf{7}$
- $\Delta<0$: $h_{\pm}$ 각 1근 → 총 $1+1+3=5$

> **겹침 불가 증명**: $h_{+}$와 $h_{-}$가 공통근 $t_0$를 가지면 $2a=0$, 모순. 분기 3의 근 $t=0,\,\pm\!\sqrt{a^2+1}$을 $h_{\pm}$에 대입하면 $\pm a\neq0$. ∎

따라서 **(나) "7개"** $\;\Longleftrightarrow\; 4(a^2+1)^3=27a^2$.

## 1-3. 파라미터 결정

$u=a^2>0$ 으로 놓으면

$$4(u+1)^3=27u \;\Longrightarrow\; 4u^3+12u^2-15u+4=0.$$

유리근 시도 $u=\tfrac{1}{2}$:

$$4\!\cdot\!\tfrac{1}{8}+12\!\cdot\!\tfrac{1}{4}-15\!\cdot\!\tfrac{1}{2}+4 = \tfrac{1}{2}+3-\tfrac{15}{2}+4 = 0.\;\checkmark$$

인수분해:

$$4u^3+12u^2-15u+4 = (2u-1)(2u^2+7u-4) = (2u-1)^2(u+4).$$

양수 해: $u=\tfrac{1}{2}$ (유일). 따라서 $a^2=\tfrac{1}{2}$.

**미결 파라미터 없음**: $s$는 자유이나 답 $f'(\alpha)=2a^2=1$은 $s$에 무관. ✅

$f'(\alpha) = 2a^2 = 2\cdot\dfrac{1}{2} = \mathbf{1}.$

### 수치 검증 ($s=0$ 대입)

$g(t)=t^3-\tfrac{1}{2}t$, 근: $0,\,\pm\tfrac{1}{\sqrt{2}}$.

**분기 3**: $t^3-\tfrac{3}{2}t=0$ → $t=0,\;\pm\!\sqrt{\tfrac{3}{2}}$ — **3근** ✓

**$h_{+}(t)=t^3-\tfrac{3}{2}t+\tfrac{1}{\sqrt{2}}$**: 이중근 $t=\tfrac{1}{\sqrt{2}}$ 확인:

$$h_{+}\!\!\left(\tfrac{1}{\sqrt{2}}\right)=\tfrac{1}{2\sqrt{2}}-\tfrac{3}{2\sqrt{2}}+\tfrac{1}{\sqrt{2}}=\tfrac{1-3+2}{2\sqrt{2}}=0.\;\checkmark$$

단근: $-\tfrac{2}{\sqrt{2}}=-\sqrt{2}$. 고유근 **2개** ✓

**$h_{-}(t)=t^3-\tfrac{3}{2}t-\tfrac{1}{\sqrt{2}}$**: 이중근 $-\tfrac{1}{\sqrt{2}}$, 단근 $\sqrt{2}$. 고유근 **2개** ✓

7개 고유근 전체:

$$\left\{0,\;\pm\!\sqrt{\tfrac{3}{2}},\;\tfrac{1}{\sqrt{2}},\;-\sqrt{2},\;-\tfrac{1}{\sqrt{2}},\;\sqrt{2}\right\}$$

모두 상이한지 확인: $\sqrt{3/2}\approx1.225$, $1/\sqrt{2}\approx0.707$, $\sqrt{2}\approx1.414$ — 모두 다름. ✅ 총 **7개** ✓
