## 프라블럼 메이커

### 문제 초안
> 함수 $f(x) = x^4 + bx^2 + c$ ($b,\;c$는 실수)에 대하여 $g(x) = \left|\,f(x)\,\right|$라 하자.
> 다음 조건을 만족시킨다.
>
> **(가)** 함수 $g(x)$가 미분가능하지 않은 점의 개수는 $2$이다.
>
> **(나)** 방정식 $f(x)=0$의 서로 다른 실근 중, 함수 $g$가 $x=\alpha$에서 미분가능한 실수 $\alpha$가 존재한다.
>
> **(다)** 함수 $g$의 극댓값은 $M$이다. $\quad$*(양의 상수 $M$은 추후 결정)*
>
> $f(x) = 0$의 서로 다른 실근 중 가장 큰 값을 $\gamma$라 할 때,
> $\left(\,f'(\gamma)\,\right)^{2}$ 의 값을 구하시오.

*(주: $M$의 구체적 수치는 다음 단계(수치 결정)에서 최종 답이 자연수가 되도록 조정한다.)*
*(참고: $c=0,\;b=-2\sqrt{M}$이 유도되면 $\gamma=\sqrt{-b}$이고, $(f'(\gamma))^2 = 32\,M^{3/2}$이다. 예컨대 $M=1$이면 답 $=32$, $M=4$이면 답 $=256$ 등.)*

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 역할 |
|---|---|---|
| **(가)** $g$의 미분불가능점 $= 2$개 | 사실 B — 절댓값 접힘(cusp)이 $x=\pm\sqrt{a}$의 2곳에서 발생, 즉 $f$의 교차근이 정확히 2개 | $b<0$ 확정 및 영점의 교차/접촉 분류에 대한 부분 제약 제공 |
| **(나)** $f(\alpha)=0$이면서 $g$가 $\alpha$에서 미분가능 | 사실 A+B 합류 — 접촉근 $x=0$의 부호 불변성 ($F(x)-\sqrt{a}=-x^2 \leq 0$) | (가)와 연립하여 $c=0$ 강제 (숨은 사실 $\sqrt{a}=a$의 대수적 번역) |
| **(다)** $g$의 극댓값 $= M$ | 사실 B — 반사 볼 꼭짓값 높이 $a$ | 구조 확정 후 스케일 파라미터 $b$ 고정 |
| **발문** $(f'(\gamma))^2$ | 교차근 $\gamma = \sqrt{-b}$에서의 접선 기울기 | 원본 A의 "합성방정식 근 개수" 발문과 완전 분리된 추출 방식 |
