## 아이디어 셀렉터

### 수학적 상황
### 수학적 객체
최고차항의 계수가 1이고 이차항의 계수가 0인 삼차함수 f(x) = x³ + px + q (p < 0이어서 극대·극소를 가짐, 세 실근 r₁ < r₂ < r₃ 존재)

### 제약 A가 부과하는 것
방정식 **f(x − f(x)) = 0**의 서로 다른 실근의 개수가 정확히 N개.

이 방정식은 f의 세 근 rᵢ에 대해 **f(x) = x − rᵢ** (i = 1, 2, 3)로 분해된다. 세 방정식은 모두 기울기 1인 평행한 직선 y = x − rᵢ와 삼차곡선 y = f(x)의 교점 문제이다. 평행하므로 서로 다른 분기의 해가 중복될 수 없다. 각 직선과 곡선의 교점 수(1, 2, 또는 3)의 합이 N이 되어야 하며, 교점 수는 직선의 수직 위치와 변환된 삼차식 g(x) = x³ + (p−1)x의 임계 간격 D = 2((1−p)/3)^{3/2}의 비교로 결정된다.

### 제약 B가 부과하는 것
f(x) = 0의 세 실근의 절댓값의 합 **|r₁| + |r₂| + |r₃| = S**.

이차항이 없으므로 r₁ + r₂ + r₃ = 0이 자동 성립. 이에 의해 절댓값합은 **양의 근들의 합의 정확히 2배**로 축약된다 (P003의 영합근 환원). 이 조건은 p, q에 대한 하나의 등식이지만, 근의 부호 분배가 확정되지 않으면 절댓값을 벗길 수 없어 단독으로는 p, q를 결정하지 못한다.

### 공유 파라미터
**p와 q** 모두가 A와 B에 공유됨.
- A에서: 임계 간격 D는 p에 의존, 직선 위치 q + rᵢ는 p와 q 모두에 의존
- B에서: 절댓값합은 근 rᵢ를 통해 p, q 모두에 의존
- 핵심 연결 고리: 세 직선의 위치값 (q+r₁, q+r₂, q+r₃)의 합이 3q로 고정되며, 이 값들의 분포가 A의 근의 개수를 결정하고, 동시에 rᵢ의 절댓값 구조가 B를 결정

### 강제 결과
A+B 동시 적용 시:
1. 세 평행선 중 **접선이 되는 분기가 유일하게 결정**됨 — 근의 합이 0이라는 대칭이 위치값의 분포를 제한하여, 최대 절댓값 근에 대응하는 직선만이 접선 조건을 만족할 수 있음
2. B의 절댓값합 조건이 근의 **부호 분배를 확정**하여 접선 조건의 절댓값을 벗겨냄
3. 결과적으로 **p가 S만의 닫힌 식으로 결정**되고, 이어서 q도 결정되어 f가 완전히 확정됨

**P001** (자기합성의 평행선 분해와 접선 강제), **P003** (영합근의 절댓값합 환원)

### 수험생의 발견 순간
**1. 수험생이 처음 시도하는 것:**
f(x − f(x)) = 0을 전개하려 하지만 9차식이 되어 직접 풀기 불가능함을 깨닫고, x − f(x) = rᵢ로 치환하여 **세 개의 삼차방정식 f(x) = x − rᵢ로 분해**한다. "아, 세 직선이 모두 기울기 1로 평행하구나"를 인식하고, 각 직선과 곡선의 교점 수를 따로 세는 전략을 세운다.

**2. 막히는 순간:**
각 직선 y = x − rᵢ와 곡선의 교점 수는 q + rᵢ의 값과 임계 간격 D의 대소에 의해 결정됨을 파악하지만, **rᵢ 자체가 p, q에 의존**하므로 q + rᵢ를 구체적으로 계산할 수 없다. 총 교점 수를 N으로 맞추는 조합(예: 3+2+2)은 알지만, 어떤 rᵢ가 접선 분기인지 결정할 **추가 정보가 부족**하여 진행이 멈춘다. "p와 q 사이의 관계는 하나 얻었는데, 미지수가 둘이라 하나가 남는다..."

**3. 다른 제약을 끌어와서 연립하는 과정:**
절댓값합 조건 |r₁|+|r₂|+|r₃| = S를 활용한다. r₁+r₂+r₃ = 0이므로 **S = 2·(양의 근들의 합)**으로 축약. 이것이 근의 크기에 대한 정보를 주므로, 세 직선의 위치값 q + rᵢ의 크기 순서를 결정할 수 있게 된다. 특히 "절댓값이 가장 큰 근"이 어떤 부호인지가 확정되고, 그에 대응하는 직선의 위치가 D와 정확히 일치하는 접선 조건으로 환원된다.

**4. 발견:**
"근의 합이 0이니까, 세 직선의 위치값 q+r₁, q+r₂, q+r₃도 합이 3q로 묶여 있고, 이 중 하나가 접선 조건을 만족하면 **나머지 둘의 위치가 자동으로 결정**되는 거구나! 그런데 절댓값합 조건이 어떤 근이 접선 분기인지를 **유일하게 선별**해주니까, 접선 등식에 이 정보를 넣으면 q가 소거되고 **p가 S만으로 딱 정해지네!**"

— 각 제약이 혼자서는 1차원 자유도를 남겼지만, "근의 합 = 0"이라는 공유 구속 위에서 두 조건이 만나는 순간 접선 분기의 선택과 부호가 동시에 확정되어 모든 파라미터가 결정된다.
