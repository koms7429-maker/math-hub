## [2] 각 아이디어가 만드는 계수 관계식 (기호 수준)

**함수 설정**: $f(x) = a(x-\alpha)(x-\beta)(x-\gamma)$, 세 실근 $\alpha, \beta, \gamma$ (서로 다름), 최고차 계수 $a \neq 0$.

**보조 함수**: $g(x) = f(x) - x$. 이는 $a$가 최고차 계수인 삼차식이며, 극대값 $M_g$와 극소값 $m_g$를 갖는다. $g$의 임계점은 $g'(x) = f'(x) - 1 = 0$의 두 근 $p, q$ ($p < q$)이다.

### 아이디어 P001이 만드는 관계식 R₁

$f(x - f(x)) = 0$은 $f$의 각 근 $r_i \in \{\alpha, \beta, \gamma\}$에 대해

$$f(x) = x - r_i \quad \Longleftrightarrow \quad g(x) = -r_i$$

로 분해된다. 세 방정식은 모두 **동일한 삼차함수 $g$의 수준집합**이며, 수준값만 $-\alpha, -\beta, -\gamma$로 다르다.

각 분기의 실근 개수 $n(r_i)$는 수준값 $-r_i$와 $g$의 임계값 $M_g, m_g$의 대소에 의해 결정된다:

$$n(r_i) = \begin{cases} 3 & \text{if } m_g < -r_i < M_g \\ 2 & \text{if } -r_i = M_g \text{ or } -r_i = m_g \\ 1 & \text{if } -r_i < m_g \text{ or } -r_i > M_g \end{cases}$$

총 근의 개수 $N$에 대한 조건:

$$\boxed{R_1: \quad N = n(\alpha) + n(\beta) + n(\gamma), \quad \text{단 각 } n(r_i) \text{는 } {-r_i} \text{와 } M_g(a,\alpha,\beta,\gamma),\; m_g(a,\alpha,\beta,\gamma) \text{의 대소에 의존}}$$

$R_1$은 $N$이 특정 값으로 고정될 때, 파라미터 $(a, \alpha, \beta, \gamma)$에 대한 **비선형 부등식 및 등식의 케이스 분류**를 생성한다. 그러나 $R_1$만으로는 어떤 근이 임계값과 정렬되는지, 파라미터 간 구체적 대수 관계가 무엇인지 결정되지 않는다 (다수의 케이스가 열려 있음).

### 아이디어 P019가 만드는 관계식 R₂

$f$의 근 $r_j$에서 접선을 차감하면:

$$f(x) - f'(r_j)(x - r_j) = f(x) - f'(r_j)\,x + f'(r_j)\,r_j$$

$f(r_j) = 0$이므로 $x = r_j$는 항상 단근이다. **$f'(r_j) = 1$**인 경우에 한해, 이 식은 $g(x) + r_j = f(x) - x + r_j$가 되어:

$$g(x) = -r_j \quad \text{가 } x = r_j \text{에서 이중근을 가짐}$$

이는 $g(r_j) = -r_j$ (자동 성립: $g(r_j) = f(r_j) - r_j = -r_j$) **그리고** $g'(r_j) = 0$, 즉 **$r_j$가 $g$의 임계점**이라는 것과 동치이다.

따라서:

$$\boxed{R_2: \quad f'(r_j) = 1 \quad \Longleftrightarrow \quad r_j \in \{p, q\} \text{ (}g\text{의 임계점)이며, } -r_j \text{는 }g\text{의 임계값}}$$

구체적으로, $f'(r_j) = a\prod_{k \neq j}(r_j - r_k) = 1$이라는 **$(a, \alpha, \beta, \gamma)$에 대한 대수 방정식**이 하나 생성된다.

$R_2$만으로는 $f'(r_j) = 1$이 하나의 방정식이므로, 4개 파라미터 $(a, \alpha, \beta, \gamma)$ 중 하나의 자유도만 소멸시키며, 어떤 근 $r_j$에서 이 조건이 성립하는지도 미정이다.

## [3] 파라미터 공유 확인

| 관계식 | 관여하는 파라미터 | 관여하는 핵심 객체 |
|---|---|---|
| $R_1$: $N = \sum n(r_i)$ | $a, \alpha, \beta, \gamma$ (임계값 $M_g, m_g$와 수준값 $-r_i$ 모두 이들에 의존) | $g$의 임계값 $M_g, m_g$ |
| $R_2$: $f'(r_j) = 1$ | $a, \alpha, \beta, \gamma$ ($f'(r_j) = a\prod_{k\neq j}(r_j - r_k)$) | $g$의 임계점 $p$ 또는 $q$ |

**공유 파라미터**: $(a, \alpha, \beta, \gamma)$ — 4개 파라미터 전부가 두 관계식에 공유된다.

**공유 객체**: 보조 함수 $g(x) = f(x) - x$. $R_1$은 $g$의 **임계값**이 수준값 $-r_i$와 어떤 대소관계인지를 다루고, $R_2$는 $g$의 **임계점**이 $f$의 근과 합치하는지를 다룬다. 두 조건은 $g$의 임계 구조를 서로 다른 측면(값 vs 위치)에서 구속한다.

→ **연동 가능** 확인.

## [4] 수학적 상황 서술

### 핵심 매개 객체: $g(x) = f(x) - x$의 임계 구조

$g$의 두 임계점 $p < q$는 $f'(x) = 1$의 근이다. $g$의 임계값은 $M_g = g(p)$, $m_g = g(q)$ (단, $a > 0$이면 $p$에서 극대, $q$에서 극소; $a < 0$이면 역순).

$f$의 근 $\alpha, \beta, \gamma$에서 $g(r_i) = -r_i$이므로, 세 수준값은 $-\alpha, -\beta, -\gamma$이다.

### R₂가 $g$의 임계 구조에 부과하는 조건

"$f$의 근 $r_j$에서 $f' = 1$"이 성립한다고 하자. 이는 $r_j$가 $g$의 임계점이라는 뜻이다.

**핵심**: $g$의 임계점이 $f$의 근과 합치하면, 해당 수준값 $-r_j$가 자동으로 $g$의 임계값이 된다:

$$-r_j = M_g \quad \text{또는} \quad -r_j = m_g$$

이 사실은 $R_1$의 대소 비교에 직접 개입한다: **해당 분기 $g(x) = -r_j$의 근 개수가 정확히 $n(r_j) = 2$로 고정**된다 (임계값에서의 수준집합은 이중근+단근).

### R₁이 R₂에 역으로 부과하는 조건

$N$이 특정 정수로 고정되면, $n(\alpha) + n(\beta) + n(\gamma) = N$에서 각 $n(r_i) \in \{1, 2, 3\}$의 가능한 조합이 이산적으로 제한된다.

**비자명한 강제 구조**: $n(r_j) = 2$는 $-r_j$가 임계값이라는 것인데, 이는 $R_2$의 조건과 동치이다. 따라서:

$$N \text{의 특정 값} \quad \Longrightarrow \quad \text{어떤 분기에서 } n = 2 \text{가 필수적} \quad \Longrightarrow \quad \text{해당 근에서 } f' = 1 \text{이 강제}$$

예를 들어 $N = 7$이면 가능한 조합은 $(3,3,1), (3,1,3), (1,3,3), (3,2,2), (2,3,2), (2,2,3)$ 등인데, 만약 **세 수준값 $-\alpha, -\beta, -\gamma$ 전부가 열린 구간 $(m_g, M_g)$ 안에 있으려면** 세 근 모두 특정 범위에 있어야 하고, 이것이 가능한지 여부가 파라미터 구속을 만든다. 만약 $(3,3,1)$만 살아남으면, $n(r_k) = 1$인 근 $r_k$는 $-r_k$가 임계값 바깥에 있어야 하고, 나머지 둘은 안에 있어야 한다. 반면 $(2,2,3)$이 살아남으면, **두 근에서 동시에 $f' = 1$이 강제**되어 $R_2$의 대수 방정식이 두 개 생성된다.

### 쌍방향 구속의 폭발적 결과

**방향 ①: R₂ → R₁에 대한 정보 주입**

$f'(r_j) = 1$이 하나의 근 $r_j$에서 성립하면:
- 해당 분기 $n(r_j) = 2$가 확정됨
- $-r_j$가 임계값이므로, **나머지 수준값 $-r_k$ ($k \neq j$)와 이 임계값의 대소가 구속**됨
- 예: $-r_j = M_g$이면, $-r_k < M_g$인지 $-r_k > M_g$인지가 나머지 분기의 근 개수를 결정

**방향 ②: R₁ → R₂에 대한 정보 역주입**

총 근 개수 $N$의 값이 특정 조합만을 허용하면:
- "$n = 2$인 분기의 개수"가 0개, 1개, 또는 2개로 확정됨
- 이에 따라 $f' = 1$이 성립하는 근의 개수가 결정됨
- **$f' = 1$인 근이 2개**이면: $f'(r_j) = 1$과 $f'(r_k) = 1$, 즉 $g$의 두 임계점 $p, q$ 각각이 $f$의 근과 합치하여 $r_j = p$, $r_k = q$. 이때 **$f$의 세 근 중 두 개가 $f'(x) = 1$의 두 근과 일치**하므로, 삼차함수 $f$와 이차함수 $f'$가 공유하는 근 관계가 매우 강한 대수적 구속을 만든다.

### 연립 시 드러나는 비선형 구조

$f$의 근 $\alpha, \gamma$ (중 두 개)가 $f'(x) = 1$의 근이기도 하다고 하면:

$$f'(\alpha) = 1, \quad f'(\gamma) = 1$$

$f'(x) = 1$은 이차방정식이므로 Vieta에 의해:

$$\alpha + \gamma = \frac{2(\alpha + \beta + \gamma)}{3}, \quad \alpha\gamma = \frac{\alpha\beta + \beta\gamma + \gamma\alpha - \frac{1}{a}}{3}$$

첫 번째 식에서 $\beta$를 $\alpha, \gamma$로 표현하는 관계가 나오고, 두 번째 식에서 $a$와 $\alpha, \beta, \gamma$의 추가 결합 방정식이 생성된다.

동시에, $n(\alpha) = n(\gamma) = 2$이므로 총 근 개수 $N = 4 + n(\beta)$이고, $n(\beta) \in \{1, 2, 3\}$을 결정하려면 수준값 $-\beta$와 임계값 $M_g = g(\alpha) = -\alpha$, $m_g = g(\gamma) = -\gamma$의 대소를 비교해야 한다:

$$n(\beta) = 3 \iff m_g < -\beta < M_g \iff -\gamma < -\beta < -\alpha \iff \alpha < \beta < \gamma$$

$$n(\beta) = 2 \iff -\beta = -\alpha \text{ 또는 } -\beta = -\gamma \quad (\text{불가: 근이 서로 다름})$$

$$n(\beta) = 1 \iff -\beta < -\gamma \text{ 또는 } -\beta > -\alpha \iff \beta > \gamma \text{ 또는 } \beta < \alpha$$

따라서 **$\beta$가 $\alpha$와 $\gamma$ 사이에 있는지 여부**가 $n(\beta)$를 이분적으로 결정하며, $N$의 값($N=7$ vs $N=5$)이 **$\beta$의 위치를 $\alpha$-$\gamma$ 사이로 강제하거나 바깥으로 배제**한다. 이것은 $R_1$과 $R_2$의 연립에서만 드러나는 비자명한 위치 구속이다.

최종적으로 Vieta 관계에서 나온 $\beta = \frac{\alpha + \gamma}{2}$ (또는 다른 결합식)과 위치 구속 $\alpha < \beta < \gamma$ (또는 그 역)이 동시에 성립해야 하므로, 파라미터 공간이 이산적으로 닫히며, 남은 자유도는 하나의 스칼라 등식(예: $a$와 근 간격의 관계)으로 소멸한다.

## [5] 수험생의 발견 지점

수험생은 다음 세 순간에 예상치 못한 구조를 발견한다:

1. **자기합성 분해에서 보조 함수 $g$의 등장 (P001의 번역)**: $f(x-f(x))=0$을 처리할 때, 각 분기 $f(x) = x - r_i$를 $g(x) = -r_i$로 재해석하는 순간 — "세 분기가 모두 **동일한 삼차함수 $g$의 수평선 교점 문제**"임을 인식하고, 문제가 $g$의 그래프와 세 수평선 $y = -\alpha, -\beta, -\gamma$의 교점 개수의 합으로 환원됨을 깨달을 때. 이 순간 문제의 구조가 "삼차함수와 수평선의 교점 개수" 문제로 급격히 단순화된다.

2. **"$n(r_j) = 2$이면 $f'(r_j) = 1$이다"라는 숨겨진 동치의 발견 (P001 ↔ P019 교차점)**: 특정 분기에서 근 개수가 정확히 2라는 조건을 처리할 때 — $g(x) = -r_j$가 이중근을 가지려면 수준값이 임계값과 일치해야 하고, 이는 $g'(r_j) = 0$, 즉 **$f'(r_j) = 1$과 동치**임을 발견하는 순간. 수험생은 "합성방정식의 근 개수 조건이 원래 함수의 특정 근에서의 **도함수값을 1로 고정**한다"는, 조건 간 전혀 다른 층위를 연결하는 다리를 발견한다. 이것이 P019의 접선 차감 구조와 정확히 일치하며, $(x-r_j)^2 \cdot q(x)$ 인수분해를 통해 나머지 단근의 위치까지 Vieta로 추적 가능해진다.

3. **근의 위치 구속의 역결정 (케이스 소거를 통한 폭발)**: 두 근에서 $f' = 1$이 성립한다고 확정된 후, Vieta 관계식이 나머지 근 $\beta$의 위치를 $\alpha, \gamma$로 표현해 주지만, $N$의 값이 $\beta$가 $(\alpha, \gamma)$ 사이에 있어야만 달성 가능한 값($N = 7$)인지 바깥이어야 달성 가능한 값($N = 5$)인지에 따라 **$\beta$의 배치가 이분적으로 강제**되는 순간. 수험생은 "합성방정식의 총 근 개수가 원래 함수의 **근의 순서 관계**를 역으로 결정한다"는 사실을 발견하며, 이 순서 관계가 Vieta 조건과 결합하여 모든 파라미터를 닫히게 만든다.

- **수학적 객체**: 삼차다항함수 (동일) — 그러나 이번에는 보조 함수 $g(x) = f(x) - x$가 핵심 매개 객체로 기능하며, 문제 표면에는 등장하지 않음
- **조건 형태**: 이전 "함수 형태 지정(x² 계수 부재) + 도함수값 등식 + $f(f(x))=0$ 근 개수" → 이번 **"세 실근 전제 + $f(x-f(x))=0$ 근 개수 + $f(x)=x$ 근 개수"의 3-조건 구조** (이유: 두 개수 조건이 모두 보조 함수 $g(x) = f(x) - x$를 경유하지만, $g$는 문제에 명시되지 않아 학생이 직접 구성해야 함. 이전 시도의 "도함수값 등식"처럼 미분 정보를 직접 주는 조건이 없음)
- **충돌 표현**: 이전 "대수적 항등식 충돌 — x² 부재와 도함수 등식의 연립이 기함수 구조를 폭발적으로 생성" → 이번 **"추론적 충돌 — 합성방정식의 근 개수가 $n(r_j)=2$인 분기를 강제하고, 이것이 $f'(r_j)=1$과 동치임을 학생이 스스로 발견해야 함. 조건 어디에도 도함수값 $1$이 언급되지 않음"** (이유: 충돌의 핵심이 "계수 합치"가 아니라 "보조 함수의 임계 구조 추론"에 있으므로, 발견 경로가 근본적으로 다름)
- **발문 유형**: 이전 수치 계산 $(f'(\alpha))^2$ → 이번 **수치 계산 $(\gamma - \alpha)^2$** (근의 배치 간격으로, 모든 자유 파라미터가 소거된 뒤 남는 불변량)

**차별성 확인**: 조건 형태 축(도함수 등식 → 이중 개수 조건, $f(f(x))$ → $f(x-f(x))$)과 충돌 표현 축(대수적 항등식 충돌 → 보조 함수 임계 구조 추론)이 모두 변경됨. ✓

## [무대 설정]

최고차항의 계수가 $1$인 삼차함수 $f(x)$와 그 세 실근 $\alpha < \beta < \gamma$를 무대로, 파라미터는 $(\alpha, \beta, \gamma)$. 보조 함수 $g(x) = f(x) - x$는 문제에 등장하지 않으며, 학생이 자기합성 방정식 분해 과정에서 스스로 구성해야 하는 숨겨진 매개 객체이다.

## [조건 배치 계획]

| 조건 | 담당하는 수학적 상황 요소 | 위장 효과 |
|---|---|---|
| 함수 정의: 최고차항 계수 $1$인 삼차함수 | $g(x) = f(x)-x$도 최고차항 계수 $1$인 삼차식임을 내재 | 학생은 $f$만 인식; $g$의 존재와 그 최고차 구조를 능동적으로 추출해야 함 |
| **(가)** $f(x)=0$이 서로 다른 세 실근 $\alpha<\beta<\gamma$를 가짐 | 삼중근·이중근 배제; $n(r_i)$ 분석에서 $n = 2$일 때 이중근이 근 자체와 합치하는 시나리오의 전제 | 무해한 전제 조건으로 보임 |
| **(나)** $f(x-f(x))=0$의 서로 다른 실근의 개수가 $7$ | $R_1$: 합성방정식의 분해 $\sum n(r_i) = 7$에서, **$(n(\alpha), n(\beta), n(\gamma)) = (2,3,2)$가 유일한 실현 가능 조합**이 되도록 강제. 이는 $f'(\alpha) = f'(\gamma) = 1$, $\beta = (\alpha+\gamma)/2$ (등간격 배치)를 생성 | "근 개수 7"이라는 정수 조건이 **도함수값 $= 1$과 근의 등간격 구조**를 결정한다는 것은 보조 함수 $g$의 임계 분석을 완료한 후에야 인식됨 |
| **(다)** $f(x)=x$의 서로 다른 실근의 개수가 $3$ | $g(x) = 0$이 세 실근을 가짐 $\Leftrightarrow$ $g$의 극댓값 $> 0 >$ 극솟값 $\Leftrightarrow$ $-\alpha > 0 > -\gamma$ $\Leftrightarrow$ $\alpha < 0 < \gamma$ | 학생은 "고정점이 3개"라는 독립적 조건으로 인식하지만, 실제로는 **(나)에서 구성한 $g$와 동일한 객체**의 부호 조건이며, $\alpha < 0 < \gamma$를 강제하여 $(3,3,1)$ 등 다른 케이스를 소거하는 필터로 작용 |

**충돌 배치 핵심**:

- **(나)** 단독: $\sum n(r_i) = 7$이라는 것만으로는 $(2,3,2)$, $(3,3,1)$, $(1,3,3)$ 등 다수의 조합이 열려 있다.
- **(다)** 단독: $f(x) = x$의 근이 3개라는 것은 $g$의 임계값의 부호 조건에 불과하며, 파라미터를 결정하지 않는다.
- **(나)+(다)** 연립: (다)가 $\alpha < 0 < \gamma$를 강제하면, 세 수준값 $-\alpha, -\beta, -\gamma$의 부호·대소 구조가 확정되어, (나)를 만족하는 유일한 조합이 $(2,3,2)$로 수렴한다. 이 순간 $f'(\alpha) = f'(\gamma) = 1$이라는 **문제 어디에도 언급되지 않은 도함수 조건**이 폭발적으로 드러나며, Vieta 관계를 통해 등간격 구조와 $d^2 = 1/2$이 결정된다.

## [문제 초안]

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** 방정식 $f(x) = 0$은 서로 다른 세 실근 $\alpha,\; \beta,\; \gamma\;$ $(\alpha < \beta < \gamma)$를 갖는다.
>
> **(나)** 방정식 $f(x - f(x)) = 0$의 서로 다른 실근의 개수는 $7$이다.
>
> **(다)** 방정식 $f(x) = x$의 서로 다른 실근의 개수는 $3$이다.
>
> $(\gamma - \alpha)^2$의 값을 구하시오.
