## 아이디어 셀렉터

### 수학적 상황
**수학적 객체**: 세 서로 다른 실근을 갖는 모닉 삼차다항함수 $f(x) = x^3 + ax^2 + bx + c$.

**사실 A가 부과하는 제약 (P001)**:
자기합성 방정식 $f(x - f(x)) = 0$의 근의 총 개수가 특정 값으로 주어진다. 이 방정식은 $f(x) = x - r_i$ ($i = 1, 2, 3$)인 세 분기로 분해되며, 모든 분기가 기울기 1인 평행선과의 교점 문제이므로 분기 간 해가 겹치지 않는다. 각 근 $r_i$는 자동으로 해이고, 분기에서 접선(이중근)이 되려면 $f'(r_i) = 1$이어야 한다. 총 근의 개수 제약이 **정확히 2개의 분기에서 접선이 발생할 것**을 강제한다.

**사실 B가 부과하는 제약 (P011)**:
삼차식의 세 근에서 평가한 도함수값의 합은 $f'(r_1) + f'(r_2) + f'(r_3) = a^2 - 3b$라는 항등식을 만족한다. 이는 근의 기본 대칭함수만으로 결정되는 고정된 값이다.

**숨은 사실 (A+B 동시 작용)**:
$f'(x) - 1$은 이차식이므로 $f'(r_i) = 1$을 만족하는 근은 최대 2개이다. 2개의 접선 분기를 강제하면, 접선 조건을 갖는 두 근 $r_1, r_2$는 이차방정식 $f'(x) = 1$의 근이 되어 $r_1 + r_2 = -2a/3$을 만족한다. 비에타 공식 $r_1 + r_2 + r_3 = -a$와 결합하면 $r_3 = -a/3$, 이는 정확히 $f$의 **변곡점**의 $x$-좌표이다. 따라서 삼차함수의 변곡점이 반드시 $x$-축 위에 놓이며, 이로 인해 삼차함수의 점대칭 중심이 근이 되는 구조가 강제되어 계수 $c$가 $a, b$의 함수로 확정된다.

---

### 수험생의 발견 순간
1. **수험생이 처음 시도하는 것**: $f(x - f(x)) = 0$을 전개하려 하지만 9차식이 되어 막막하다. $f(t) = 0$의 근 $r_i$를 이용해 $f(x) = x - r_i$ 세 분기로 분해하고, 각 분기의 근의 개수를 세기 시작한다. 각 $r_i$가 자동 해임을 확인하고, 나머지는 이차방정식의 판별식으로 분류한다.

2. **막히는 순간**: 총 근의 개수 조건을 맞추려면 "일부 분기에서 접선(이중근)이 필요하다"는 것까지는 도달하지만, $f'(r_i) = 1$ 조건이 최대 몇 개의 근에서 성립할 수 있는지, 그리고 접선이 되지 않는 나머지 근에서 어떤 일이 일어나는지를 연결하지 못한다.

3. **발견**: "$f'(x) - 1$은 이차식이니까 $f'(r_i) = 1$인 근은 최대 2개밖에 안 되잖아!" — 이 순간, 접선 조건이 2개인 두 근의 합을 이차방정식의 근과 계수 관계로 구하고, 비에타 공식과 합산하면 **세 번째 근이 $-a/3$, 즉 변곡점의 위치로 강제됨**을 발견한다. 변곡점이 근이라는 기하적 조건이 $c$를 결정하여 문제가 풀린다.

4. **숨은 사실과의 대응**: 이 발견은 STEP 2-2의 "접선 분기의 최대화 → 나머지 근이 변곡점에 고정"이라는 숨은 사실과 정확히 대응한다. 수험생은 "자기합성의 분기 구조"(조합론적)와 "도함수의 차수 제한"(대수적)이라는 두 독립적 제약이 만나 "변곡점이 근"이라는 기하적 결론을 만들어내는 순간을 경험한다.
