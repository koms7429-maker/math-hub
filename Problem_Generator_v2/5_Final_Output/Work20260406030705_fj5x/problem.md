## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
- **수학적 객체:** 최고차항의 계수가 1이고 $f(0) = 0$인 삼차함수 $f(x) = x^3 + ax^2 + bx$와 그 적분함수 $H(x) = \int_0^x f(t)\,dt$.

- **제약 A가 부과하는 것:** $H(x) \ge 0$이 모든 $x > 0$에서 성립한다. (원점부터의 누적 적분이 항상 비음수.) 이는 $H(x) = \frac{x^2}{4}\cdot h(x)$에서 이차식 $h(x)$가 양의 실수 전체에서 비음수임을 요구하며, 계수 $(a, b)$가 포물선 $b = \frac{2a^2}{9}$ 위 또는 위쪽의 2차원 영역에 놓이도록 한다. 그러나 $(a, b)$의 구체적 값은 미결.

- **제약 B가 부과하는 것:** $H(\alpha) = 0$이 특정 양수 $\alpha$에서 성립한다. (원점부터 $\alpha$까지의 정적분이 0.) 이는 $(a, b)$ 사이에 하나의 선형 관계식을 부여하여 자유도를 1로 줄이지만, $a$의 값은 미결.

- **공유 파라미터:** 이차계수 $a$와 일차계수 $b$. A는 이 둘에 대해 **부등식**(영역 조건)을, B는 **등식**(직선 조건)을 각각 부과한다. B의 등식이 A의 부등식에 대입될 때, $a$에 대한 완전제곱 부등식 $(2a + 9)^2 \le 0$이 출현하여, 유일한 해 $a = -9/2$가 강제된다.

- **강제 결과:** 부등식과 등식의 교차에서 이차식 $h(x)$의 판별식이 정확히 0이 되어, 적분함수가 완전제곱 $H(x) = \left[\frac{x(x-3)}{2}\right]^2$으로 결정된다. $f(x) = x^3 - \frac{9}{2}x^2 + \frac{9}{2}x = x(x - \frac{3}{2})(x-3)$.

- **영감 source_pid:** **P022** (비음수 분해 영점 고정 — 비음수 함수가 0이 되면 각 성분이 독립적으로 0이 되어 미지수 결정), **P016** (부호분기 적분의 해집합 유한화 — 적분의 비음수성과 영점 조건의 결합이 구조를 강제)

### 수험생의 발견 순간
1. **처음 시도:** 제약 B를 먼저 전개한다. $H(3) = 0$을 계산하여 $b = -\frac{9}{2} - 2a$라는 관계식을 얻는다. "$b$는 $a$로 표현되지만, $a$를 결정할 조건이 없다…" 하고 막힌다.

2. **막히는 순간:** $a$가 자유 파라미터로 남아 있어 $f$가 유일하게 결정되지 않는다. 제약 B만으로는 삼차함수의 무한한 1-파라미터 족이 존재한다.

3. **제약 A를 끌어옴:** "$H(x) \ge 0$이 $x > 0$ 전체에서 성립해야 한다"를 떠올리고, $b = -\frac{9}{2} - 2a$를 $h(x) \ge 0$ 조건에 대입한다. 정리하면 $-\frac{9}{2} - 2a \ge \frac{2a^2}{9}$이 되고, 양변을 정리하면 $(2a+9)^2 \le 0$이 나온다.

4. **발견:** "완전제곱식이 $\le 0$이라니, 이건 $= 0$일 수밖에 없다!" 부등식 조건(A)과 등식 조건(B)이 **서로 반대 방향에서 $b$를 조이고** 있었기 때문에 — A는 $b \ge \frac{2a^2}{9}$ (아래에서), B는 $b = -\frac{9}{2} - 2a$ (위에서) — 이 두 조건이 양립하는 유일한 점이 포물선과 직선의 **접점**이며, 그 접점에서 $a = -\frac{9}{2}$, $b = \frac{9}{2}$가 결정된다. 적분함수 $H(x)$가 완전제곱 다항식이 되는 것은 "비음수이면서 동시에 영점을 갖는" 조건의 필연적 귀결이다.

# STEP 1. 구조 분석

## 1-1. 조건이 만드는 방정식 체계

$f(x) = x^3 + ax^2 + bx$ (최고차항 계수 $1$, $f(0) = 0$)

$$g(x) = \int_0^x f(t)\,dt = \frac{x^4}{4} + \frac{ax^3}{3} + \frac{bx^2}{2} = \frac{x^2}{12}\bigl(3x^2 + 4ax + 6b\bigr)$$

$Q(x) = 3x^2 + 4ax + 6b$로 놓으면

$$g(x) = \frac{x^2}{12}\,Q(x)$$

**(가)** $g(x) \ge 0\;(\forall\, x > 0)$

$x > 0$일 때 $\dfrac{x^2}{12} > 0$이므로 **(가) ⟺ $Q(x) \ge 0\;(\forall\, x > 0)$.**

$Q$는 이차계수 $3 > 0$인 위볼록 포물선이므로, $Q$가 양의 실수 영역에서 음수가 되려면 **양의 근**이 존재해야 함. 따라서

$$\text{(가)} \;\Longleftrightarrow\; Q\text{가 양의 근을 갖지 않음.} \tag{★}$$

$Q$의 꼭짓점: $x_v = -\dfrac{2a}{3}$, 판별식 $\Delta_Q = 16a^2 - 72b$.

- **경우 I** ($a \ge 0$): $x_v \le 0$이므로 $Q$는 $x > 0$에서 단조증가. $Q(0) = 6b \ge 0$이면 충분. 즉 **$b \ge 0$**.
- **경우 II** ($a < 0$): $x_v > 0$이므로 최솟값 $Q(x_v) = 6b - \dfrac{4a^2}{3} \ge 0$, 즉 **$b \ge \dfrac{2a^2}{9}$**.

**(나)** $g(3) = 0$

$$\frac{9}{12}\bigl(27 + 12a + 6b\bigr) = 0 \;\Longrightarrow\; 9 + 4a + 2b = 0 \;\Longrightarrow\; b = -\frac{9 + 4a}{2} \tag{L}$$

## 1-2. 분기 분석

**(나)의 직선 (L)을 각 경우에 대입)**

| 경우 | (가) 조건 | (L) 대입 결과 |
|---|---|---|
| I: $a \ge 0$ | $b \ge 0$ | $b = -\frac{9+4a}{2} < 0$ → **모순** (탈락) |
| II: $a < 0$ | $b \ge \frac{2a^2}{9}$ | $-\frac{9+4a}{2} \ge \frac{2a^2}{9}$ → 아래 전개 |

경우 II 전개:

$$-\frac{9+4a}{2} \ge \frac{2a^2}{9} \;\Longrightarrow\; -9(9+4a) \ge 4a^2 \;\Longrightarrow\; 4a^2 + 36a + 81 \le 0$$

$$\Longrightarrow\; (2a + 9)^2 \le 0$$

$(2a+9)^2 \ge 0$이므로 **등호만 성립**: $\;a = -\dfrac{9}{2}$.

### 판정 1-2

분기 I 탈락, 분기 II에서 유일해 $a = -\tfrac{9}{2}$ 확정. **CONTINUE.**

## 2-1. 유일해의 수치 확인

$$a = -\frac{9}{2}, \quad b = -\frac{9 + 4(-9/2)}{2} = -\frac{9-18}{2} = \frac{9}{2}$$

$$f(x) = x^3 - \frac{9}{2}x^2 + \frac{9}{2}x = x\!\left(x^2 - \frac{9}{2}x + \frac{9}{2}\right)$$

근 분해: $x^2 - \dfrac{9}{2}x + \dfrac{9}{2} = 0$의 근 $= \dfrac{9/2 \pm 3/2}{2} = 3,\;\dfrac{3}{2}$

$$\boxed{f(x) = x\!\left(x - \frac{3}{2}\right)\!(x - 3)}$$
