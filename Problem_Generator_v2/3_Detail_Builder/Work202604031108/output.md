# STEP 1 — 구조 분석 (기호만 사용)

## 1-1. 조건이 만드는 방정식 체계

**무대.** $f(x)=x^4+bx^2+c$ ($b,c$는 실수), $g(x)=\lvert f(x)\rvert$.

$u=x^{2}$로 치환하면 $h(u)=u^{2}+bu+c$. $h=0$의 두 근을 $u_{1},u_{2}$라 하자.

| $f(x)=0$의 실근 구조 | $g$의 미분불가능점 개수 | 접촉근 존재 여부 |
|---|---|---|
| $u_1>0,\;u_2>0$ (이근) | 4 | 없음 |
| $u_1>0,\;u_2=0$ ($c=0,\;b<0$) | 2 ($\pm\sqrt{u_1}$) | $x=0$ (이중근) |
| $u_1>0,\;u_2<0$ ($c<0$) | 2 ($\pm\sqrt{u_1}$) | 없음 |
| $u_1=u_2>0$ ($c=b^2/4$) | 0 (전부 접촉) | $\pm\sqrt{u_1}$ |
| $u_1=0,\;u_2<0$ 등 기타 | 0 | $x=0$ |

## 1-2. 분기 분석

**(가) 단독** ("$g$의 미분불가능점 $=2$"):
- **Case A**: $c=0,\;b<0$ → 교차근 $\pm\sqrt{|b|}$ 2개, 접촉근 $0$ 1개
- **Case B**: $c<0$ (임의) → 교차근 $\pm\sqrt{u_1}$ 2개, 접촉근 없음

→ (가) 단독으로는 $c$의 부호만 제한 ($c\le 0$), $c=0$ 미확정.

**(나) 단독** ("$f(\alpha)=0$이고 $g$가 $\alpha$에서 미분가능인 $\alpha$ 존재"):
- $c=0$ (접촉근 $x=0$) 또는 $c=b^{2}/4$ (완전제곱, 모든 근이 접촉) 등

→ (나) 단독으로는 $c=0$과 $c=b^{2}/4$가 공존.

**(가)+(나) 연립**:
- Case B ($c<0$): 교차근만 존재, 접촉근 없음 → (나) 불성립. **탈락**.
- $c=b^{2}/4$ ($b<0$): $f=(x^{2}+b/2)^{2}\ge 0$, 미분불가능점 0개 → (가) 불성립. **탈락**.
- **유일 생존**: $c=0,\;b<0$.

### 판정 1-2
연립 후 **유일 구조**: $f(x)=x^{2}(x^{2}+b),\;b<0$. ✅ CONTINUE.

## 1-3. 자유도 계산

| 미지수 | 조건 | 남은 자유도 |
|---|---|---|
| $b,c$ (2개) | (가)+(나) → $c=0$ (1식 소비, $b<0$ 부등식은 별도) | 1 ($b$) |
| $b$ (1개) | (다) $g$의 극댓값 $=M$ → $b^{2}/4=M$ → $b=-2\sqrt{M}$ | 0 |

### 판정 1-3
자유도 0, 유일해. ✅ CONTINUE.

## 1-4. 탐색 전략 수립

$b=-2\sqrt{M}$이므로 $\gamma=\sqrt{|b|}=\sqrt{2\sqrt{M}}$, $f'(\gamma)=2|b|^{3/2}$, $(f'(\gamma))^{2}=4|b|^{3}=32M^{3/2}$.

정수성 조건: $32M^{3/2}\in\mathbb{Z}$ → $M$이 완전제곱수일 때 자연스러움.

| $M$ | $b$ | $\gamma$ | $(f'(\gamma))^{2}$ |
|---|---|---|---|
| 1 | $-2$ | $\sqrt{2}$ | 32 |
| **4** | **$-4$** | **$2$** | **256** |
| 9 | $-6$ | $\sqrt{6}$ | 864 |

✅ CONTINUE. 후보 3개 확보, Phase 2로 이동.

## 2-2. 채택: $M=4$

$$f(x)=x^{4}-4x^{2},\quad g(x)=\lvert x^{4}-4x^{2}\rvert$$

**검증:**

**(가)** $f(x)=x^{2}(x-2)(x+2)$. 교차근 $\pm 2$, 접촉근 $0$.
$g$의 미분불가능점: $x=\pm 2$ (부호 전환) → **2개**. ✅

**(나)** $f(0)=0$이고, $g(x)=x^{2}(4-x^{2})$ (근방에서 $\ge 0$), $g'(0)=0$. $g$는 $x=0$에서 미분가능. $\alpha=0$ 존재. ✅

**(다)** $g$의 극대: $x=\pm\sqrt{2}$에서 $g(\pm\sqrt{2})=\lvert 4-8\rvert=4$. 극댓값 $=4$. ✅

$\gamma=2$, $f'(2)=4\cdot 8-8\cdot 2=32-16=16$, $(f'(2))^{2}=256$. ✅

**유일성:** (가)+(나) → $c=0,b<0$, (다) → $b=-4$. 유일. ✅

**손계산:** 모든 계산이 정수 범위. ✅

## 완성된 문제

> 함수 $f(x) = x^{4} + bx^{2} + c\;$ ($b,\;c$는 실수)에 대하여 $g(x) = \left|\,f(x)\,\right|$라 하자.
> 다음 조건을 만족시킨다.
>
> **(가)** 함수 $g(x)$가 미분가능하지 않은 점의 개수는 $2$이다.
>
> **(나)** $f(\alpha) = 0$이고 함수 $g$가 $x = \alpha$에서 미분가능인 실수 $\alpha$가 존재한다.
>
> **(다)** 함수 $g$의 극댓값은 $4$이다.
>
> $f(x) = 0$의 모든 실근 중 가장 큰 값을 $\gamma$라 할 때,
> $\left(\,f'(\gamma)\,\right)^{2}$ 의 값을 구하시오.

$$\boxed{256}$$
