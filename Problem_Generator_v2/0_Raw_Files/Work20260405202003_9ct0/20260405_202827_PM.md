## [0] 이번 시도의 차별성 선언

| 축 | 이번 선택 | 이유 |
|---|---|---|
| **수학적 객체** | 삼차 다항함수 (변곡점 위치를 미분 조건으로 제시) | 원본 상황의 핵심인 "이차항 소멸"을 f″(0)=0이라는 무해한 미분 조건으로 위장 |
| **조건 형태** | (가) 이계도함수 등식, (나) 구간 최댓값 비교, (다) 곱-적분 등식 — **세 가지 서로 다른 유형 혼합** | 각 조건이 이질적 도구(미분, 극값 분류, 적분)를 요구하여 상호 연결을 은닉 |
| **발문 유형** | 구체적 함수값 계산 (2f(2)) | f의 완전 결정 없이는 답을 낼 수 없어, 두 제약의 연립을 반드시 수행하게 강제 |
| **숨은 사실의 표현** | 간접적 — "기함수"라는 단어·대칭 언급 일절 없음 | (나)+(다)를 연립해야만 q=0(기함수)이 드러나도록 설계 |

*(첫 번째 시도이므로 이전 시도와의 비교 대상 없음)*

## [무대 설정]

최고차항의 계수가 1인 삼차함수 f(x) — 파라미터: 이차항 계수 a, 일차항 계수 p, 상수항 q.

---

## [조건 배치 계획]

| 수학적 상황 요소 | 번역 대상 조건 | 설계 의도 |
|---|---|---|
| f(x) = x³ + px + q (이차항 없음) | **(가)** f″(0) = 0 | f″(x) = 6x + 2a이므로 a = 0을 강제. "이차항이 없다"를 명시하지 않고 미분 조건으로 우회 |
| 구간 최댓값 대칭 (제약 A) | **(나)** 구간 [-1, 0]에서의 최댓값 = 구간 [0, 1]에서의 최댓값 | 극점·끝점 경우 분류를 유도. p = −3/4를 결정하지만 **q는 등장하지 않아** 미결 상태를 남김 |
| 곱미분 적분 소멸 (제약 B) | **(다)** ∫₋₁¹ f(x)f′(x) dx = 0 | 표면상 복잡한 적분으로 보이나, (f²)′/2 인식 시 끝점 평가로 축약. **q = 0 ∨ p = −1** 분기를 생성 |
| 숨은 사실: q = 0 (기함수) | **조건에 직접 없음** | (나)의 p = −3/4가 (다)의 p = −1 분기를 소멸시켜, q = 0만 생존. 학생이 직접 발견해야 함 |

**은닉 구조 점검**: 
- (가) 단독 → f(x) = x³ + px + q 형태만 확정. 무해.
- (나) 단독 → p를 결정하지만 q 자유. 불완전.  
- (다) 단독 → f(1)² = f(−1)²라는 분기만 생성. 불완전.  
- **(나) + (다) 연립 → p = −3/4가 "p = −1" 분기를 죽이고 q = 0을 강제** → 비로소 f가 유일 결정.

## [문제 초안]

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** $f''(0) = 0$
>
> **(나)** 함수 $f(x)$의 구간 $[-1,\, 0]$에서의 최댓값과 구간 $[0,\, 1]$에서의 최댓값이 같다.
>
> **(다)** $\displaystyle \int_{-1}^{1} f(x)\,f'(x)\, dx = 0$
>
> $2f(2)$의 값을 구하시오. $\quad$ [4점]

## [조건-상황 대응표]

| 조건 | 수학적 상황 요소 | 역할 |
|---|---|---|
| **(가)** f″(0) = 0 | 이차항 소멸 (변곡점이 y축 위) | f(x) = x³ + px + q 형태를 강제 |
| **(나)** 구간 최댓값 동치 | 제약 A — 구간 최댓값 대칭 | 극점 위치 분석을 통해 p = −3/4 결정, **q에 무관** |
| **(다)** ∫f·f′ = 0 | 제약 B — 곱미분 적분 소멸 | (f²)′/2 역이용 → f(1)² = f(−1)² → "q = 0 ∨ p = −1" 분기 생성 |
| *발문* 2f(2) | 강제 결과 — f의 완전 결정 확인 | (나)+(다) 연립으로 p = −1 분기 소멸 → q = 0 → f(x) = x³ − (3/4)x 유일 결정 필요 |
