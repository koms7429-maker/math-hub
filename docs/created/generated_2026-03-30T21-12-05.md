# STEP 1. 구조 분석 (기호만 사용)

## 1-1. 조건이 만드는 방정식 체계

**전문:** $f(x) = x^3 + bx^2 + cx + d$, 최고차항 계수 $1$.

**(가):** $f(0)=0$이고 세 실근이 모두 $\ge 0$, 서로 다름.
$$f(x) = x(x-\alpha)(x-\beta),\quad \alpha > 0,\;\beta > 0,\;\alpha \neq \beta$$
전개: $f(x) = x^3 - (\alpha+\beta)x^2 + \alpha\beta\,x$. 
자유 파라미터: $s = \alpha+\beta$, $p = \alpha\beta$.  ($s > 0,\;p > 0$)

**(나):** $f(x - f(x)) = 0$의 서로 다른 실근 개수 = $8$.

핵심 함수:
$$g(x) = x - f(x) = -x^3 + sx^2 + (1-p)x$$

$f(x-f(x))=0 \;\Longleftrightarrow\; g(x) \in \{0,\;\alpha,\;\beta\}$이므로 세 분기:
- **분기 $r=0$:** $g(x)=0$
- **분기 $r=\alpha$:** $g(x)=\alpha$
- **분기 $r=\beta$:** $g(x)=\beta$

**각 분기의 인수분해:**

$g(r_i) = r_i - f(r_i) = r_i$ ($f$의 근이므로 $f(r_i)=0$). 따라서 $x = r_i$는 분기 $g(x) = r_i$의 확정 근.

*분기 $r=0$:*
$$g(x) = 0 \;\Longrightarrow\; x\bigl[-x^2 + sx + (1-p)\bigr] = 0$$
나머지 이차: $x^2 - sx - (1-p) = 0$, 판별식 $= (s)^2 + 4(1-p) = (\alpha-\beta)^2 + 4 > 0$. 
→ **항상 3개 실근.**  
$x=0$이 이차식의 근이 되는 조건: $-(1-p)=0$, 즉 **$p=1$** $\Rightarrow$ 중근 발생, 서로 다른 근 **2개**.

*분기 $r=\alpha$:*
$$g(x)-\alpha = -(x-\alpha)(x^2 - \beta x - 1)$$
이차 $x^2 - \beta x - 1 = 0$의 판별식 $= \beta^2 + 4 > 0$.
→ **항상 3개 실근.**
$x=\alpha$가 이차식의 근 $\Leftrightarrow$ $\alpha^2 - \beta\alpha - 1 = 0$, 즉 **$p = \alpha^2 - 1$** $\Rightarrow$ 중근, 서로 다른 근 **2개**.

*분기 $r=\beta$:*
$$g(x)-\beta = -(x-\beta)(x^2 - \alpha x - 1)$$
마찬가지로 **항상 3개 실근.** 중근 조건: **$p = \beta^2 - 1$**.

**서로 다른 분기 간 공유 근:**
$g(x_0) = r_i$이고 $g(x_0) = r_j$ $\Rightarrow$ $r_i = r_j$, 모순. → **불가능.**

따라서 총 서로 다른 실근 수 $= 9 - (\text{분기 내 중근 수})$.

## 1-2. 분기 분석

8개 $= 9 - 1$이므로 정확히 **1개 분기에서 중근** 발생.

| 접선 조건 | 관계식 | 남은 분기 가능 여부 | 비고 |
|---|---|---|---|
| (iii) $p = 1$ | 분기 $0$에서 중근 | (i) $p=\alpha^2-1$ 불성립, (ii) $p=\beta^2-1$ 불성립 필요 | $\alpha^2 \neq 2,\;\beta^2 \neq 2$ |
| (i) $p = \alpha^2 - 1$ | 분기 $\alpha$에서 중근 | (iii) $p\neq 1$, (ii) $p\neq\beta^2-1$ 필요 | $\alpha \neq \pm\beta$ |
| (ii) $p = \beta^2 - 1$ | 분기 $\beta$에서 중근 | (iii) $p\neq 1$, (i) $p\neq\alpha^2-1$ 필요 | $\beta \neq \pm\alpha$ |

세 경우 모두 1개 방정식 → 자유도 $2-1=1$.

**(다):** $f''(t)=0$ $\Rightarrow$ $t = \dfrac{s}{3}$, $\;f(t) = \dfrac{s(9p - 2s^2)}{27} = m$.

→ 추가 방정식 1개 → 자유도 $1-1=0$.

## 1-3. 자유도 계산

| | 개수 |
|---|---|
| 자유 파라미터 ($\alpha,\beta$) | 2 |
| (가)에서의 방정식 | 0 (범위 조건만) |
| (나)에서의 방정식 (접선 조건 1개) | 1 |
| (다)에서의 방정식 | 1 |
| **남은 자유도** | **0** |

## 1-4. 탐색 전략

**접선 조건 (iii): $p = 1$ 채택** (가장 깔끔한 인수분해를 유도)

$p = 1$이면 $g(x) = -x^3 + sx^2 = x^2(s-x)$.

$h(x) = x^2(s-x)$의 극대값: $h'(x) = x(2s-3x) = 0$에서 $x = \frac{2s}{3}$, $h\!\left(\frac{2s}{3}\right) = \frac{4s^3}{27}$.

분기 $\alpha,\;\beta$가 각각 3개 근 $\Leftrightarrow$ $0 < \alpha < \frac{4s^3}{27}$ 및 $0 < \beta < \frac{4s^3}{27}$. ($\alpha,\beta > 0$ 이미 보장)

**(다):** $f(t) = \frac{s(9 \cdot 1 - 2s^2)}{27} = \frac{s(9 - 2s^2)}{27} = m$.

$m$을 지정하면 $s$가 결정. $s$에 대한 방정식: $2s^3 - 9s + 27m = 0$.

**탐색 순서:** $m$을 정수 $\{-1, -2, 1, 2\}$ 순서로 시도, $s > 0$ 정수해 우선 탐색.

이어서 분기 $\alpha,\;\beta$에 대해 $(i), (ii)$ 불성립 확인, 그리고 다른 접선 조건 경우에서 $\alpha,\beta > 0$ 해가 없음을 확인하여 유일성 보장.

## 시도 1: $m = -1$

**방정식:** $2s^3 - 9s + 27(-1) = 0 \;\Longrightarrow\; 2s^3 - 9s - 27 = 0$.

$s = 3$: $2(27) - 9(3) - 27 = 54 - 27 - 27 = 0$. ✅

인수분해: $2s^3 - 9s - 27 = (s-3)(2s^2 + 6s + 9)$.
$2s^2 + 6s + 9$의 판별식 $= 36 - 72 = -36 < 0$ → $s = 3$이 유일한 실수 해.

**$s = 3,\; p = 1$:**
$$f(x) = x^3 - 3x^2 + x = x(x^2 - 3x + 1)$$

근: $\alpha,\beta = \dfrac{3 \pm \sqrt{5}}{2}$. 
$\alpha = \dfrac{3+\sqrt{5}}{2} \approx 2.618 > 0$, $\;\beta = \dfrac{3-\sqrt{5}}{2} \approx 0.382 > 0$, $\;\alpha \neq \beta$. ✅

**(나) 검증: 총 서로 다른 실근 = 8?**

$g(x) = x^2(3-x)$. 극솟값 $g(0)=0$, 극대값 $g(2) = 4$.

| 분기 | $r$ 값 | $r$과 극값 관계 | 서로 다른 실근 수 |
|---|---|---|---|
| $r=0$ | $0$ | $r = $ 극솟값 (접선) | $x=0$ (중근), $x=3$ → **2개** |
| $r=\alpha$ | $\approx 2.618$ | $0 < 2.618 < 4$ | **3개** |
| $r=\beta$ | $\approx 0.382$ | $0 < 0.382 < 4$ | **3개** |

분기 간 공유 근 없음 (위 증명). **총 $2+3+3 = 8$개.** ✅

**추가 접선 조건 비활성 확인:**
- (i): $p = \alpha^2 - 1$? $1 = \frac{(3+\sqrt5)^2}{4} - 1 = \frac{14+6\sqrt5}{4} - 1 = \frac{10+6\sqrt5}{4} \approx 5.85 \neq 1$. ✅ 비활성
- (ii): $p = \beta^2 - 1$? $1 = \frac{(3-\sqrt5)^2}{4} - 1 = \frac{14-6\sqrt5}{4} - 1 = \frac{10-6\sqrt5}{4} \approx -0.85 \neq 1$. ✅ 비활성

**(다) 검증:**
$$f''(x) = 6x - 6,\quad f''(t)=0 \Rightarrow t = 1$$
$$f(1) = 1 - 3 + 1 = -1 = m \quad \checkmark$$

**유일성 검증 (다른 접선 조건 경우):**

*접선 조건 (i): $p = \alpha^2 - 1$, (다): $f(t) = -1$.*

$\beta = \alpha - 1/\alpha$, $s = 2\alpha - 1/\alpha$. (다) 조건:
$$\frac{s\bigl(9(\alpha^2-1)-2s^2\bigr)}{27} = -1$$
정리하면 $(2\alpha^2-1)(\alpha^4-\alpha^2-2) + 27\alpha^3 = 0$, 즉
$$2\alpha^6 - 3\alpha^4 + 27\alpha^3 - 3\alpha^2 + 2 = 0$$

$\alpha > 0,\;\beta > 0$ 필요. $\beta = \alpha - 1/\alpha \ge 0 \Rightarrow \alpha \ge 1$.

$\alpha \ge 1$에서의 다항식 값:
| $\alpha$ | 다항식 값 |
|---|---|
| $1$ | $2-3+27-3+2 = 25 > 0$ |

*접선 조건 (ii)*는 $\alpha \leftrightarrow \beta$ 대칭이므로 동일한 결론.

**→ $f(x) = x^3 - 3x^2 + x$가 유일한 해.**

## 완성된 문제

> 최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.
>
> **(가)** $f(0) = 0$이고, 방정식 $f(x) = 0$의 서로 다른 세 실근은 모두 $0$ 이상이다.
>
> **(나)** 방정식 $f(x - f(x)) = 0$의 서로 다른 실근의 개수는 $8$이다.
>
> **(다)** $f''(t) = 0$인 실수 $t$에 대하여 $f(t) = -1$이다.
>
> $f(4)$의 값을 구하시오. $\boxed{20}$

## 출처 아이디어
**수학적 객체**: 최고차항의 계수가 1이고 서로 다른 세 실근 $r_1 < r_2 < r_3$을 갖는 삼차다항함수 $f(x)$.

**사실 A가 부과하는 제약** (자기합성 평행선 분해):
방정식 $f(x - f(x)) = 0$은 세 개의 하위 방정식

$$f(x) = x - r_1, \quad f(x) = x - r_2, \quad f(x) = x - r_3$$

으로 분해된다. 이들은 기울기 1인 평행직선이므로 서로 다른 분기의 해가 겹치지 않는다. 총 실근의 개수에 제약이 주어지면, 각 분기가 3개(횡단 교차), 2개(접선 교차), 1개(단일 교차) 중 하나를 기여해야 하며, 특정 총수를 달성하려면 일부 분기에서 반드시 접선 조건 $f'(\text{교점}) = 1$이 강제된다.

**사실 B가 부과하는 제약** (등도함수값의 변곡점 대칭):
$f'(x) = 1$을 만족하는 두 점 $p_1, p_2$는 도함수 $f'(x)$(이차식)의 대칭축, 즉 변곡점 $s = \frac{r_1 + r_2 + r_3}{3}$에 대해 대칭이다: $p_1 + p_2 = 2s$. 삼차함수의 변곡점 대칭($f$가 점 $(s, f(s))$에 대해 점대칭)에 의해, 두 접선의 $y$절편 $c_i = p_i - f(p_i)$의 합은

$$c_1 + c_2 = 2s - 2f(s)$$

로 변곡점의 좌표만으로 결정된다.

**숨은 사실 (A+B 동시 성립 시 강제되는 것)**:
총 근의 개수 조건이 두 분기의 동시 접선을 강제할 때, 대응하는 두 근 $r_i, r_j$가 $c_1, c_2$와 일치해야 하므로 $r_i + r_j = 2s - 2f(s)$가 성립한다. 비에타 공식 $r_1 + r_2 + r_3 = 3s$와 결합하면, 접선 분기에 참여하지 않는 나머지 근 $r_k$의 값이

$$r_k = s + 2f(s)$$

로 확정된다. 이는 자기합성 방정식의 "해의 개수"라는 이산적·조합론적 조건이, 변곡점 대칭이라는 기하학적 경로를 거쳐, 근의 구체적 값이라는 대수적 결론을 강제하는 것이다.

## 수험생의 발견 순간
1. **수험생이 처음 시도하는 것**: $f(x - f(x)) = 0$을 전개하려 하지만, 직접 대입하면 9차 방정식이 되어 계산이 불가능함을 깨닫고, $f(t) = 0$의 근 $r_i$를 이용해 $f(x) = x - r_i$ 세 개로 분해한다. 각 분기의 해 개수를 그래프적으로 파악하여 총수를 조합한다.

2. **막히는 순간**: 접선 분기가 필요하다는 것까지는 도달하지만, "어떤 근이 접선 분기를 담당하는가?"와 "접선 조건이 세 근 사이에 어떤 관계를 만드는가?"에서 자유도가 너무 많아 보여 막힌다. 세 근이 독립적 매개변수처럼 보이기 때문이다.

3. **발견**: $f'(x) = 1$의 두 해가 변곡점에 대해 대칭이라는 사실을 인식하는 순간, 두 접선의 $y$절편 합이 $2s - 2f(s)$로 고정됨을 깨닫는다. 이것이 접선 분기를 담당하는 두 근의 합을 고정하고, 비에타 공식과 결합하면 나머지 한 근이 $s + 2f(s)$로 완전히 결정된다. **"근의 개수를 세었을 뿐인데, 근의 값이 정해지다니!"**

4. **STEP 2-2의 숨은 사실과의 대응**: 발견의 핵심인 "$r_k = s + 2f(s)$"는 정확히 STEP 2-2에서 도출한 숨은 사실—변곡점 대칭이 접선 쌍의 $y$절편 합을 고정하고, 비에타 공식이 나머지 근을 확정한다—에 대응한다. 수험생은 "이산적 개수 조건 → 미분기하학적 대칭 → 대수적 값 확정"이라는 세 단계의 논리적 연쇄를 단일 등식으로 압축하여 발견하게 된다.
