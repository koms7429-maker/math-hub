# [STEP 1] 구조 분석

## 1-1. 조건이 만드는 방정식 체계

$f(x) = x^3 + ax^2 + px + q$로 놓으면:

| 조건 | 도출되는 방정식 |
|---|---|
| **(가)** $f''(0)=0$ | $f''(x)=6x+2a$이므로 $2a=0$ → $\boxed{a=0}$ |
| **(나)** 구간 최댓값 동치 | $f(x)=x^3+px+q$에서 극점 $x=\pm\sqrt{-p/3}$의 위치에 따른 **경우 분류** → $p$에 대한 방정식 |
| **(다)** $\int_{-1}^{1}f(x)f'(x)\,dx=0$ | $\frac{1}{2}[f(1)^2-f(-1)^2]=0$ → $(1+p+q)^2=(-1-p+q)^2$ → $\boxed{4q(1+p)=0}$ |

**(다) 전개 상세:**

$$\int_{-1}^{1}f\cdot f'\,dx = \int_{-1}^{1}\frac{1}{2}\bigl(f^2\bigr)'\,dx = \frac{f(1)^2-f(-1)^2}{2}=0$$

$$\Rightarrow\; \bigl[(1{+}p{+}q)-(q{-}1{-}p)\bigr]\bigl[(1{+}p{+}q)+(q{-}1{-}p)\bigr]=0$$

$$\Rightarrow\; (2+2p)(2q)=0 \;\Rightarrow\; q=0 \;\text{ 또는 }\; p=-1$$

## 1-2. 분기 분석

**(나)의 경우 분류** — $\alpha = \sqrt{-p/3}$로 놓으면:

| 경우 | $p$ 범위 | 구간 내 극점 | 최댓값 비교 결과 |
|---|---|---|---|
| A | $p\ge0$ | 없음 (단조증가) | $f(0)=f(1)$ → $1+p=0$ → $p=-1$ **모순** ✗ |
| B | $-3\le p<0$, $\alpha<1/\!\sqrt{3}$ | 양쪽 구간 모두 존재 | $\max[-1,0]=f(-\alpha)=2\alpha^3+q$, $\max[0,1]=f(1)=1-3\alpha^2+q$ |
| C | $-3\le p<0$, $\alpha\ge1/\!\sqrt{3}$ | 양쪽 존재, $f(1)\le f(0)$ | $2\alpha^3+q=q$ → $\alpha=0$ **모순** ✗ |
| D | $p<-3$ | 구간 밖 (단조감소) | $-1-p+q=q$ → $p=-1$ **범위 모순** ✗ |

**생존 분기: B만.**

경우 B의 방정식:

$$2\alpha^3 = 1-3\alpha^2 \;\Longrightarrow\; 2\alpha^3+3\alpha^2-1=0$$

인수분해: $(\alpha-\tfrac{1}{2})\cdot 2(\alpha+1)^2=0$ → $\alpha=\frac{1}{2}$ (유일한 양의 실근)

$$\therefore\; p = -3\alpha^2 = -\frac{3}{4}$$

검증: $\alpha=\frac{1}{2}<\frac{1}{\sqrt{3}}$ ✓

### 판정 1-2

분기가 유일하게 $p=-\frac{3}{4}$로 수렴. 복수 분기 없음. **CONTINUE.**

## 1-3. 자유도 계산

| 단계 | 결정된 파라미터 | 남은 자유 파라미터 |
|---|---|---|
| 초기 | — | $a,\,p,\,q$ (자유도 3) |
| (가) 적용 후 | $a=0$ | $p,\,q$ (자유도 2) |
| (나) 적용 후 | $p=-3/4$ | $q$ (자유도 1) |
| (다) 적용 후 | $q=0$ 또는 $p=-1$ | $q=0\;\because p=-3/4\ne-1$ → **(자유도 0)** |

### 판정 1-3

자유도 0, 해가 유일. **CONTINUE.**

## 1-4. 탐색 전략

PM의 수치 $p=-3/4$이 자연스럽게 결정됨. $f(x)=x^3-\frac{3}{4}x$가 유일해. 발문 $2f(2)$는 정수값을 산출할 것으로 기대. **CONTINUE.**

# [STEP 2] 수치 탐색 + 포장 설계

## (1) 후보 나열 및 계산

**후보 1 (PM 제안값 그대로):** $p=-\frac{3}{4},\;q=0$

$$f(x)=x^3-\frac{3}{4}x$$

$$f(2)=8-\frac{3}{2}=\frac{13}{2},\qquad 2f(2)=13$$

**조건 역검증:**

| 조건 | 검증 |
|---|---|
| (가) | $f''(x)=6x$, $f''(0)=0$ ✅ |
| (나) | $\max_{[-1,0]}f = f(-\tfrac{1}{2})=\tfrac{1}{4}$, $\max_{[0,1]}f=f(1)=\tfrac{1}{4}$ ✅ |
| (다) | $\frac{1}{2}[f(1)^2-f(-1)^2]=\frac{1}{2}[\frac{1}{16}-\frac{1}{16}]=0$ ✅ |

- **정수성:** 답 $13$ ✅
- **유일성:** 자유도 0으로 확인 ✅
- **손계산:** 인수분해 $2\alpha^3+3\alpha^2-1=(\alpha-\frac{1}{2})\cdot2(\alpha+1)^2$ 수준 — 4점 문항으로 적정 ✅

후보 1이 모든 기준 통과. **채택.**

## (2) 포장 설계

| 포장 요소 | 결정 | 근거 |
|---|---|---|
| **발문** | $2f(2)$의 값 | $f(2)=13/2$이므로 $2$를 곱해 정수 $13$ 산출. 깔끔함 |
| **조건 순서** | (가)→(나)→(다) | 단순→복잡 순서. (가)가 형태 축소, (나)가 $p$ 결정, (다)가 $q$ 결정이라는 자연스러운 흐름 |
| **조건 표면 형태** | 모두 PM 초안 유지 | 등식 형태가 자연스럽고 변경 불요 |
