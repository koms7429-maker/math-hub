## 1. 아이디어 셀렉터 (Idea Maker)

### 수학적 상황
**수학적 객체**: f(x) = x³ − 3px (p > 0), 임계점 x = ±√p, 극값 ±2p^{3/2}.

**사실 A (P003)가 부과하는 제약**: 세 근의 합이 0이므로 f는 원점 대칭(기함수)이며, 임계점과 극값이 ±√p, ±2p^{3/2}로 완전 대칭 배치된다.

**사실 B (P020)가 부과하는 제약**: 길이 L인 닫힌 구간 [a, a+L]을 평행이동할 때, 구간 위 f의 최댓값과 최솟값이 모두 변하지 않아야 한다. 이는 끝점 f(a), f(a+L)이 극값 ±2p^{3/2}의 역할을 정확히 대체해야 함을 뜻한다.

**숨은 사실 (A+B 동시 적용)**: L = 2√p로 설정하면, 끝점 조건 f(a) = −2p^{3/2}는 a³ − 3pa + 2p^{3/2} = 0, 즉 **(a + √p)²(a − 2√p) = 0**으로 인수분해된다. a = 2√p는 구간이 극점 영역을 벗어나 최댓값이 폭발하므로 기각되어, **a = −√p(이중근)만 유일하게 생존**한다. 따라서 극값 범위를 온전히 담는 창의 위치는 단 하나이며, 이것이 이중근에 의해 보장된다.

---

### 수험생의 발견 순간
1. **처음 시도**: 구간 [a, a+2√p] 위에서 f의 최대·최소가 극값과 같도록 a에 대한 방정식을 세운다.
2. **막히는 순간**: 삼차방정식 a³ − 3pa + 2p^{3/2} = 0이 나오고, 일반적 풀이로는 a의 범위를 좁히기 어렵다.
3. **발견**: 근의 합 = 0 대칭성 덕분에 a = −√p가 이미 극점(=자명근)임을 알아차리고 인수분해하면 **(a+√p)²(a−2√p) = 0** — 자명근이 **이중근**이다. 나머지 근 a = 2√p는 구간이 단조 증가 영역에 빠져 극솟값을 포착하지 못하므로 탈락.
4. **대응**: 이 이중근 구조가 바로 STEP 2-2의 숨은 사실 — "끝점 방정식의 이중근이 유일한 위치를 강제한다" — 에 해당하며, 근의 합 = 0이라는 대칭 조건이 없었다면 이중근 인수분해 자체가 불가능했을 것이다.

## 2. 프라블럼 메이커 (Problem Maker)

### 문제 초안
> 최고차항의 계수가 1인 삼차함수 $f(x)$와 양수 $L$이 다음 조건을 만족시킨다.
>
> **(가)** 모든 실수 $x$에 대하여 $f(-x)=-f(x)$이다.
>
> **(나)** $f(t)=f(t+L)$이고, 닫힌 구간 $[t,\; t+L]$에서 $f$의 최댓값과 최솟값의 차가 $f$의 극댓값과 극솟값의 차와 같은 실수 $t$가 존재하도록 하는 **양수 $L$의 최솟값**이 $L$이다.
>
> **(다)** $f'(0)=-a$ &emsp;($a$는 양의 정수)
>
> $f(L)$의 값을 구하시오.

*(수치 슬롯: $a$는 $a=3k^{2}$인 양의 정수. detail\_builder가 $k$를 결정하여 $f(L)=18k^{3}\in\mathbb{Z}$를 확보할 것.)*

---

##

### 조건-상황 대응표
| 조건 | 수학적 상황 요소 | 역할 |
|---|---|---|
| **(가)** $f(-x)=-f(x)$ | 사실 A: 원점 대칭(기함수) | $f$를 $x^3-3px$로 한정; 이후 $f(x)=M$ 방정식의 자명근 $x=-\sqrt{p}$ 포착 및 인수분해의 열쇠 |
| **(나)** 전반부 "$f(t)=f(t+L)$" | 양 끝점 동치 조건 | 길이 $\sqrt{p}$ 구간(끝값이 서로 다른 극값)을 차단; 끝점이 같은 수평선 위 → 삼차방정식 $f(x)=c$의 근 구조로 유도 |
| **(나)** 후반부 "진동폭 = 극값 차, 최소 $L$" | 사실 B: 극값 범위를 온전히 담는 최소 창 | "최소 $L$" 조건이 $f(x)=M$의 **이중근** 구조를 강제하지만 직접 드러내지 않음 |
| **(다)** $f'(0)=-a$ | 파라미터 $p$ 확정 ($p=a/3$) | $L=3\sqrt{p}$에 대입 → 최종 답 $f(L)=18p\sqrt{p}$ 계산 |
| *(숨은 사실)* | $(t+\sqrt{p})^{2}(t-2\sqrt{p})=0$ 이중근 | (가)+(나) 연립 시 학생이 스스로 발견; 이중근이 유일한 구간 위치를 강제하여 $L=3\sqrt{p}$를 도출 |
